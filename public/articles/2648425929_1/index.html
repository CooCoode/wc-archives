新版SeeDAO Logo
WHY?
为什么我们要更换logo？
随着SeeDAO的不断发展和成员数量的增加，我们已经明确自己的叙事为连接全球加密游民的数字城邦。原本的眼睛盾牌是在最开始SeeDAO成立时所设计的，它已经不足以表明SeeDAO现在想要连接全球的愿景，并且也可能显得过于封闭和稚嫩。这是我们此次更换logo的主要原因。
我们希望通过新logo更加清晰地传达出SeeDAO的使命和愿景：成为一个连接全球加密游民的数字城邦，并促进城邦成员在网络中的“连接、流动、交换、自由”。
(
新Logo
 在表达什么？
)
我们在新logo中寄予了非常多的意涵和希冀。

第一个意涵是全球主义而不是民族主义。我们希望连接全球的加密游民，构建beyond nation states的数字城邦。我们希望建立起一架连接东方和西方的桥梁和网络，帮助信息、人员、资金和想法更好地在这张网络中流动，而不是走向封闭与对立。

第二个意涵是铭记我们的文化根源。太极是对中华文明的高度概括、抽象和总结。无论我们走到全球任何一个地方，我们都不会忘记自己的文化根源。我们要把追求阴阳平衡和辩证之精神延续到赛博空间，不仅要在在文化演变和治理思路上形成自己的特色，还要对全球DAO的发展有所贡献。

第三个意涵是在剧烈的变动中不断革新。太极寓意着流动和变化，而我们这个时代的变动尤为激烈。古老的文明和智慧虽然不会过时，却需要结合这个时代的语言、技术和环境时常革新。SeeDAO的logo不是古板的太极图，而是结合了年轻时尚元素之后的改版。——这象征了我们的态度：不做守旧派，在急剧变革的时代找到一条新路。

最后还是那份希冀：连接、流动、交换、自由。SeeDAO不是一张无情的商业网络，而是一个以人为本的数字城邦。黑白之间你中有我，我中有你，追求的是人与人之间交集与“间性”。我们不要圆与圆之间的相切式社交，而要八卦里的黑白相融、流动交换；不要仅基于金钱的Web3体验，而要追求更加自由、交融、协作的网络社会。

让我们这些加密游民们从亚洲走向全球，连接世界，游牧阴阳。

SeeDAO logo
谁可以使用？
)
根据SIP，SeeDAO L2及以上贡献者有权使用SeeDAO logo，且logo使用实行备案制。
任何人在使用SeeDAO logo前都需要前往logo使用备案库中备案登记，其链接为https://tally.so/r/w4QxNd。备案登记完成后，社区成员即可使用logo（下载页面
https://seedao.notion.site/f5d19f35f2e14b81b3335ca1fbf8d03d），不需要另外再得到品牌小组的许可。
同时，备案信息会公开在受全体SeeDAO成员公共监督的notion链接中
https://www.notion.so/seedao/SeeDAO-8cab098f1be14413b220516e0c7084e0 。任何社区成员均可以对logo的使用产生质疑，并向SeeDAO品牌小组反馈质疑内容。如果经过品牌小组鉴定，该logo使用确实损毁了SeeDAO品牌形象，则将向logo使用者提出提醒，且该logo使用者在未来三个月内都不得再使用SeeDAO logo。如果该成员连续两次受到提醒，则此后都不得再使用SeeDAO logo。
如果外界合作方不经SeeDAO允许而直接使用了SeeDAO logo，并且造成SeeDAO 品牌形象损毁的，将被列至SeeDAO品牌合作黑名单中。
