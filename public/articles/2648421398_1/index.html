NFT 项目的 7 种市场营销策略

作者: 翻译公会 Diamond 发布时间: 2022-07-21 11:29:28
NFT 项目营销综合指南。

摘要：

本文中作者首先指出了 Web3 中客户的角色转变，随后给出了 7 个关于 NFT 项目营销的策略，并进行解读以及详细的方向。是一篇非常实用，对 web3 builder 很有参考价值的文章。

毋庸置疑，Web3 和 Web2 的进入市场战略和客户行为都大不相同。Web2 公司在销售和营销团队上大量投资，因为传统营销漏斗的重点集中在创造商机、获取和留存客户。

而在 Web3 中，一种新的企业建设模式已经浮出水面。该模式利用区块链的 NFT 将用户转变为所有者，取代了中心化领导团队；后者通过使用消费者数据和免费的用户生成内容来对产品和服务做出所有决策。这转变了 Web3 企业的营销方式，也转变了客户对其角色的看法。本文的重点将放在特定类型的 Web3 企业，尤其是 NFT 项目。您可以在此处找到 NFT 的详细概述。

那么，NFT 营销是什么样的呢？

我们又可以从 Web2 营销中借鉴哪些实践经验？

在一个每周都有跑路项目、有熊市或出现两者交叉情况的空间中，个人应该如何做好准备并且经得起未来的考验？

简而言之，你要如何打入市场并说服潜在客户将时间、金钱和精力花在你的 NFT 项目上？

我想先给 Web3 领域的新手概述一下过去 40 年互联网和营销的演变。

Web1、Web2 和 Web3 中互联网和营销的演变概述

信息来源：https://twitter.com/womenof_web3

正如 a16z 所说，「Web3 让网络参与者团结起来，共同为网络的发展这一目标齐心前进。」

那么这对于 Web3 营销，特别是 NFT 营销来说意味着什么？

以下是营销 NFT 项目时要考虑的 7 个原则：

  1. 将你的客户视为你品牌价值的共同创造者。社区的强大就相当于品牌的强大。

  2. 建立一个明确且独特的使命来激发和建设社区。

  3. 社区成立后确立品牌大使，并给予提升和激励。

  4. 把实用性当作你在市场上的竞争优势，也是一种对社区的激励。

  5. 利用与项目相一致的关键意见领袖（KOL）和战略合作伙伴关系，以此提高品牌知名度。

  6. 有效发展社交媒体的同时制定增长战略方针。

  7. 分析和监控重要的衡量指标，根据需要通过反馈进行更新迭代。

让我们逐一拆解这些原则：

Web2 企业通过漏斗图来审视自己的客户群体，着重将客户从漏斗顶部的意识阶段和考虑阶段，发展为漏斗底部的转化阶段和留存阶段。

在 Web3 中，代币提供了另一种吸引和获取新客户的方式。NFT 项目可以使用代币来吸引早期用户，并作为对他们进行品牌建设贡献的经济奖励。这些早期用户不仅会成为布道者，将更多人带入网络并宣传社区利益，他们还会：

a. 为项目提供未经过滤的反馈循环，了解哪些是有效的，哪些是无效的。

b. 作为项目的内测人员，并愿意在出现问题时耐心地提供帮助。

因为有了金融属性这一优势，NFT 项目激励客户进行宣传，帮助发展社区并从中获益。社区成员和客户可以获得直接激励，他们希望项目能成功。

“

最主要的转变是「让客户成为所有者」，这样早期用户和贡献者就能获得他们帮助平台创造的部分价值，并最终对其进行治理且获得更多控制权。我们本质上是将这些公司提供给用户，这是我们在这里创造的最大的社会转变。

—— Bradford Stephens（Blockchain Capital 联合创始人兼管理合伙人）

”

将您的客户视作共同创造者和合作伙伴，与您一起创造价值并从中获益。因此，可以在整个漏斗模型中加入激励措施。当面临艰难的决定时，可以收集社区的反馈，将决策众包出去。

在 Web3，「进入市场」策略变成「进入社区」策略。社区是每一个 NFT 项目是否能成功的核心因素。那么，如何有效地构建社区？

NFT 项目需要通过明确的任务来了解其社区建设，以了解其存在的原因，即您的 NFT 项目试图解决什么问题，以及您为什么是解决该问题的最佳团队？这需要有一个高质量的社区和适当的组织治理模式来支持，以完成这项任务。

制定一份宣言可以有效地吸引社区成员，让成员对 NFT 项目的使命和长期目标产生共鸣。例如，Meta Angels NFT 宣言指出，「Meta Angels 的价值在于其网络、成员的质量以及我们的伙伴关系和编程深度保持稳定，无论市场波动如何。我们非常关心为我们的成员创造真正的、切实的价值，我们不会基于非理性的二级市场来做出决策。即使我们会因为没有遵循游戏规则而受到打击，也绝不会把炒作看得比踏实做事重要。」

制定社区战略时，考虑 4 个关键方面：

a. 社区设计

  • 项目的目标是什么？

  • 你想传达怎样的价值观？

  • 你有什么独特的仪式？（例如：周三烧脑竞赛）

  • 创建一个自给自足的社区需要哪些角色？（例如：Discord / 社区经理、营销员、合作经理、工程师、项目经理）

  • 你的路线图和商业计划是什么？

  • 如何记录社区的宗旨、愿景、价值观和路线图？

b. 社区管理

  • 优先级成员将获得哪些角色和访问权限？会有成员等级吗？

  • 社区角色将如何通过漏斗获得激励？（包括意识阶段、考虑阶段、转化阶段和留存阶段）

  • 如何进行治理和决策？（例如：去中心化自治组织）

  • 为社区提供哪些支持角色？何时提供以及如何提供？

c. 社区参与

  • 如何欢迎社区的新成员？（例如：给每个新成员配对一个入驻引导伙伴）

  • 如何培养关系并建立信任？（例如：根据成员为社区做贡献的时长来解锁特殊福利）

  • 如何在社区成员之间建立联系？（例如：Discord 社区活动，给予渠道推广一定奖励）

  • 如何鼓励社区成员以充分实现利益并提高他们的整体参与度？（例如：提供一个白名单名额来铸造你的 NFT？成员通过参与到工作群来为社区做出贡献？)

  • 与社区成员互动的频率如何？

d. 社区共创

  • 您是否在社区内组织会员可以参与的项目？如何组织工作？例如 Web3 公会是一个有具备加密行业经验的开发者、设计师和思想家组成的集体，他们共享资源（无论是知识还是劳力）以追求一个共同目标。

  • 社区成员能否就他们希望作为社区支持的计划进行投票？（参考 Decentraland DAO 是如何让代币持有者对影响平台内容和未来运转的决策进行投票的）

来自 Odyssey DAO 的 Peter Yang 通过研究三个案例（VeeFriends、Moonbirds 、Meta Angels）概述了 NFT 项目如何建立繁荣的会员社区。

“

所有这三个项目都通过以下方式建立了繁荣的 NFT 会员社区：a）创作者致力于为持有人提供长期价值；b）社区应优先考虑关心实际效用而不是想快速套现的人；c）效用会将人们聚集在一起并鼓励他们长期持有。

”

在 Web3 中建设最令人兴奋的事情之一就是，项目有机会加深与客户的关系。像 Discord 这样的平台允许项目用 100% 的时间建立超级粉丝社区并接触他们的客户。这种未经过滤的访问为项目提供了机会，以吸引来源创新和共同创建计划，同时获得反馈。

在这些超级粉丝社区中，通常可以找到少数人愿意宣传你的品牌，并与之同舟共济。这些人就是你的品牌大使。

如何识别品牌大使？

一个优秀的品牌大使通常是独立的个体，通过对社区做出持续有价值的贡献，他们赢得了社区成员的信任和尊重。因此，他们在社区内部拥有一定的影响力。

品牌大使很有价值，因为他们为您的项目带来信誉，并有能力扭转负面评价。他们还可以抓住那些还在犹豫、观望的用户，引导并欢迎他们进入社区。

如何激励品牌大使？

项目可以采取经济激励或非经济激励的措施来激励品牌大使。

1）经济激励

  • 额外的代币——根据大使对社区的贡献,为他们提供额外的 NFT。这可能包括一个阶层贡献模式，即高度活跃的大使解锁稀有的 NFT 或其他代币。

  • 品牌功能的独家访问权 —— 允许大使根据自己的贡献解锁对其他功能的访问权。这可能包括艺术品空投、独家商品或合作产品的代金券。

2）非经济激励

  • 白名单——与其他将要铸造的项目进行合作，大使将被纳入这些项目的白名单，在铸造时可以获得小幅折扣。

  • 社会和公众认可——大使通常已经是品牌的拥护者，因此在公众面前对他们的贡献给予认可可以加深这种关系。通过使用主页公告栏进行公示或向更广泛的社区转发他们的工作成果，可以实现这一激励。

  • 社区内部地位——在社区中赋予大使地位或头衔颇有裨益。作为这个头衔的一部分，项目可以为大使提供为品牌出谋划策的机会。

NFT 实用性是奖励 NFT 持有者作为社区一份子的方法，并且给予他们一种归属感。当定义一个项目的实用性时，你需要考虑 NFT 为客户服务产生的价值以及团队是如何独特地传达这些服务的。为项目的实用性以及路线图设定设定明确、现实的期望。这将吸引来一些致力于项目使命的社区成员，而不是那些为了套利或要求从项目中获得即时效用的人。

NFT 项目实用性的原则：

a. 接触独家合作伙伴和顾问（虚拟和现实生活）

项目可以给代币持有者提供接触独家合作伙伴和顾问的机会，帮助他们发展自己的项目。访问权限可以采用 Discord 聊天，现实世界里的会议或者研讨会的形式。例如，Crypto Packaged Goods NFT 的实用性在于它是「将 Web3 中的优秀人才整合到一起的代币」，其核心成员包括：Jaime Schmidt、Chris Cantino、Randi Zuckerberg 和 Zeneca 等等。此外，持有 CPG NFT 还能为会员提供与领头企业家交谈和电话研讨会的机会。

b. 产品的访问权限（虚拟和实物）

项目可以考虑与一些 Web3 品牌合作或利用和 Web2 品牌的关系，为代币持有者提供一系列实体和虚拟产品的访问权限。Surge Women NFT 就是一个利用与其他品牌的合作伙伴关系为其代币持有者提供实用性例子的项目。正如其网站所强调的，Surge Passport NFT 将包含一个由教育、金融和 Web3 工具组成的生态系统，为代币持有者提供游戏中的加密皮肤。其中一些实用例子包括：

  • 购买 NFT 域名的优惠券

  • 区块链学习培训和高级大师班

  • 数字钱包代金券

  • 获得发展资助金的机会

  • 访问网络峰会

c. 未来价值的承诺（社交地位、代币价值上升）

那些在推出前迅速崛起或在推出前被名人认可的项目，可以利用这种承诺的吸引力突出代币持有者的未来价值。例如，拥有 Bored Ape Yacht Club (BAYC) 意味着持有者不仅有资金，同时加入了名人社区。正如 BAYC 所说，当你购买 Bored Ape 时，不仅仅是购买了一个头像或一件可能稀有的艺术品，等于同时获得一个俱乐部的会员资格，该俱乐部的福利和产品将随着时间的推移而增加。你的无聊猿将会作为你的数字身份为你打开数字大门。

随着媒体越来越去中心化，在提高项目知名度方面你需要与 KOL 以及一些志同道合的社区合作来发挥重要作用。

什么是关键意见领袖（KOL）？

关键意见领袖 (KOL) 是一个领域的权威，他们的建议受到该领域一些人的尊重，且他们通常有一个专门针对其利基市场的目标受众。

如何联系已确定的 KOL 并建立有意义的合作伙伴关系？

一旦你确定了与你品牌一致的 KOL 或战略合作伙伴，请通过电子邮件或社交媒体与他们联系。且需要考虑以下建议：

a. 写一封直击主题的函件

b. 简短和个性化（研究 KOL，了解他们产生共鸣的内容，并考虑如何为他们将你的项目变得生动）。

c. 为他们提供激励和价值。考虑一下你的项目可以为他们提供什么样的经济或非经济价值，来换取合作伙伴关系。

d. 考虑如何博人眼球——如果你能讲述一个关于项目的引人入胜的故事，就更有可能建立一些有趣的合作伙伴关系。

利用这些合作伙伴关系，且作为社交媒体内容日程的一部分。主持推特空间或播客来采访 KOL 和志同道合的社区，让他们邀请他们的粉丝并展示您的项目。与这些个人和一些其他品牌合作，为他们的粉丝提供特殊福利或社区准入权。

如何衡量与合作伙伴关系的成功与否？

跟踪伙伴关系成功的指标可分为两个关键类别：

1）品牌知名度指标，代表提高社区对您的项目的认知。这包括：

  • 增加关注者或社区成员（例如在 Twitter、Instagram 或 Discord 上）

  • 点赞、评论、转发

  • 注意力

  • 流量

2）直接响应指标，代表社区行为和购买意愿。这包括：

  • 网站点击量

  • 铸造的作品数

  • 潜在客户或电子邮件订阅者

除了与这些战略合作伙伴一起定义社交媒体的理念之外，同时还要去制定社群增长战略，共同扩张该NFT项目所涉及的范围和整体热度。而其中设计和制定战略方案的关键四个步骤包括：

a. 明确目标

设置可持续发展目标。例如短期目标可以包括作品全部售罄、更多 Discord 社区成员、扩张社交媒体软件数量，比如，同期发展 Twitter 和 Instagram。

b. 规划社交内容

建立社区的目标角色，并开发能够引起社群共鸣和互动的社交媒体内容，获得更多创造力，毕竟机会是无限的。

c. 构建内容日历

一旦你确立了你的首要目标并且完成了计划的社交内容，就可以构建一个社交媒体日历来确保准确性和即时性。

d. 衡量并实施你的战略计划

一旦你开始持续的发布社交媒体内容，必须去评估这些内容所带来的成果，来反映社区整体情况的变动，长时间的分析可以让你明确什么是必须的，什么是无意义的。接下来最重要的几个关于衡量社交媒体的指标包括：

  • 用户心智——人们看到内容的次数，由印象数和覆盖率确定。

  • 分享互动——用户的反应、评论、点击次数和对于本文内容的分享意愿

  • ROI —— 来自外部外源的转换和推荐链接

构建 NFT 项目时需要跟踪的一些关键指标，可以被分类为经济类和非经济类。

这些指标可以用来确定 NFT 质量、社区增长速度。项目所处阶段（例如铸造前 vs. 铸造后）的不同，应当关注的衡量标准也会有些差别。鉴于这个领域的变化速度，相关的指标需要经常检测并且随着市场的迭代，你的方法也需要随之更新。

经济指标包括：

a. 预估市值

可以通过你的 NFT 在 7 天内的平均价格乘以它的总供应量来计算。较高的预估市值表明，更多代币持有者可能愿意支付更高的价格来购买 NFT。推荐一个用于计算衡量预估市值的工具：rarity.tools。

b. 成交量

代表了对于 NFT 收藏品的总需求。

c. 地板价

可以用来监控 NFT 收藏品的入场价格。

非经济指标包括：

a. 相关社交媒体的社区增长规模

可以通过分析 Twitter、Discord 和 Instagram 指标来衡量。

b. 唯一代币持有者的数量

这是 NFT 收藏品数量的另一种度量标准，用于表示其可以接收到的感知力。独立持有者越多，品牌意识和参与范围就越广。

c. 社区参与的质量

可以通过项目使用的主要社交媒体平台来衡量。举个例子，在 Discord 上追踪社区活跃度：

  • 成员的积极性和黏性

  • 出席参加社区会议

  • 参与社区治理（对什么进行投票，以及投票的频率）

  • 根据规定的任务完成工作

d. 在社区成员之间建立新的关系和信任

可以通过情绪分析调查来衡量。

e. 钻石手人数

指未对项目进行售卖的持有者数量，可以表明社区对项目愿景和效用拥有长期信念的情况。

f. 售完时长

是指销售完全部的 NFT 收藏品所需要的时间。

g. 未来路线图交付

指项目团队在 NFT 的发展路线图中完成并交付每一步进展的速度和效率，也可以指代币持有者能够看到的成果的速度。
