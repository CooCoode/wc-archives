文学与 NFT 的想像世界的重合时刻。

こころ在日文中，是「心」的意思

“こころ，就是我。发生在こころ身上的所有事情，全都曾经发生在我自己身上。

こころ的所思、所想、所感，全部都来自我的心。”

                                                              ——《心》新版序

近日，香港作家董启章第二本 NFT 书《心》真诚上市；

继 2023 年 4 月与台湾飞地书店之飞地出版合作《天工开物·栩栩如真》，也是飞地出版第一本 NFT 书。

NFT 书之 meme 图  |  来源：飛地

董启章经典作品《天工开物．栩栩如真》2005年由台湾麦田出版社出版。18年后，经典藉由新科技重生。由飞地出版策展，飞地创办人张洁平参与编辑，漫画家柳广成设计封面，LikeCoin 赞赏公民创办人高重建担任技术总监，小说透过 NFT 书形式重新面市，为华文文学界的首例。

书中以14个物件、三代人的故事，糅合个人成长与家族变迁，串连起香港历史。这是董启章的第一次NFT出版实验，也邀请读者一起进入可能的世界 —— 回溯旧物承载的历史，在想像的文字工场里感受创造。全新章节带读者在人工智能、虚拟货币、区块链与NFT之间穿梭冒险⋯⋯

以上为台湾小说家骆以军为新版《天工开物．栩栩如真》撰写的推荐叙。

虚构一个想像世界，然后在里面生活。文学与 NFT 的想像世界终于重合，这也是 Web3 关注的创作者经济。

实现的背后依靠 LikeCoin 链，使自主性的去中心化出版成为现实。👏

为了让人能自主性去中心化出版，去 build 一条链。这本身就是一件值得「赞赏」的事。于是，我看着平时在 Matters 写作获得的 LikeCoin 不禁想更多地了解背后这条出版公链。

 文倩在 Matters 的钱包

阅读相关的书籍《区块链社会学：金钱、媒体与民主的再想像》、《白皮书》、《年度报告》、DAO 社群会议记录和官方公告等。

LikeCoin 是一个分散的出版基础设施，它通过分散的注册、奖励、编辑和治理来重塑出版业。分布式出版的核心是分布式注册中心，由 ISCN 提供支持。

它本身是一个基于 "Cosmos SDK" 的公共区块链。与其他公共块链不同，主要从事内容发布工作。Cosmos SDK 是一个开源框架，主要用于构建多资产投资（multi-asset）的公共权益证明（PoS）和被授权的权威证明（PoA）区块链。

按赞最早是 Facebook 的核心价值，发帖人得到按赞反馈，感觉良好。很少有人思考它的实际意义并不是以行动支持作者，而是喂饲数据给强大的人工智能。它变相反映：请多给我看跟这个贴文相关的内容和广告吧！

按赞习惯培养了十几年，但「赞赏」这知行合一的行为本身则有千年以上。

《北齐书 • 元文遥传》著“有人将《何逊集》初入洛，诸贤皆赞赏之”。

中文世界很巧妙，简单如「赞赏」一词组成的赞和赏还可以细分成两个意思，前者是表达一份赞同，后者更有行动支持之意。线上冲浪我们常说按赞，因为 Facebook 确实只有前半部分。

发起人高重建想多走一步，化赞为赏：读者按赞、作者获赏（金）。沿着这条思路，他和几位志同道合的伙伴于 2017 年着手设计「LikeCoin 赞赏币」。
