摘要

动态 NFT（Dynamic NFT，简称 dNFT）是数字资产所有权的未来。市场趋势表明，功能型与元宇宙概念 NFT 的流行，会进一步促进人们在大量用例中使用动态 NFT 技术。所以，在 dNFT 项目的早期进行投资和孵化有着巨大的潜力。以下研究介绍了市面上的一些有吸引力的用例和项目实例。

背景

什么是 NFT？

「NFT 是一种数字资产，可以代表现实世界中的艺术作品、音乐作品等东西，也可以代表游戏内的道具、视频等。它通常使用加密货币在线上买卖，并采用与许多加密货币相同的底层软件来编写。」

—— 《福布斯》：什么是 NFT

https://www.forbes.com/advisor/investing/cryptocurrency/nft-non-fungible-token/

NFT 是非同质化代币（Non-Fungible Token）的缩写。要想真正理解它，我们必须知道代币、同质化、非同质化的含义：

  • 代币是编码在区块链上的数据单元，通过智能合约强制执行。根据 Coinbase 的说法：

    「在＂代币＂的另一个越来越普遍的含义中，存在着一个更为具体的内涵，即，它用来描述运行在其它加密货币区块链上的各类数字资产。」- Coinbase

    https://www.coinbase.com/es/learn/crypto-basics/what-is-a-token

  • 同质化：与实物货币和其他加密代币一样，同质化意味着代币之间可以互换。每一枚代币价值相等，完全相同。

  • 非同质化：每个代币都有数字签名，让它成为独一无二的。每个 NFT 都有自己的价值，除非和另一个 NFT 价格相等，否则不能互换。每个代币都具有密码学上的唯一性。

非同质化引入了「数字稀缺性」这一概念，如 Arry Yu 所说：

「本质上，NFT 创造了数字稀缺性。」—— Arry Yu, WTIACBC

https://www.forbes.com/advisor/investing/cryptocurrency/nft-non-fungible-token/

我们已经厘清了 NFT 的含义，接下来我们要来看一些早期用例，探索一下它们的吸引力何以延续至今。一直以来，NFT 最流行的用例都在于将数字艺术货币化方面。整个 2020 - 2021 年期间，NFT 的影响力不断攀升，进入了主流视野。

目前市面上的各种知名 NFT 都可以归入所谓「静态 NFT」的类别，因为这些 NFT 在编码时，其元数据（核心属性）被锁死，永久不可变更。对于视频、图像、gif、3D 模型等形式的数字艺术作品来说，这没什么问题。我们来看一下静态 NFT 的一些核心特征。

静态 NFT（sNFT）的特征：

https://www.zeeve.io/blog/dynamic-nfts-new-world-for-nfts/

  • 为原创艺术而生

  • 给予用户明确的所有权

  • 由于具有元数据被锁死的特点，sNFT 最常见的用例是艺术品收藏。不论是过去还是现在，提高艺术家们的赚钱潜力都是一个颇具吸引力的价值主张。

    https://www.financialexpress.com/digital-currency/what-are-dynamic-nfts-dnftsand-how-do-they-differ-from-static-nfts/2597266/

sNFT 的缺陷：

  1. 不可更改的元数据：用例有限，特别是在现实资产、基于进度的游戏和区块链幻想体育联赛等方面的应用受限。

  2. 有限的增值空间：由于数字内容很容易下载、复制，除了鉴真证明和链上唯一性，sNFT 通常很难提供其他价值。因此，不难发现，最成功的 sNFT 项目都是那些围绕其自身建立了强大社区的项目。（比如：BAYC，CryptoPunks，Cryptokitties）

  3. 虚假内容：

    「根据 OpenSea 对自身市场的分析，其上架 NFT 中有超过 80% 是盗版作品。」—— 《连线》

    https://www.wired.com/story/nfts-dont-work-the-way-you-think-they-do/

    在 OpenSea 这种主流 NFT 市场上，尚有如此大量的盗版艺术作品，很明显，数字真品鉴定的落实还有很长的路要走。

静态 NFT 是第一代非同质化代币，其特点是不可修改、有限的价值主张、和盗版风险。接下来我们将探讨下一代 NFT ——动态 NFT。

什么是动态 NFT？

动态 NFT 是静态 NFT 的迭代。它可以根据数据的反馈，实时更改 NFT 的数据，而智能合约的逻辑也能够用来根据外部因素的刺激对 NFT 的属性进行微调。动态 NFT（dNFTs）的另一个有趣的定义是：

「有生命的真实资产。能够在外界的刺激下生长、变化、分解、再生。」—— CoinTelegraph

https://cointelegraph.com/news/nft-potential-takes-quantum-leap-with-koii-network-s-new-dynamic-nft-standard

dNFT 接受什么样的数据呢？有两种：「链上数据」和「链下数据」。链上数据可以通过智能合约直接访问，链下数据则需要由名为「预言机（Oracle）」的实体验证后，再添加到区块链中。

什么是预言机？

预言机可以传递来自链外数据源（比如 Web2 网站，传感器等）的实时数据。它也有一个有趣的定义：

「为智能合约提供验证过的、最新的数据，并触发其内部状态变化的服务。」—— Visionary Art

https://visionaryart.io/mag/nft/the-future-of-nfts-dynamic-nfts-and-physical-nfts/

预言机对动态 NFT 的实现至关重要，因为链下数据是 dNFT 拓展到不同用例的关键。

「对于动态 NFT 的普及而言，预言机不可或缺。」- Supra Articles

https://supraoracles.com/academy/web3-oracles-the-crux-of-dynamic-nfts/

了解完预言机的概念后，我们可以回到对动态 NFT 的定义中：

动态 NFT = NFT + 预言机

动态 NFT 具有各种静态属性，此外还有通过预言机输入来不断更新的动态数据。这是一个随时间改变其表现的 NFT。下面是一个常见的动态 NFT 合约的流程：

  1. 人工/程序向智能合约请求 NFT token URI

  2. 智能合约收到处理请求

  3. 智能合约调用链上数据，输出结果

  4. 智能合约从预言机调用链下数据，输出结果

  5. 智能合约决定要显示的内容

动态 NFT 是第一代 NFT 技术（静态 NFT）和实时数据反馈的结合。这些数据反馈可以是链上的，也可以是链下的。对于链下数据（关系到更多吸引人的用例）来说，需要将它们添加到区块链上，以供智能合约进行分析。预言机在执行这一服务方面发挥着关键作用。

动态 NFT 是第一代 NFT 技术（静态 NFT）和实时数据反馈的结合。这些数据反馈可以是链上的，也可以是链下的。对于链下数据（关系到更多吸引人的用例）来说，需要将它们添加到区块链上，以供智能合约进行分析。预言机在执行这一服务方面发挥着关键作用。

流量证明：浏览量

  • 描述：流量证明机制，可以显示某个用户给予一个网站的特定部分、一段内容或 NFT 的注意力总量。它相当于一个浏览量计数器或一个计时器，用来对用户参与内容的情况进行计数。这一用例将会被用来获取链上网站的参与度分析。

  • 价值主张：相当于 Web3 中的谷歌分析，它允许以开源的方式来跟踪内容的参与度和流行度，并能按查询次数或订阅的方式实现直接盈利。

  • 目标用户：需要跟踪自己的 NFT 的参与度，或中心化社交媒体平台上其它内容参与度的内容创作者。

超身份证明：动态网站可以成为 NFT

  • 描述：

  • dNFT 可以用来表示用户不断变化的线上身份。

  • dNFT 可以存储动态数据，这意味着动态网站可以铸造成动态 NFT。它让用户能够围绕其身份创建自己的个人网站。

  • 与此相对应的静态 NFT 是以太坊域名服务（ENS），它允许以太坊钱包用户以 NFT 形式创建一个容易记忆的用户名地址。这种 NFT 整合到了以太坊的各个市场和平台内，并且像以 .com 和 .org 结尾的 DNS 域名那样，提供了统一的域名后缀。

    「去年，.eth 域名的月注册量增长了 10 倍。」—— Visual Capitalist

  • 此处的用例是以太坊域名服务的多链动态版本。

  • 价值主张：通过动态 NFT，用户能够创建自己的个人身份，并在多条链、多个 dapp 上进行展示。

  • 目标用户：希望在多个链（不仅仅是 ENS）上个性化定制身份的 Web3 用户。

收益分享 / 影响力证明

  • 描述：

  • dNFT 能够用于再平衡嵌套版税中的收益分享激励问题。影响力证明则是一个登记表，记录了所有的合作信息以及某段特定代码的变更情况。

  • 有了动态 NFT，用户就能够将版税嵌套成多个内容条目，这样一旦某个新用户决定购买 NFT 并使用其 IP，版税就可以在该内容条目的所有贡献者和合作者之间进行分割。

  • 示例：一个 TikTok 视频走红后，总会被不断二次创作。这种情况下，如果把最初的那个 TikTok 铸造成一个动态 NFT 的话，每当有用户为了追赶潮流对其进行二次创作时，他就会支付给原创作者一小笔费用。现在，这个经过二次创作的 tiktok 视频的智能合约中就包含了原创作者和第一个进行二次创作的人，两个人就可以平等地分享该视频此后再被二创时所产生的收益了。这只是一个简单的针对数字 IP 和 内容赢利再平衡激励的模型，它也是当今社交媒体网络所面对的两个最大的问题。

基于玩家进度的区块链游戏

  • 描述：dNFT 可用于更加高效地跟踪玩家的游戏进度。在本用例中，dNFT 代表了玩家的游戏化身，可通过完成各类任务/关卡/游戏内的成就来进行更新。随着化身的更新，游戏中的新功能也得以解锁。

  • 价值主张：动态跟踪玩家的游戏进度，并根据玩家与游戏的互动程度，解锁不同的游戏体验。

  • 目标用户：对进度类区块链游戏感兴趣的玩家。

三个潜在用例：

  • 活动售票：增强活动参与者之间的收益分配与投票

  • 房地产：房产证可以铸造成动态 NFT。通过预言机，可以实时编译链下数据，以准确地评估房产价值，从而减少与财产评估相关的摩擦。

  • 教育：dNFT 可以用来追踪学生课程与学业表现。学生的大学成绩和表现情况可以通过预言机写进区块链。在链上添加学术记录的合法合规性是这一用例的主要阻碍。

潜在风险

  • 虚假内容/盗版思维：NFT 的不当重发

「根据 OpenSea 对自身市场的分析，其上架 NFT 中有超过 80% 是盗版作品。」—— 《连线》

  • 兼容性：NFT 的跨链兼容性

  • 合法合规性：2021 年总销售额约为 2860 万美金 —— 税收

  • NFT 通常需要缴纳资本利得税（类似于出售股票获利）。是否对 NFT 投资者征收长期资本利得税尚未确定，但既然 NFT 被视作收藏品，或许会课以更高的税率。

  • 美国国税局还未将 NFT 纳入征税体系。

  • 信息访问：预言机中心化的问题

「如果预言机的运营一直不能去中心化，长此以往，将导致更多的腐败、价格操纵以及网络攻击。」- Supra Oracles

  • 因为动态 NFT 直接通过预言机获取链下信息，正确的预言机协议设计是动态 NFT 成功的关键。对于像 Chainlink 或 Bond 这样的平台来说，在去中心化方面还有很长的路要走。

以上列出的前三个风险也是静态 NFT 目前正在面临的问题。因为动态 NFT 是新兴的技术，所以严格意义上的风险尚未浮现。实际上，静态 NFT 面临的问题，都需要通过动态 NFT 的实现来解决。

目前，我唯一能确定的 dNFT 风险就是预言机中心化的问题，它也是访问链下数据时广泛面临的问题。

结语

主要内容

动态 NFT 是静态 NFT 的迭代。第一代 NFT 技术（静态NFT）和实时数据反馈的结合就是动态 NFT。这些数据可以是链上的，也可以是链下的。

推动增长的趋势：

  1. 销量和总销售额均呈现增长

  2. 二级市场主导一级市场，即便在熊市中也是如此

  3. 为了通过作品赚钱，数字艺术家正转向直接面对消费者的 D2C 模式

  4. 大品牌正在采用 NFT 作为融资的手段 

  5. 元宇宙 NFT 市场与功能型 NFT 市场在总销量、总销售额、利润总额、活跃钱包数量和流动资产总量等方面，都经历了非比寻常的增长。这种增长可通过除数字艺术之外，NFT 技术实用性应用需求的增加来解释。

潜在的杀手级应用：动态 NFT

以下所列的潜在应用概念，代表在开发该用例的协议项目上有进行早期投资和孵化的机会：

  • 流量证明：浏览量

  • 超身份证明：动态网站可以成为 NFT

  • 收益分享：影响力证明

  • 基于进度的区块链游戏

潜在风险

下面前三个风险代表了当前静态 NFT 面临的问题，实际上，这些问题都需要通过应用动态 NFT 来解决。

  • 虚假内容/盗版思维

  • 兼容性

  • 合法合规性

  • 信息访问

唯一能确定的 dNFT 风险就是预言机中心化的问题，它也是访问链下数据时广泛面临的问题。
