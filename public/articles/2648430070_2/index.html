Rust 共学社区招募跨越 Web2 与 Web3 深入底层重构世界

Rust 作为一门系统级编程语言，在安全性、并发性和高性能方面具有卓越的表现，这些显著优势也让 Rust 连续七年成为开发者最喜爱语言。

现如今，Rust 在区块链、网络编程、嵌入式系统和系统编程等领域展现出强大潜力，优异的性能和广泛的应用，使其成为越来越多项目开发的首选，也有了区块链第一语言的标签。

本期共学课程，我们将通过系统学习与实战开发的模式，辅以好用的 AI 问答工具，帮助开发者更加轻松掌握 Rust，在共学社区，你可以与来自各个行业的小伙伴一同探讨问题，获得更加高效地成长。

或许未来 ChatGPT 等人工智能工具将在各个领域发挥重要作用，简单重复的代码编写任务不再重要，但对于底层原理的深度理解、与社区不断碰撞交流的重要性，将会愈加凸显。

Creators Notion & Github

共学社区 / Rust Co-Learn

我们将在 6 周的时间里学习 Rust，一起从入门到进阶，通过实战项目来熟悉这门语言，最后一周我们将用 Rust 产出一个应用 Demo。

共学期间，每周一次 Office Hour 环节，收集大家当周的问题，针对当周内容集中答疑和讨论，共同提升对于 Rust 实战开发的理解与认知。

我们还将邀请 Rust 领域资深嘉宾，根据大家的需求和反馈，发起不定期技术或案例分享，启发项目灵感，并为组队开发提供帮助与建议。
