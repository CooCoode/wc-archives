用NFT讲故事：自上而下 Vs. 自下而上

NFT 能够也应该成为讲故事的极佳方式，那么究竟是项目方主导的「自上而下」创作好，还是同人们「自下而上」地共同创作好呢？作者在文中对两者进行了介绍和比较；并给出了很多有趣的实例。

一个好故事能让你心有所感。这故事如果还归你所有，那这些情绪就更加强烈了。无论故事新旧，NFT 都能够成为讲述它们的极佳方式。

过去几个月中，有些项目一直在探索这个跨界领域。然而一些项目更倾向于采用「自上而下」的方式讲故事，而作为一种新媒体， NFT 能够、也应该探索「自下而上」的叙事手法。

「自上而下」是别人讲故事给我们听，就好比你跑去电影院看星球大战，或者玩 Bethesda（译注：著名游戏发行商）新出的角色扮演游戏。「自下而上」则是我们围绕虚构的世界发明出自己的故事：是拿着买来的光剑在院子里疯跑，一边用嘴发出模糊的嗖嗖声，一边用力场朝你的狗扔网球；也是在魔兽世界中暴风城外的小酒馆里玩角色扮演。

在这篇文章中，我想和大家分享用 NFT 讲故事会是个什么样子，以及我认为未来它会如何发展。

01

「自上而下」的 NFT 幻想文学项目

以下这些例子的主要目的是用媒体讲故事给观众听。其中的确有一些含有来自粉丝的贡献和互动，但故事的主干仍然是由创建它的人、小组、DAO 或企业所主导的。

「自上而下」的幻想文学可以用直接围绕 NFT 创作的形式讲述。

Gridlock节选（2020年12月）

这是我自己对直接创作 NFT 文学的尝试。以下是我的处女作 Hope Runners of Gridlock 的节选。你也可以在 NFT作家藏品 上看到我最初的更多想法。

Few Understand (2021)

Few Understand （2021）

出自 Kalen Iwamoto 创作的 “Few Understand” 系列故事。

增加了口语和视觉元素的诗歌也在蓬勃发展！

Wild One (2021)

一种带有视觉元素的口语体诗歌，来自 Artemis Wylde。

请点击链接（https://foundation.app/@ArtemisWylde/foundation/35599）前往欣赏。

然而一批新项目已经现身，它们的目的是讲述发生在 NFT 之外的故事。在这些例子中，NFT 自身成了故事内的物件、元素或角色。在某种意义上，这些项目将 NFT 视为可销售的商品。这是一种新的（预售或售后）众筹形式。因为这些项目具有让你自由发挥或做贡献的属性，所以它更多地介于「自上而下」和纯粹「自下而上」两者之间。

Stoner Cats (2021)

Stoner Cats 项目卖出了 800 多万美元的 Stoner Cats NFT。此外，还会有一部动画剧集讲述这些猫的冒险故事，其中一些故事情节可以由 NFT 持有者投票决定或参与创作。

Glue Factory (2021)

与 Stoner Cats 类似，粉丝们可以从即将上映的动画剧集中购买马匹。他们还在尝试建立社区作家工坊和人才基金（如果你的品种出现在剧集中，你就可以从中获利）。

Deadheads(2021)

类似的想法。动画剧集 + 一些影响故事的能力 + 3D 收藏。

Aku(2021)

Aku 讲述了一个想成为宇航员的年轻黑人男孩的故事。他的太空头盔让他可以探索许多个世界。它正在以章节形式发布，以实际的视频片段作为 NFT。它已获选被制作成一部故事片。

Untitled Frontier (2021)

在这儿也推销一下我自己的项目。Untitled Frontier 制作免费的科幻小说，并以 NFT 的形式出售故事中的数字商品。我们确实计划在一段时间内激励更多「自下而上」的故事创作。

Pillheads (2021)

Pills 是围绕以太坊的叙事，为与太坊有关的项目和人员添加了一些色彩。Pills 所有者可以通过投票对内容做贡献。他们也在一些案例中探索了做“任务”，这种体验很独特，粉丝们实际上参与到以太坊中去为故事和叙事增添内容。

Elektra (2021)

一个神奇的项目，粉丝们一起创造故事和音乐。例如: https://zora.co/songcamp/4280

02

「自下而上」的幻想文学项目

虽然以上那些「自上而下」的项目意在促进互动和贡献，但它们主要还是项目方 “认可” 或 “分封” 的。当粉丝们以不请自来的方式主动掌控 NFT，并将自己创作的故事灌输给它，「自下而上」讲故事的方式就应运而生了。它不再仅仅是让粉丝投票或贡献内容。

这就是将 NFT 嵌入幻想文学的新奇有趣之处。与之前的其他媒介不同，它更直接地让粉丝不仅能投入情感建设虚构的世界，而且还能期待从中获得相应的经济回报。同人小说并不是什么新鲜事物，在过去，我们就看到过一些大获成功的先例：《五十度灰》最初是《暮光之城》的同人小说；《时间的救赎》一开始是刘慈欣的畅销三部曲《三体：地球往事》的同人小说，最终也成为经典。然而，这毕竟是可遇而不可求。无论是出版作品还是孩子们在自家后院里创造自己的角色，也不管你是否喜欢，同人小说都有一个巨大的长尾。

NFT 所带来的能力，是将源起和故事作为虚构世界的一部分添加到 NFT 中。如果成功了，它就可以实实在在地给作家带去价值。打个比方：一个星战迷从商店里买了把光剑，改装一下之后为它写了个故事。假使这本同人小说出了名，作为为这个世界创造故事的象征，这把改装的光剑就可以当成豪华版商品拿去出售了。但如你所见：在现实世界中这个构想举步维艰。

NFT 能为「所有权 + 源起」这对组合更有效地赋能，它可以将同人小说纳入内核，并与之合并成为新媒体。这是一个介于商品 + 同人小说 + 投资 + 所有权之间的新媒体。试想，你将能够购买或拥有欧比旺（译注：星战主角之一），甚至还能给他增加新的故事情节...... 这些承诺的原型是社交媒体用户纷纷使用他们的 NFT 做头像。加密朋克（cryptopunk） 的新噱头，来自于 Jay-Z 在推特上 “穿戴” 了它。虽然还没有引发任何同人创作，但这件事本身也有一部分是元交互的结果。只是因为 Jay-Z 拥有加密朋克，它就增值了。

以下是一些「自下而上」讲故事的早期例子，即粉丝接管 NFT 并在原始项目之外创作新的故事。

Punks Comic (2021)

用加密朋克讲故事的有趣的实验。要想阅读这本漫画，你必须得是 10,000 个NFT 持有者之一。它还有个额外的机制，如果粉丝选择 “烧毁” 自己的漫画，就能获得 DAO 的访问权。

Jenkins The Valet (2021)

这个例子可能是我的最爱。一个团队从以 ”游艇俱乐部“ （译注，yache club，无聊猿的开发商）为背景的所有故事中选定了一只无聊猿，他是个 ”泊车猿“。粉丝可以购买作家工坊 NFT 来为这个 ”泊车猿“ 的故事做贡献。最后创作出来的故事本身也会成为 1/1 的 NFT。听上去挺复杂，但最有趣的部分是，人们已经为 ”泊车猿“ 创作了丰富的源起和故事。

Citizen 2890 (2021)

新故事虽然还没有明确，但 FlamingoDAO 购买的一个外星人加密朋克已经在自己的 Twitter 帐号上发了不少推。这个构思与 fuckintrolls 不谋而合。

爆个料：关注一下 Alethea。用户可以在自己持有的 NFT 上添加 GTP-3 引擎，为自己的 NFT 嵌入原生的小故事，从而更容易地进入幻想文学的世界。（译注：GTP-3是自回归语言模型的简称，该模型能够产生人类可理解的自然语言）

03

冲就对了！

除了 Twitter 用户们用 NFT 做头像以外，可能还有更多「自下而上」讲故事的例子，但仍然不是很普遍。可做的事儿很多，粉丝们可以一起购买一个现成的 NFT，创作故事加以润色，如果成功了，既可以继续购买其它 NFT 再这么玩一次，也可以再把它卖掉，将自己投入的时间和精力变现。感兴趣的话，可以去 PartyBids 看看如何上手。

要搞明白如何促进「自下而上」讲故事并不容易。这个领域刚出现不久，还很混乱，也可能会遇到一些坑。贡献的内容可能质量很差，甚至烂透。当前的加密市场中不乏对所有社会成员都不友好的恶搞表情包。话虽如此，在 NFT 出现之前，就有合作创作幻想文学的项目取得成功。这些问题都是老生常谈。在我的另一篇相关文章《 泛谈有限叙事和无限叙事》中有更详细的阐述。

关于这些项目的另一个大问题是知识产权（IP），以及该如何对待 IP 的问题。一些「自上而下」的项目无论现在或将来都仍有可能试图保留 IP。但是，我认为这是一个长期错误的做法。NFT 之所以能够积累价值，正是因为它可以被嵌入新的背景，并激发围绕它进行的故事创作。如果你限制了准入，那么你只不过等于关上了大门。大部分增值就只能依靠 NFT 本身，而不能依靠 IP 了。尽管如此，还是会有一些项目会找到如何有效管理这个中间地带的办法。然而，的确路会更难走。

正如 Jay Springett 在 Twitter 中回答的──我们需要的是自由的IP和强大的同人圈。我们正面临着小说和叙事的新时代，很高兴看这一切徐徐揭幕！欢迎分享本文没有提到的好例子!
