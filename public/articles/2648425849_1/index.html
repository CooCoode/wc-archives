去中心化程度
完全去中心化 ⬤ ⬤ ⬤ ⬤ ⬤ 完全中心化
如何创立：首个 DeFi 指数协议到其 DAO 化

继如今被炒得沸沸扬扬的 “DeFi summer” 后，随着 2020 年 9 月 DeFi 脉冲指数 (DPI) 推出，Index 应运而生。这一款早期加密货币指数产品（与 Set Labs 共同打造）的创新发布引起其第一批大型客户群体——对蓝筹 DeFi 项目感兴趣的加密货币投资者的注意。这些 Index 的早期用户，又分别把 Index 介绍给了自己的社区，很快贡献者蜂拥而至。如何才能让社区决定 Index 的未来呢？何不纵身一跃，成立一个 DAO。

当时 Defi 赛道有众多参与者，团队果断将产品 DAO 化（在当时还是一个很新颖的概念），再加上对金融和加密货币充满好奇的领导者们形成的全新的、去中心化的社区，建立了坚实的基础，这些因素共同造就了 Index Coop。

我曾与 Brad Morris（Index Coop的”社区蜂巢”现任领导者，负责监督人力资本、资源配置、人才流动、人力资源以及工具和运营）交谈，以了解他对社区成立之初成功要素的看法。他说：“大家希望成为社区的一员，可以在产品构建过程中帮助宣传。热忱之源和愿景之初都是为了打造去中心化的金融产品，让大家都能「薪薪」向荣。” 与 Web3 中许多项目一样，他们早期的成功很大程度上归功于「伯乐」，也是他们最大的支持者和拥护者，所带来的快速牵动力。

现状如何

如今，Index 拥有十几种行业指数产品，其中标志性的 DPI 产品市值为 2780 万美元，包括 Uniswap，Aave，Maker，Loopring 和 Synthetix 等顶级资产。

Index 的所有产品及其链接都在官网主页上：https://indexcoop.com/products

每推出一个新产品，Index 上就会建立一个新的子社区，每个子社区都以自己的方式为 DAO 做贡献。由于 Index 没有 CEO，决策工作通常以渗漏矩阵的模式进行，该矩阵包括经 Index 委员会选出的最高级别成员，其次是蜂巢（及蜂巢领导者）以及小队领导层。

时光飞逝，Index Coop 已经在行业中享有盛誉，无论你是否熟悉加密货币或 DeFi，都可以在这里获得最便捷入职的体验。他们的 Discord 现有 15000 名成员，社区充满了良益且有深度的对话和辩论，关于如何管理组织的机制。

从与贡献者的交谈中，我看到他们的构建周期框架，与其他新生项目相比，更显成熟。他们引入了一些新的机制，比如，让他们的投资者将投票委托给新选出的领导委员会。核心团队现在是按照季度工作，每个“季度”6 个月，为一组新的产品发布或者某些核心目标冲刺。

随着最初的新奇和热潮渐退，Brad 表示，社区周边的问题也有所变化，就如任何从初创公司转变为规模化公司所经历的一样。社区成员和利益相关者最初会进行多种尝试，问：“可行吗？这个看起来如何？”而现在的主要问题更现实一些：“如何保障财务收益？”
为何能成功

或许，与 DAO DeFi 模型的新颖性有一定关系，Index 吸引了一流的社区领袖、组织设计思想家和建设者，他们提出了一个健全的入职培训、兼职和全职工作分配体系（薪资以现金与代币结合的方式）、赏金榜和其他参与策略。迄今为止，他们仍是新社区用户入职流程最完善的社区之一，同时还有精心维护的 Notion 协作页面。

Index Coop 的社区手册是公开透明文档的一流示例，面向现有及潜在社区成员。

“作为去中心化组织，我们的成功（作为一个成功的 DAO 组织）是显而易见的。我们有利可图，寻找适合市场的产品，为我们的用户服务，建立了一个真正伟大的团队，我们帮助团队成员取得成功。” Brad 表示，“我们正在寻找中心化和去中心化之间的平衡点，在前进的过程中，进一步推动去中心化。”

不可否认，这种清晰的愿景和目标，很大程度上是通过社区各种“浴火试炼”实现的。作为首批完全去中心化的 DAO 运营组织之一，Brad 表示，他们很快意识到社区全员参与决策所面临的一些挑战。

“我们无法快速有效地做出决定，作为一个 DAO，这真的让我们很受挫。”他们现在正尝试通过基于产品的社区为客户构建更多内容，并寻找快速成长的方法。“我们刚刚选举了一个领导委员会；敏捷决策非常重要。我们相信这些人会迅速做出决定，且不必经由所有人。”
如何决策

  • 核心团队（全职员工）

  • 领导委员会

  • 猫头鹰级别（金、银、铜）和 $INDEX 代币持有者：通过 Index 论坛提出建议
Web3 三大共勉
  • 先发优势
  • 作为吸引一流社区领袖之地而享有盛誉
  • 可访问性（通过文档和入职）
给 Web3 创始人和建设者的建议

“在完全去中心化行不通的情况下，这种良性的紧张局面势必出现。如果任何人都能决策，是无法形成一致决定的。但是，完全中心化也行不通，因为你并没有从集体智慧中获益。我们正试图在两者之间寻找平衡点。”
