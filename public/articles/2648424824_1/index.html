Web 3.0： 娱乐行业的下一个结构创新

作者: 翻译公会 bo 发布时间: 2023-01-06 13:40:14
娱乐行业将走向何方？

摘要：

纵观娱乐业，虽说已经历无数重大变革：从家庭娱乐到特许经营到流量大战，而 Web 3.0 的出现，对娱乐行业的改变将是颠覆性的。无论是从创作者内容角度、还是收入分配、社区建设。本文中 fearture 3 试图从这些角度引领大家走进娱乐行业的大变革。

每隔一段时间，就会出现一项创新，引发整个行业的变革，迫使其运作方式发生转变。这类创新通常是技术进步的结果，它们可能是全新的商业模式或战略创意，也可能是新的流程。一些颠覆性的创新，需要整个行业的公司进行重构、重组。而那些不能适应潮流变化的公司，有些甚至是行业里的佼佼者，很快就会没落，最终被时代的潮流抛弃。

这种迫使整个行业进行重构的创新，就是所谓的结构创新。当下，媒体和娱乐业正在步入一个新的时期，面临着新一波的结构创新。区块链技术将彻底改变工作室、创作者和社区之间的关系；与此同时，也将影响受众对内容的体验以及内容的传播方式。

Web 3.0 是娱乐业的下一个结构创新。每个行业都会因结构创新而经历重大的变革时期，而娱乐业似乎更是这样的热土。是什么让结构创新有如此大的威力？Feature 又是如何在适应潮流同时，借助区块链流媒体设备 Feature³ 对其施加影响？在回答这两个问题之前，让我们简要回顾一下过往撼动娱乐业的其他结构创新。

娱乐行业结构创新简史

01.

暑期大片

暑期大片这个概念，我们并不陌生。每年夏天都会有一大波新片上映，豪华的明星阵容、顶尖的视觉效果、铺天盖地的营销活动，当然更离不开巨额的预算。由于好莱坞电影公司的大规模投资，这些电影通常也极其卖座。大片这个概念始于于 1975 年夏天上映的《大白鲨》，它是首部获得 1 亿美元票房的电影。1977 年夏天，乔治卢卡斯的《星球大战》打破了记录，获得 7.75 亿美元的票房。电影公司清楚地意识到，夏天是电影卖座的好时机。自此以后，为了顺应暑期大片这个新的业务重心，各影业公司对预算分配、拍摄规划、营销部门和发行职能都进行了重构。这些都是电影行业运作方式上的巨大转变。

家庭娱乐

家庭娱乐将电影带出影院，带入千家万户。一些家庭拥有能播放电影胶卷的设备，但直到 20 世纪 80 年代，VHS 录像带技术才使在家观影得到大规模的普及。电影公司不得不将大量的营销费用和制作经费用于推动家庭发行和录像带销售。在整个 90 年代和 21 世纪初都是这种情况。虽然 DVD 和昙花一现的蓝光取代了家庭录像带，但这些进步并没有使电影公司从根本上改变他们的运营方式。但家庭娱乐的出现显然做到了。

特许经营狂潮

2003年，《加勒比海盗》取得出人意料的巨大成功后，迪斯尼高层嗅到了特许经营内容的机会，开始为发展特许经营重新分配资源。两年后的 2005 年，鲍勃艾格出任迪士尼首席执行官，发展特许经营成为华特迪士尼公司的首要战略任务。为了适配新的战略方向，届时排名第四的迪士尼影业历经多次变革，包括三次大规模重组。迪斯尼还收购了皮克斯、漫威和卢卡斯影业来为新的发展方向提供支撑。整个行业都开始清楚意识到，要想持续成功，一定要发展特许经营内容，但没有其他影业公司像迪斯尼那样全面拥抱创新。结果就是，到了 2015 年，迪士尼已经成为第一大制片厂，没有任何竞争对手能够匹敌。

流量大战

虽然网飞早在 2012 年就从邮购 DVD 转型流媒体业务，但直到 2017 年，迪士尼和亚马逊才开始认真对待来自网飞的竞争──流媒体时代真正开启了。很快，华纳兄弟的 HBO Max、环球影业的孔雀（Peacock）、派拉蒙+ 以及无数其他公司接踵而至，都想从这个娱乐业的新领域里分一杯羹。

为了迎合流媒体创新，电影公司必须极大地增加内容产出，重新设计整个公司的发展方向和内容分发模式。营销负责人不得不掌握全漏斗营销的技能（如今尤其集中在受众留存方面）。流媒体还改变了电影公司的业务资金分配，影院票房和家庭娱乐业务的资金遭到蚕食。最后，流媒体还对公司文化产生了次生效应，公司需要吸纳并整合那些与现有员工拥有完全不同技能的人才。流媒体从根本上改变了一切，要适应新的娱乐业标准，就必需进行大规模的重组重构。

区块链是一个娱乐平台

02.

纵观今天的娱乐业，我们正处于下一个席卷整个行业的结构创新的前夕。Web 3 技术从根本上改变了内容的开发、营销和传播方式。几年后，它将在整个行业引发另一波大规模重组。由于区块链能够在一系列丰富的应用场景中发挥效用，它的影响堪称是革命性的。让我们来一窥究竟。

直接面向社区

区块链技术和 NFT 极大地降低，甚至消除了创作者和社区之间的界限。过去，创作者需要影业公司的资金支持，才能大规模地制作和发行自己的 IP。创作者依赖各种代理人，经由这些人向影业高层或其他守门人推销他们的创意。最终，由后者决定其创意是否有足够的价值，是否值得花费重金推向市场。

随着区块链技术的出现，创作者开始绕过陈旧的流程和传统意义上的守门人。他们可以将自己的 IP 作为 NFT 直接出售给消费者。长久以来横亘在故事讲述者和世界之间的障碍被消除了，新一代创作者能够自己建立欣欣向荣的粉丝和支持者社区。

以创作者为中心的发行

创作者意识到，自己的成功不再需要影业公司。一些人选择直接向社区兜售内容，并且获得了成功。因此，娱乐业将被迫改变发行业务的模式。直到目前，传统流媒体服务对创作者的盘剥仍然非常严重，所有交易中的大部分利润都落入了流媒体服务商的手中，IP 创作者分得的收益非常有限。创作者开始察觉 Web3 的价值，认识到自己能够拥有前所未有的权力。

Feature 等公司正在让天平倒向创作者一边，让等式重获平衡。通过 Feature³，我们致力于发展能够让创作者的作品大放异彩的合作伙伴关系，确保更公平的报酬，提供完全透明的数据分析，并帮助他们打造独特的沉浸式内容体验。

创作者社区的关系

没有了守门人，创作者与社区直接连接在了一起，这种变化催生了新的、更高层级的责任和交流。像威尔史密斯这样的演员，通过与粉丝的接触和交流，加速推动了事业的进一步发展。汤姆克鲁斯和德维恩约翰逊都曾坐在影院的后排，体会粉丝对他们表演的感受。但是，由于 Web2 传统社交平台功能的限制，即使社交媒体活跃度很高的创作者也没办法得到直接的反馈。Web3 提供了全新的互动基准，以及审视机制。Discord 之类的 Web3 社交平台允许他们与粉丝直接沟通，对话变得轻而易举。

传统的娱乐业模式通常让批评者保持在一段距离以外，他们只能「抛出」所谓烂番茄来表达不满；而 Web3 将迫使创作者正面回应反馈。这是一个重大的变化，它将促进创作者和Web3 专家之间的伙伴关系，让他们在这样一个崭新的领域里携手共进。
