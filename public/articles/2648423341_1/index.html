SeeDAO | NFT为何昂贵？它的价值源于何处？

作者: 翻译公会Queeny 发布时间: 2022-11-03 15:47:21
分类: NFT
本文通过各种详实的例子深入浅出地从稀缺性、艺术性、效用性、永存性这些维度来解释 NFT 的价高的原因，让人们看到万物皆可 NFT 并通过 NFT 来重新定义价值新的可能性。

"本质上，NFT 就是代币化了的 JPEG 图片。如果你想知道，过去几年中它怎么会卖到数百万美元，大多数人会轻描淡写地告诉你：因为它们是艺术，艺术本身就具有价值。"

确实。很多人想得就这么简单。

但事情并非如此。在公开市场上，我们看到像 NBA Top Shot 和 头像类 (PFP) NFT 等数字收藏品（如 CryptoPunks 和 BAYC ）的价格高得惊人。而他们中的任何一个都不是真正意义上的艺术品或艺术。那么问题来了：究竟是什么让 NFT 变得价值连城？

一、稀缺等式

See DAO

无论是艺术品还是数字收藏品，有一个因素肯定会推高 NFT 的价格：稀缺性。那么 NFT 的稀缺性与猿猴小图片有什么关系？

为了回答这个问题，让我们先来上一节经济学入门课。跟着我看下去——我们会让这节课通俗易懂的。

经济学领域期望理解和解决的核心问题之一是：在一个资源有限的世界里，如何有效地应对人类近乎无限的欲望。简单地说，就是平衡供需。

一般来说（此处着重强调＂一般＂），供给和需求两股力量的相互作用会影响到一项指标，也是消费者最终会感知到的指标：价格。在某个市场中，即使一个商品的需求强劲，但如果它供给充足，那么在此市场中该商品的价格就会相对较低。

反之亦然。当对特定商品的需求远远超过供给时，它就会成为稀缺商品，在相应市场中就能获得更高的售价。

为了真切地说明这个概念，让我们来看看为什么在日本水果被广泛视为一种奢侈品。岛国日本多山的地形几乎没有给大规模农业经营留下空间。因此，正如 Paku 在一份Wa-Shoku 报告中提到的，那里的大多数果园要么是＂家庭经营，要么由小型企业经营＂。
然而，日本水果产业在规模上不足，却在质量上得以弥补。通常，在典型的日本超市货架上，你会看到唯美的草莓、葡萄和西瓜，它们都是作为礼物送给＂亲戚、商业伙伴和客户，以感谢那些关照你的人＂，Paku 说。
好吧，那么，是什么使 NFT 如此有价值？

二、NFT 及其稀缺性

See DAO

NFT 可以非常昂贵（或很有价值，取决于你问谁），是因为它设法做到了不可能的事：将稀缺性引入全球数字市场。NFT 不仅仅意味着豪掷千金买一张图片——它也给了你所有权。

购买 NFT 时，你不只是购买图像本身。你购买的是刻在区块链上，指向该特定的图像或数字资产的永久代币。并且，万物皆可 NFT。除了艺术品，音乐、游戏内资产、虚拟土地，还包括其他各种数字商品，通常都可在NFT 交易市场上进行买卖。
如何通过将某物铸成 NFT来影响其价格呢？每当你 将一个数字商品铸成 NFT ，它有效地让你限制其供应数量，从而 "抬高 "其价格。尽管这听起来像是人为抬高数字商品价格的捷径，但这并非 NFT 的全部。
那么，NFT 还有哪些特性使其更具价值？

艺术性

首先，是艺术。在艺术领域，大师的作品的公开市场价高达数百万美元。对于张三李四等普罗大众来说，艺术作品高昂的价值也同样难以理解。以 Mark Rothko 的许多作品为例，在艺术市场上的售价达数千万美元。表面来看，其实很难理解为什么一块画着橙色、红色和黄色的画布会价值近 8700 万美元。

Mark Rothko作品 图源：WikiART

简单地说，价格高是因为它是现存的同类画作中唯一的一幅，也因为它是由 20 世纪最杰出的艺术家之一创作的。

这就是为什么迄今为止我们所看到的最昂贵的 NFT 交易中，绝大多数都是独此一件的作品。就像 Rothko 的《橙色、红色和黄色》，这些特殊的 NFT 是同类作品中的唯一。又比如 Beeple 的 Everydays，该 NFT 系列在 2021 年售出近7000 万美元 ，这些作品也确实是独一无二的。Everydays 收录了 Beeple 超过 14 年创作的5,000 件独特作品，它们是这位数字艺术家以每天一首的方式努力创作的。

效用性

有时，NFT 拥有远超艺术品或收藏品的广泛应用范围，因此获得高价。归功于区块链交易内在的安全性，NFT 被广泛用作筹集资金的手段。

从我们目前见过的最昂贵的 NFT 交易之一可以中观察到这一点。2022 年初AssangeDAO 成功安排了数百万美元 的 Pak's Clock NFT 售卖活动，该交易（截至发稿时）是有史以来第二昂贵的 NFT 交易。

为解救澳大利亚编辑和社会活动家朱利安·阿桑奇而设立的NFT

图源WikiArt

确切地说，该 NFT以 16,953 个 ETH 价格售出，当时价值约为 5,270 万美元。销售所得计划用于资助 WikiLeaks 创始人 Julian Assange 的法律辩护。在该例中，NFT 的效用——要用来做什么，是让它如此有价值和昂贵的原因。

永存性

最终，NFT 也可为全球互联网用户提供货币以外的价值。通过限制数字商品的供给量，帮助 NFT 艺术家和项目负责人为他们的数字商品注入了稀缺性，同时，区块链也可以为这些物品的无限期供应发挥作用。
传统上，数字商品的供应几乎是无限的。游戏、书籍、音乐和电影的数字拷贝理论上可以无限次购买。你永远不会看到数字商品缺货——但其实不然，某个时候，它们也会停售。当数字市场被关闭时，人们就再无可能获得某些纯数字化的商品。
让我们以任天堂为例。
该公司对盗版和仿冒品极力抵制的态度可谓是声名昭著。这家日本游戏巨头，致力于消灭每一家存有它已经停售的游戏的网站。不幸的是，这使得对该公司海量游戏的保存及防丢失变得极其困难。
有了 NFT，只要区块链存在，链上的所有资产就也能得到留存。随着区块链技术不断发展演化，我们也将不断看到新的 NFT 类型。也许有一天，你将能够拥有你最喜欢的电影的数字母版——当然，它可不会便宜。
