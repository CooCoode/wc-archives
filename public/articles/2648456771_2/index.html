Cursor AI 编程共学

给你的创造力加上杠杆

Cursor 是一款基于 AI 的编程工具，旨在帮助开发者更高效地编写代码，它提供智能提示和代码自动补全功能，可以简化开发流程，尤其适合初学者和有产品想法的创意份子。

有经验的开发者可能用惯了 VSCode & Copilot 组合，但 Cursor 的出现，给了不会编程的人一个崭新的工具利器，网上也有越来越多人展示自己 0 基础用 Cursor 编程的故事。

图 / 八岁小女生用 Cursor 编程

Cursor 适合有很多产品想法、明确自己想做什么的人。如果你对自己的品味有信心，享受解决问题、学习新事物的过程，并且不怕代码运行中爆出的 bug 警告，那么 Cursor 就是个不错的选择。

当然，培养系统性的工程师思维确实是个挑战，但也可以通过多做多练来克服，这样用 Cusor，就像拥有了一个不厌其烦的老师，以及一个如臂使指的工程师。

← 左滑查看 Cursor 速写个人网站，来源：https://raceli.art

AI 工具经常被称为「超级个体的杠杆」，会帮助有想法、有动手能力和解决问题意愿的人，它就像个乘数，但你也会看到，如果个人能力不大，10 和 1000 分别乘以 20，结果的差距只会更大。

因此在旅程开始之前，我们希望你对自己有充分的耐心，接受学习过程中陡峭的攀爬，目标导向地学习和完善自己，避免陷入无尽的细节泥潭，一起把 Cursor 用到最妙处。

共学介绍 / Outline

四周时间，共学如何利用 Cursor 开发应用，基于 React 和 React Native 移动应用开发技巧，以构建流行应用的核心功能为例，实践打磨自己的应用作品。

每周两场分享直播，周中周末 20:00（GMT+8）各一次，分享有回放，大家根据自身情况参与学习。在社群中，我们也会定期答疑，鼓励充分交流。

在这个过程中，我们希望和大家一起建立产品的思维逻辑，以及解决问题的方式，比如通过 AI 来分析你想实现的产品原型中技术难点要点是什么，至少需要掌握的相关知识，而不仅仅只是代码的编写。

最后一周组队或独自完成 Cursor 项目开发，主题自选，发起人和社区 Maintainer 都将积极协助，完成在此刻胜于完美，希望大家在四周的线上黑客松里都能有 takeaway。

关于共学About Collective Learning

共学没有一个权威定义，它是助人者、行动者领域的热词，强调我们的「共同学习」，是参与者共同选择对话主题、采取行动并对全过程进行反思的实践。 

共学倡导学习的非个人性，意味着我们需要就获得的信息开展交流与切磋、需要在对话中向彼此的生命经验学习，学习不仅仅是知识的输入，也与个人成长、社会变革有着积极的交互关系。

模块三

11 月 02 日 - 11 月 08 日
