NFT 能为已故的艺术家做些什么?

作者: 翻译公会 weiwei 发布时间: 2022-07-06 15:19:36
分类: 艺术家, 大英博物馆, 数字艺术, NFT
NFT 收藏家，新一代的艺术遗产赞助商。

摘要：

在艺术界，拿已故艺术家的遗作进行再创作是个禁忌。NFT 的出现，不仅打破了这项禁忌，还大有蔚然成风之势。作者从大英博物馆推出的 NFT 艺术作品展，谈到了数字艺术、摄影等领域中，围绕已故艺术家的遗作 NFT 所创新的各种应用场景。同时也发出质疑：动荡的加密货币市场，和趋利跟风的炒家，真的能够成为珍贵艺术作品的看顾人吗？

插画：Nicholas Konrad / The New Yorker

上个月，我在大英博物馆宽敞的玻璃穹顶圆形大厅里漫步时，走进礼品店，看到了让我再次驻足的东西。 在纪念品雨伞和明信片中间，有一块印着二维码的牌子，上面写着＂参观我们的 NFT 商店＂。 在衣帽间附近，我看到一些玻璃橱窗，里面陈列着 19 世纪日本著名艺术家葛饰北斋的版画。其中一幅画的标签上写道，这幅 1833 年的木版版画已被制成了非同质化代币（NFT）。2022 年 1 月，一位化名 pixeldrip.eth 的收藏家购买了它，仅花费了四千多美元。标签上还写着：＂扫描查看我们最新的商品。＂大多数访问者可能不知道这些术语是什么意思，但不必介意。在世界最著名的其中一个博物馆里，为何会显耀地展示着一位江户时代艺术家的数字复制品，仿佛它们本身就是有价值的艺术作品?

大英博物馆的 NFT 展览是与巴黎一家名为 LaCollection 的公司合作完成的，这间公司由Jean-Sébastien Beaucamps 于 2021 年初创立。Beaucamps 向博物馆提出合作，并赢得了一份为期五年的独家合同，从博物馆的藏品中制作 NFT；很快他们也将宣布与一家美国机构的合作。比起其他 NFT 市场，LaCollection 旨在提供更具策划性的展览。＂我认为 OpenSea 之类平台体验更像是 eBay＂Beaucamps 告诉我，并补充说在售的 NFT 还包括J. M. W. Turner（$1,000 - $9,400）和 Piranesi（$520-$2,100）的作品，＂（我们）把博物馆的体验延伸到了网上。＂当然，大英博物馆已经把自己的艺术品制作成了商业复制品——这就是礼品店的作用。但 NFT 创造出的光环：真品和稀有，将这些产品归入了不同的类别。据大英博物馆的一名发言人说，＂某种意义上，每一个 NFT 都是独一无二的，因此与海报或 T 恤之类的东西非常不同。＂ 

毫无疑问，葛饰北斋从未创作过数字艺术。J. M. W. Turner 和 Giovanni Battista Piranesi 也没有，后者以古罗马主题的铜版画闻名，于 1778 年去世。然而，通过大英博物馆的展览，借助拥抱加密新贵们对 NFT 收藏的狂热，所有这些都成了利用艺术家遗作的新努力的一部分。去年，安迪・沃霍尔原创的数字艺术作品被制作成 NFT 在佳士得拍卖。今年 1 月，毕加索的一个 NFT 项目导致这位艺术家的后代和他的遗产公司之间结怨。这些时髦产品可能会开辟出新财路，或至少会吸引新的关注。由于疫情的影响，大英博物馆推出 NFT 时能够进馆参观的人极少。＂他们意识到 NFT 是完美的营销工具，可以抵达那些迄今尚未接触到的群体，＂Beaucamps 说，并补充到，与艺术博物馆的常客相比，NFT 的受众＂年轻得多＂、＂男性多了一点＂，也更加国际化。博物馆从每一笔 NFT 交易中抽取佣金，并从二级市场的交易费中抽成。大英博物馆拒绝对迄今为止获得的收益发表评论，但它计划在今年晚些时候推出一系列新版本。

在艺术界，拿已故艺术家的作品来创作新品通常是禁忌。艺术家去世后出版的遗作比其生前亲自监督的作品卖得要便宜。大英博物馆曾推出过不同＂稀有度＂的 NFT 版本，最高可达＂极稀有＂，但这样的标签与原始艺术作品的格式或供给无关。与此形成某种对比的是，美国画家 Lee Mullican 的遗产管理公司的某个项目，试图利用 NFT 来包装数字原生艺术。去年，遗产管理人 Cole Root 决定出售一些数字作品的 NFT，这些作品是 Mullican 在上世纪 80 年代用电脑软件创作的，但他一生中从未打印出来让他满意的作品。＂不难想象，他一定会为自己的工作能穿越网络屏幕而开心，＂Root 这样评价 Mullican。Root 与一家名为 Verisart 的 NFT 公司合作，该公司曾与 Roe Ethridge, Quayola, 和 Rob Pruitt 等艺术家合作。将这位已故画家晦涩的数字作品作为区块链艺术品发行，符合某种逻辑，这是其他一些围绕遗作的努力所缺乏的。Mullican NFT 的价格为 1 以太币（译注：发文时约合 1250 美元），附有所有权合同、真品证书和一份作品的源文件（尽管技术上不能确保这些材料在转售时仍与 NFT 艺术作品保持关联）。Root 告诉我，他认为 Mullican 的 NFT 并非艺术作品本身的迭代，而是＂包装艺术̱品的画框或板条箱＂。

Julian Sander 是德国摄影师 August Sander 的曾孙，他的看法更加尖锐。＂我反对那种认为 NFT 本身具有价值的想法，＂他说。＂它一文不值，其实只是一个信息书签。＂尽管如此，今年 2 月 Julian 开始了一项相当于数字化集体存档的实验，赠送了一万份由他的祖先拍摄的印相制作的 NFT。August Sander 出生于 1876 年，于 1964 年去世，他最著名的作品是类型肖像巨著《二十世纪的人》。这些 NFT 是 Sander 家族为整理艺术家的底片作品而制作的实物印刷品的图像，并经过了几代人的注解。Julian 在科隆经营 Julian Sander 画廊(Galerie Julian Sander)，展示摄影作品，但从年轻时起他就一直是一名程序员，为艺术品收藏开发库存软件。他在 2020 年注意到 NFT 热潮，并开始与一家名为 Fellowship 的摄影NFT 组织合作。这项技术＂提供了一个类似维基百科的机会，将不同来源的信息整合到一个永久公开的地方，＂Sander 告诉我。＂我的目标是把 August Sander 的历史放到一个人人都能看到的地方，因为我的家人已经研究了四代。”

《马戏团大篷车里的少女》(1926年) August Sander 摄

《年轻的农民》（1921年）August Sander 摄

从字面意义上说，NFT 只是存储在去中心化记录上的一行代码，指向特定的图像或其他资产。但 NFT 也附加了可以永久存储在区块链上的元数据。August Sander NFT 上的元数据包括了一些信息，比如某张照片被收藏在哪个收藏集里、由谁洗印，以及最初被收录在艺术家的哪本书里。其结果便是艺术家遗产机构常常闭门搞出来的那种存档，但是是一种数字化的公开版本。某次发布会宣称，每一位 NFT 收藏者都是＂August Sander 遗产的看顾人＂。在 OpenSea 上，查看者可以滚动一格一格的印相，就像研究人员在筛选艺术家的档案一样。有清晰、构图严谨的肖像，也有朦胧、随意的风景和室内照。在照片的边缘有手写的笔记，标明了照片的拍摄对象、拍摄地点或拍摄的大致日期。底片上的胶片帧也清晰可见，提醒着人们真实照片的物理存在。对于 Julian 来说，大量的 NFT 是展示一系列作品的恰当方式，符合 Sander 希望能被人们从整体上认知的愿望。

到目前为止，August Sander NFT 通过转售的累计交易额已经达到了 400 以太币，约合 100 万美元，Sander 获得了 7.5% 的版税，Fellowship 获得了 2.5%。Julian 希望这样的二级市场能够驱动一个自我维持的机制，来保存 August Sander 的档案。他说，他打算继续在收藏中添加元数据，最终让 NFT 的收藏者能够兑换他们所购买的数字版印相的实体版本。他的项目将存档工作与动荡的加密货币市场绑定到一起，既是一项有趣的实验，也是一场怪异的赌博。正如我们本周所看到的，支撑 NFT 收藏活动的财富可能会迅速消失。同为加密货币的跟风人，能够成为像艺术这样经久不衰的事物可信任的看顾人吗？LaCollection 的创始人 Beaucamps 对此态度淡然：＂在艺术史上的许多不同时期，这种事情都发生过，＂他说，＂新一波收藏家就是新一代的赞助人。＂

Kyle Chayka 是《纽约客》的特约撰稿人，著有《渴望更少：极简主义生活》。
