本文根据 NFT 的价值驱动因素，将 NFT 分为收益类、身份类以及效用类三个类别；作者同时对三类 NFT 的价值来源进行了简要分析和介绍。文章试图提供一个可将各种 NFT 概念化的总体思考框架。

仅在 2021 年的短短几个月内，NFT 就成了一个家喻户晓的名词。第三季度 NFT 市场交易量激增，超过 107 亿美元。Visa 以 15 万美元买下了一个 CryptoPunk。昆汀·塔伦蒂诺宣布，将《低俗小说》中一幕未曾披露的场景作为 NFT 拍卖。塔可贝尔、汉堡王、麦当劳和必胜客都铸造了自己的 NFT。

显然，NFT 会长期存在。许多人对同质化代币的分类和估值进行了探索，却很少有人考虑为这个新兴的资产类别制定价值驱动因素的框架。

Andrew Steinwold 是 NFT 领域公认的思想领袖，同时也是 Sfermion 的创始人兼一般合伙人。他在 Zima Red 上撰文，根据价值驱动因素将 NFT 资产分为收藏品、游戏资产、虚拟土地、加密艺术等诸多类别。我认为，他为探索资产类别提供了非常有价值的分类方式。我希望能更进一步，通过对 NFT 价值驱动因素中重叠部分的探索，来简化他的分类。我认为基于价值来源，主要有三种类型的 NFT：收益类、身份类以及效用类。

首先是收益类 NFT，顾名思义，它具有产生收入的独特属性，与金融资产类似。而它们的＂价值＂也非常相似，即：预期能够产生未来现金流的资产的现值。该类别包括从游戏内资产到虚拟土地等一系列 NFT。通过在「边玩边赚」类型的游戏中玩耍，例如 Axies，游戏内资产可以产生收入──这是加密货币生态系统中一个新兴但正在迅猛发展的领域。许多人对该模型长期可行性的看法仍有分歧，而 Yield Guild Games（一个游戏公会 DAO）通过汇集游戏内的 NFT 资产，从 6 月到 9 月共产生了 112 万美元 游戏内奖励。显然，这些 NFT 带来的收入非常可观。

虚拟土地同样也是收益类资产。虚拟世界通过算法推出了一系列可供建造和定制的土地，或者，在某些实例中可供出租。另外一些世界则在土地上随机生成稀缺的自然资源，可以通过耕种或采矿来获取收益。Decentraland 在销售土地类 NFT 的领域中出类拔萃。本质上，虚拟土地能够产生收入这一特性模仿了地产实体。

最后，开发者正在使用 ERC-721 代币设计新的实验性金融产品。Charged Particles 之类的项目允许玩家在 NFT 中嵌入 Aave 上能够生息的 aTOKEN，本质上将这些 NFT 变成了独一无二的、可定制的固定收益产品。

第二类是身份类 NFT，旨在通过昂贵物品来彰显社会阶层，它们是实物商品（尤其是服装）高端品牌的数字等价物。这个类别无所不包，有：数字艺术、收藏品、服饰配件、铸造曲目、PFP 等等。界定身份类 NFT 的关键标准是，它们可以在网上公开展示，特别是通过社交媒体进行展示。其价值来自于藏品或艺术家的＂炒作＂或＂影响力＂。这个类别显然比收益类 NFT 更抽象，以致于很多人都在质疑它们的价值。毕竟，我可以从 Opensea 上复制粘贴 JPEG 图片，对吧？有个例子最能说明身份类 NFT 的价值：一些很吃香的品牌，比如 Supreme，为自己的商品收取高昂的溢价，高到明眼人都看得出来──标价减去生产、运输和销售的成本。我可以用极低的价格去随便买一件便宜的白 T 恤，在上面印上个 logo，但如果被人发现了，就会被视为＂假货＂或 ＂冒牌货＂。人们购买的不仅仅是那件实际的衣服，而是这个牌子，更重要的是它所代表的社会地位。此外，因为交易在链上公开，所以验证所有权只需要一个钱包地址。许多身份类 NFT 都具有稀缺性，也会持续供应，高端品牌在发布和推广新品时可以随意生产。

通常，这些 NFT 又分属两个大类：社区型和独立型。首先，在某个被认为具有排他性的社区中，社区 NFT 意味着成员资格，拥有该 NFT 等于有了张＂入场券＂；此类之中名声最响的有 CryptoPunks 和 BAYC。第二种，独立型 NFT 的价值则来自于艺术家（或干脆是铸造者）的声誉。例如，Beeple NFT 不提供 BAYC 那类社区，但确实赋予持有者社会地位（他现在甚至以类似价格出售实物艺术品）。这些 NFT 反映了艺术实物的高昂价值。

最后，效用类 NFT 是一个新兴的类别，具有真正革命性的潜力。简单来说，这类 NFT 的价值（或者净收益）就是资产对用户的效用减去其成本。更多常见的例子比如域名、游戏内的非收益类资产等等。而部分游戏内资产可以被归入效用类 NFT，最关键的区别是：除价格升值之外，没有人期待效用类 NFT 会产生收入，它们之所以存在需求仅仅是因为其可用性。一些交互式 NFT 的实验将 AI 整合到 NFT 中，创造出一个用户能够与之互动的独一无二的数字角色；换句话说，数字人格可以完全去中心化，不被任何一家公司拥有（试想：不受苹果公司控制，由个人公开拥有的独立 Siri）。

The first iNFT, Alice (based on Alice in Wonderland), sold for $478k.首个 iNFT，Alice（源于《爱丽丝梦游仙境》）以 47.8 万美元的价格售出。

该领域的先驱 Alethea 最近在 OpenSea 发布了＂Revenants＂——智能 NFT（iNFTs），在不到一周的时间里就跻身前十。该藏品包含 10,000 名历史人物，从埃及艳后到科学怪人、弗洛伊德，均使用他们的生活和经历的公开数据进行训练。人工智能嵌入到 ERC-721 代币中，使角色具有了互动性和生命力。很疯狂吧？也有一些更为轻松愉快的交互式 NFT 的例子，比如 Cryptoys 这样的项目——想象一下，生活在以太坊上会教孩子东西，能和他们互动的数字毛绒动物……

NFT 无疑是最令人兴奋、最具实验性的资产类别之一。本文的目的是：按照收益类、身份类和效用类（或组合）的分类方式，为如何概念化任意 NFT 提供一个总体思考框架。
