在经历过2022年的瓦猫之夏后，游民Buidler们相聚清迈共建未来！

这一次在清迈，不仅有瓦猫，也有黑客松！更有其他瓦猫活动！

一年一度的 Rebase Hackathon 即将开始！

Rebase社区将于2023年12月18日举办一场以「建设DeSoc去中心化社会」为主题的黑客松。此次黑客松将采用线上+线下结合的方式，并在举办形式上作出新的尝试！

DeSoc概念于2022年5月提出，通过SBT、Zuzalu、山海坞、MuChiangMai等形式，大家对DeSoc进行不同路径的尝试，在这过程中诞生大量新的生活方式、新的应用形态和新的发展思路。

在2022年的夏天，许多Buidler决定开始一段新的旅程，踏上一个未知的探索之旅。他们离开熟悉的环境，追寻着广阔天涯的自由与冒险。在这段时间里，他们探索了新的技术、理念和社群，结识了志同道合的人，并与他们一起分享着对于去中心化社会的憧憬和追求。

在这个冬天，他们将再次聚集在一个温暖的地方，带着对于过去经历的骄傲和对未来的期望。这次相聚将成为一个特殊的时刻，他们共同回顾自己的成长和发展，分享着彼此在探索DeSoc的过程中的收获和教训。

这次聚会也将是一个新的起点，大家共同探索着DeSoc的新方向。他们将讨论如何进一步推动去中心化社会的发展，探索创新的技术、治理和社会模式。他们汇集智慧和创造力，相信通过合作和共享，他们可以为建立一个更加开放、公正和民主的社会贡献自己的力量。

期待对未来充满希望的你，期待饱受现实煎熬的你，期待渴望突破自己的你，来到清迈，一起探索未来的无限可能！

本次黑客松将结合采用「线上和线下」结合的方式进行。线下地点在泰国清迈，线上通过在线会议软件接入。黑客松全体环节也将延续 Rebase 黑客松的过往传统，全程将进行线上直播，大家可以通过 Rebase 的相关社交媒体进行全程观看。

奖项设置

本次黑客松将不设立比赛名次奖金，转而设立为赞助方冠名奖项，奖项以及评选标准由赞助方自行设定并评选。以下是本次黑客松的奖项设置及要求。

SeeDAO

题目1：基于Web3账户的去中心化权限会议系统

题目2：基于永存网络的去中心化权限文本协作方案
奖金: $2000

JoyID

题目：基于 Telegram 的 Web3 应用

要求：使用 JoyID Passkey wallet SDK开发面向 telegram 生态和用户的社交应用，要求同时使用 JoyID 的便捷特性和 telegram 的社交属性

规则：创新性5分，社交性10分，web3特性10分，完成度15分   

Mask Network

题目：使用 Next.ID 开源身份图谱构建 Web3 应用

可尝试的方向：

 1、跨链个人主页和用户数据推荐

2、用户信用评分和防欺诈

3、DAO 社区工具，比如个人声誉系统

5、全链游戏用户排行榜

4、欢迎尝试你认为可行的场景…

打分规则：

我们将从提交质量、演示效果、安全与创新等角度进行综合评分，同时在比赛结束给出对项目和团队的全面反馈。

感谢以上赞助方对本次黑客松的大力支持！

如您希望成为本次黑客松的赞助方，请扫描下方二维码联系我们，共同筑建更好的Web3社区！

参与方式 

  • 采用自由组队形式参与

  • 建议每个队伍人数控制在6人以内

  • 报名截止时间2023年12月16日下午2:00

作品提交 

  • 2023年12月18日下午2点 作品提交截止

  • 未提交的项目可参加宣讲但不参与评选

  • 截止时间以 github 上提交的 PR 更新时间为准

作品展示

  • 可以部署在任意区块链上

  • demo可展示

  • 作品已在github提交PR

合作

关于 Rebase

Rebase 社区是由一群 Web3 开发者们基于爱好利用业余时间建立的非营利性开发者社区，旨在链接 Web3 开发者，建立沟通交流桥梁，助力极客促进科技创新，推动社会进步。目前，Rebase 社区有专注于不同领域的工作小组，Web3极客日报，Rebase大学，Rebase Hackathon，Who-Is-Hiring，Event-Pool，Rebase D. Ventures 和 Web3Camp 等。

如果你有兴趣参与其中，可以通过 Rebase 社区公众号同我们取得联系！
