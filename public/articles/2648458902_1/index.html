SeeDAO友推｜Rock Web5: 南塘 CodeCamp

作者: Rock Web5 发布时间: 2025-03-07 14:00:05
如何在开放与效率、抗审查与普惠之间，构建人与技术的未来？
在被金融泡沫和概念炒作充斥的加密行业里，我们看到越来越多的虚无主义情绪。日复一日的代币发行、流动性挖矿、高杠杆投机，我们是否已经忘记了区块链技术最初的愿景？它本应是一场关于自主权、透明度和真实价值创造的技术革命。
CKB Eco Fund 致力于回归中本聪白皮书的精神原点——在比特币的POW共识和UTXO模型上，通过RGB++协议、Fiber Network、USDI等工具，构建既能承载Web3抗审查性、又兼容Web2易用性的Web5新型数字基础设施，让技术Just For Fun，更好玩、更温暖，更属于每一个真实的人。
2025年3月16日，我们选择在中国安徽阜阳发起第一站Rock Web5运动，邀请全球开发者用代码回答时代命题——如何在开放与效率、抗审查与普惠之间，构建人与技术的未来？

1. Rock Web5：

回归真实有趣的创新

我们的理想是改变世界，虽然这听起来让人觉得在开玩笑，但我们是认真的 —— Jan Xie，Nervos 首席架构师

Web5 = Web2 + Web3：实用性与理想主义的结合

今天Web3陷入了认知与现实相左的困境。一方面，我们有着雄心勃勃的愿景，希望创建一个完全去中心化的网络社会；另一方面，我们看到的大多是金融投机和概念营销。Web3的理想主义与实用性之间存在巨大鸿沟。
Web5提供了一条务实而富有远见的中庸之道。它不是要完全替代Web2，而是将Web2的易用性与Web3，尤其是比特币的去中心化价值观相结合。真正的Web5架构需要建立清晰的系统边界，明确区分哪些场景需要完全的去中心化，哪些场景可以接受一定程度的中心化以换取更好的用户体验。
Web5不是非此即彼的选择，而是社会与技术同频共振的架构艺术，既保留Web3的抗审查价值，又不放弃Web2的普惠便捷，形成一种有机结合的全新技术范式。

Just For Fun，重焕技术的创新乐趣。

我们认为当前加密世界的困境源于伪需求泛滥。Rock Web5要打破这一循环，回归比特币白皮书描绘的真实应用初心：
  1. 探索Web2与Web3的融合边界：明确哪些功能需要完全去中心化，哪些可以保持中心化以获得更好的用户体验。
  2. 关注新兴场景与数字化结合：让开发者直面真实用户和需求，发掘隐藏在数字鸿沟下的巨大应用场景。
  3. 构建有意义的社区治理工具：帮助各类社区实现更高效、更透明的自治。
在我们的Rock Web5活动中，代码即土壤，技术即乐器，「Rock」是对异化的双重反抗：
  • Rock维度一：让代码重回本真和乐趣
    • 让我们重焕技术的原始快乐。我们追求的是在Web5的新范式下，开展能让开发者和用户感到兴奋的探索：比如将RGB++协议用于文化资产的数字化保存、利用Fiber Network构建链接人与AI、跨越不同场景的价值网络、或者探索RISC-V开源架构与AI的融合之路。真正有趣的代码是能激发想象的技术实践，而不仅是重复现有的模式。
  • Rock维度二：融合边界的Web5创新
    • 我们鼓励基于CKB提供的比特币生态工具（RGB++/Fiber Network/USDI）进行开发，但更重视如何有机结合Web2的成熟易用与Web3的抗审查性，创造出既解决实际问题又保持技术理想的Web5应用。理想的项目应明确划分应用的信任边界，将需高度安全保障的部分放在链上，同时利用现有Web2工具和AI技术提升用户体验，形成有机统一的整体。
未来，CKB Eco Fund还将在全球不同地区、城市，结合当地产业特点和人文底蕴组织系列Rock Web5活动。我们相信，真正的去中心化，是让每个人都能成为规则的设计师，都能成为Web5的演奏者。

2. 为什么首站选择南塘？

谈到乡村，为什么老想着种地？青年人其实可做的事情非常多 —— 温铁军，中国三农问题专家
南塘被称作「中国乡村DAO治理摇篮」，其26年的乡建史恰似Web5的预演。当城市的Web3从业者还在争论「该用Solidity还是Move」，南塘合作社的成员早已在实践中构建出混合架构的社会操作系统。
他们用罗伯特议事规则组织讨论（社区共识层），用区块链记录工分（Web3清算层），用微信群协调生产（Web2通信层）。这种多层级「土法炼钢」式的去中心化数字治理，恰恰是Web5所追求的实用主义路线。
选择南塘作为我们Rock Web5的第一站，因为这里蕴藏着契合Web5的三种文化基因：
  • 辩证思维：不迷信任何单一技术或理念，而是灵活采用能解决实际问题的方案；
  • 架构自觉：合作社已探索一套颇具启发性的多层架构，形成朴素的Web2+Web3系统雏形；
  • 务实创新：将在地需求与现代数字工具相结合，实现了一种可持续的发展模式。
在南塘这样的场域中启动我们的活动，能让开发者更直观感知技术如何服务于真实世界，从而激发出更多有意义的Web5创新。

评奖规则

1. 作品要求

  • 技术要求：

    • 参赛作品鼓励使用 CKB/RGB++/Fiber Network 相关技术，指导团队将一步步带你彻底上手Web3生态

    • 也欢迎使用各类Web2成熟组件

  • 开源提交：

    • 代码需上传至 GitHub 公开仓库，包含完整的 README 文档（介绍项目背景、功能、使用方式）

    • 代码需包含清晰的 commit 记录，避免一次性提交所有内容

2. 评审标准
  • 技术实现：结构清晰、可读性高、符合最佳实践；是否引入独特的技术方案，避免简单拼凑或照搬现有项目
  • 产品体验：是否具备良好的 UI/UX 设计；是否提供流畅、直观的用户操作体验
  • 创新性：是否提出新的思路或突破性的应用场景；是否有独特的功能实现，区别于现有类似项目
  • 展示效果：现场 Demo 是否清晰传达产品核心价值；现场答辩能否准确表达技术架构、应用场景
  • 有趣程度：作品是否具备一定的趣味性或吸引力，而非千篇一律的 Fork 项目；是否能够引发社区讨论或共鸣，提升项目传播性

5. 结语：用代码奏响Web5摇滚

当全球加密会议在五星酒店讨论「下一个牛市叙事」，我们选择在南塘开启对技术的重新思考，这里没有流动性挖矿的喧嚣，只有键盘与现实的交响；这里不追逐千倍币神话，只在乎一行代码能否创造真实价值，能否让数据更安全，能否让人们的数字生活更自由……
我们相信，真正的创新来自于对真实问题的关注和解决，来自于技术与社会需求的有机结合，来自于数字与实体世界的相互滋养。我们期待见证技术如何真正嵌入生活的肌理，如何通过Web5的思维重新定义人与技术的关系。
这不仅仅是一场CodeCamp，这是一场回归初心的旅程，一次重新发现区块链技术真正价值的探索，一场寻找Web5真实应用路径的征程。
