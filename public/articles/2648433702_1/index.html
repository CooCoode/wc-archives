最近我与许多 Web3 的从业者和投资者交谈时得知，Web3 的推广目前仍受制于基础设施的缺乏。人们普遍认为我们需要更好的钱包、Alt L1s(替代性的第一层解决方案，如 NEAR、Solana 等)、Rollups、基于 Rollup 的 Rollup(这里指的是基于 OP Stack 的 Base Chain)、多链桥、Web3 原生增长工具等等。但实际上，大部分基础设施团队面临的最大问题是缺乏足够的消费者团队来构建优秀的下游应用程序。五年前 (!)，USV 的 Dani Grant 和 Nick Grossman 在他们的帖子“基础设施阶段的神话”中提到：

“

在web3社区中，有一种普遍的观点认为我们现在处于基础设施阶段，我们需要构建更好的基础链、更好的链间互操作性、更好的客户端、钱包和浏览器。其中的核心思想是：我们需要能够让我们轻松地构建和使用运行在区块链上的应用程序的工具，一旦我们有了这些工具，我们就可以着手构建这些应用程序。

然而，在与那些正在构建基础设施的创始人交谈时，我们不断听到他们面临的最大挑战是如何吸引开发人员在这些基础设施上构建应用程序。

现在普遍的观点是，新的计算平台都始于一个离散的基础设施阶段，然后再形成一套应用程序生态系统。然而，在现实中，新的计算平台通常是从杀手级应用程序开始的，这些应用程序再反过来告诉我们需要构建的基础设施类型，然后启动“应用程序 <> 基础设施的周期”。
https://www.usv.com/writing/2018/10/the-myth-of-the-infrastructure-phase/
我相信这个框架已经解释了当前 web3 消费级产品和基础设施之间的微妙关系。我们需要更多的团队在消费者层进行构建，从而告诉我们需要构建哪些人们真正需要的基础设施。不可否认，更好的准入流程和更便宜的交易费用将催生新的消费产品，但着眼当前其实我们今天所拥有的基础架构已经足以创造一些引人注目的产品和体验。
例如，Reddit Collectibles 和 Nike .SWOOSH 这两个例子已经足以证明传统消费品牌同样可以在现有设施下打造爆品。与此同时，SoRare 和 Immortal Game 通过将现有产品（如 fantasy sports 和 chess）与 Web3 元素结合，提供独特的价值主张（拥有游戏内资产），并取得了一定的成功。
但事实是，在过去几年里，增长最快的 web3 消费产品主要是由投机驱动的，包括交易所、NFT 市场、钱包、游戏和 NFT 项目。然而，这个阶段已经结束（由于利率上升和其他宏观因素的影响）。投机并非总是坏事。事实上，我们通常需要投机者来激发对新技术的兴趣和投资。
然而，Web3 消费市场的未来需要构建引人入胜的产品和体验（我知道这是一个激进的想法）。随着时间的推移，大多数人可能不会意识到他们正在与区块链或代币进行交互。他们可能只会将 Web3 元素称为数字收藏品、积分、徽章、会员卡或其他任何更适合消费者的术语。

例如，以下是未来几年中会令我感到兴奋的一些产品和体验：

  • 数百万玩家收集和交易 NFT 游戏内资产的游戏

  • Web3 品牌和创作者与数百万 NFT 持有者建立社区
  • Web3 IP 通过媒体和娱乐产业进行文化传播
  • 由社区驱动的唱片公司，培养下一代顶级艺术家
  • Web2 品牌利用 NFT 构建的有趣、新颖和类似游戏的体验
  • 带有 NFC 芯片 的物理产品，可以通过扫描来解锁数字体验

新的基础设施无疑将有助于使这些体验更简单、更可扩展和更具有成本效益。但是，Web3 的未来取决于出色的团队构建的消费者最终会了解和喜爱的产品及体验。
