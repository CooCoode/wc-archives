本文对动态 NFT 的概念和基本特性进行了简要的介绍，并将其与静态 NFT 进行了对比，同时还介绍了动态 NFT 的一些经典用例。作者认为 2022 年是动态 NFT 爆发年，对其市场前景信心满满。

►►►

来势汹汹的动态 NFT

整个区块链世界都在为 NFT 疯狂。仅仅 1 年时间，静态 NFT 艺术就从散户和机构投资者手里吸引到了数十亿美元。并且它还将以每年超过 10% 的速度持续增长至 2030 年，有望为艺术家和收藏家带来更多收益。

尽管作为创新的利基市场它魅力十足，NFT 领域却仍然存在两个严重的问题。一个是证明 NFT 所有权的唯一性（没错，它们非常容易被＂复制粘贴＂），另一个是跨链转移 NFT。

以上两个问题都可以通过一个颠覆性创新解决：动态 NFT。本文将介绍动态 NFT 的概念并探索其对所有 NFT 爱好者承诺的前景。

►►►

静态 NFT vs. 动态 NFT

静态 NFT

  • 与链下环境孤立

  • 静态的；不能跨链转移

  • 无法对现实世界的事件做出回应

动态 NFT

  • 与链下环境互动

  • 可以跨链转移（例如在以太链和波卡链之间的 RareLink 链桥）

  • 动态地回应现实世界的事件

BAYC、CryptoPunks 和 The Sandbox 等静态 NFT 每天数百 ETH 的交易量表明，用户对 NFT 市场的未来抱有兴趣和信念。

但时代变化太快，如今 NFT 世界又现新品。动态 NFT 的到来将永远重塑 NFT 市场的格局。

OG: crystals 和 merge. 都是在加密市场上引发过轰动的动态 NFT。许多专家认为动态 NFT 将会在 2022 年成为市场焦点。所以，现在是时候来了解动态 NFT 创造的方方面面，一窥这个快速发展市场的究竟。让我们来快速看看两种 NFT 的区别，搞清楚什么是动态 NFT。

►►►

静态 NFT 的好处

2021 年 NFT 市场斩获了超过 180 亿美元的总交易额，它能如此蓬勃发展是有原因的。静态 NFT 是区块链资产的一种独特形式，具有永久不变的特性和静态记录的链上数据。区块链一直被称赞的正是它的不可篡改性，这种独特的媒介让用户存储的数据避免被操纵，用户间的交易也可以防止欺诈。

►►►

动态 NFT 的好处

动态 NFT 的智能合约具有流动性，使 NFT 可以随时间演变，来响应现实世界中的事件。该属性能够以不同方式加以应用，允许用户在不同的区块链之间转移 NFT，甚至改变他们对所有权变更的看法。从 P2E 游戏装备交换的通证，到现实生活中的幻想体育游戏，动态 NFT 也有多种用例。除此之外，正如 Solsea 上的动态 Shield NFT 一样，动态 NFT 可望具有可验证随机性函数（VRF）。

►►►

为何动态 NFT 如此重要？

随着 NFT 领域的发展，动态 NFT 代表了下一代的技术解决方案。动态 NFT 的 VRF 特性让真实性验证成为可能，并保证了游戏内道具分配的随机性。因此，在游戏中参加战役和活动时，P2E 玩家将享有公平、随机的稀有资产和战力排名分配。

动态 NFT 主要的好处是，它们能够根据经由预言机传递到区块链的真实世界数据的变化而演化。比如你可能拥有一张棒球手的 NFT 卡。一旦他们获得了比赛冠军，你的 NFT 卡也会根据现实世界传入的数据改变代码，从而显示出你的偶像的新头衔。

因此，动态 NFT 中强化的保真性让它能够在多个领域适用。例如，你可以用 NFT 代替护照，环游世界时不再需要文书工作和印章。所有的变化都反映在 NFT 的代码里，没有了作假或诈骗的机会。对于银行帐户、教育证书、保险以及任何需要输入新数据后立即更新的文件，都可以这样做。

►►►

改变 NFT 的智能合约

静态和动态 NFT 的核心是它们的 智能合约 代码，由你所使用的特定区块链启用。智能合约是一种自我执行的自动区块链组件，其中包含了触发特定行为的所有条件，或关于某个物件的基本数据。现如今，创建 Solana 动态 NFT 或在另一条链上创建 NFT 并不难，因为大部分工作都发生在链桥和预言机中。它们的功能是将动态 NFT 与外部环境相连接，并跟踪事关 NFT 代码、功能或外观的数据更新。因此，你的任务是与可靠的智能合约开发者合作并创建安全防错的 NFT。

FAQ

什么是动态 NFT？

简单来说，这种 NFT 的智能合约代码中，包含能够根据外部环境的输入进行演化的特性和数据。例如 OpenSea 上的动态 Duelers NFT 合集。该特性对于通过预言机和跨链桥转移 NFT 至关重要，因为基于区块链的静态 NFT 不能在链下使用。因此，从静态到动态 NFT 的转变有助于实现更好的跨链互操作性。

如何创建动态 NFT？

很多有远见的 NFT 粉丝和艺术家都在操心＂如何创建动态 NFT＂。你可以寻找动态 NFT 制作的线上教程，或者直接从区块链开发者处获取个人首选链的动态 NFT 智能合约源代码。你可以找到 Solana 动态 NFT 或 Ethereum 动态 NFT 手册，由此开启业务，并在热门市场上发布可盈利的系列。

动态 NFT 项目有利可图吗？

随着行业持续飞速发展，NFT 动态艺术的欣赏者越来越多。试想：NFT 的市场规模在 2020 年只是略高于 8000 万美元，而在 2021 年就超过了 170 亿美元。在短短一年内就猛增超过 21,000%。因此，对于动态 NFT 及其粉丝来说，2022 年将会非常有料。由于市场对这一新趋势兴趣上升，如果你创建动态 NFT 合集， 2022 年将会从中大有斩获。

如何让一个静态 NFT 动态化？

总的来说，已完成创建并部署到 NFT 市场 上的静态 NFT 是不可转变的。这意味着它们永远无法变成动态的 NFT 艺术品。这些最终作品旨在以当前形式收集和交换。你只能分解整个 NFT，然后使用动态智能合约代码重构，使其能够在链下数据更新时发生相应变化。查看 OpenSea 上的动态 NFT 作品，了解它们的样子并跟随市场风潮。
