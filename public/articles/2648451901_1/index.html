“如何将优质内容高效的传递给接受信息的个体？”

作者 | Chauncey Liu

投行从业者/web3老韭菜

引言

人类社会的发展，伴随着的是社会结构的日趋复杂和各类信息爆炸式的增长——这两者共同带来了一个难题：“如何将优质内容高效的传递给接受信息的个体？” 

在刚刚孕育出文明的原始时代，人类将重要信息口口相传（例如哪种草药可以治疗哪些疾病）； 

在步入封建时代的初期，人类通过兽甲、竹简等媒介记录内容，并在小范围内传播（例如逐渐形成的礼仪规范、简单的事件记录）； 

在专制帝国鼎盛的时代，人类通过书籍、布告栏等方式，能够将“合法的”内容传递给士人阶层甚至一部分平民百姓（例如皇帝的旨意、大家的著作、民间的故事等等）； 

在工业革命后的资本主义时代，人类通过发行杂志和报纸可以传递丰富度和时效性均有显著提升的内容，受众群体也大幅拓宽，社会上出现了一批能够单纯依靠产出内容为生的群体（作家、记者）； 

在如今的信息化时代，随着互联网的出现，几乎每一位人类个体都成为了重要的内容消费者，也有一大批人依赖产出内容和传播内容作为谋生手段。“内容平台”正在日新月异的发展，“创作者经济”的模式也在快速革新。

如果说DAO是人类在社会组织形式、治理与协调方式上面的一次伟大实验，那么探索适应于DAO组织形式的内容平台和创作者经济模型，也是这场革新的重要组成部分。

01

我们为什么需要DAO创作者？

1. 互联网的发展方向背离了其建设者最初的理想

生活在21世纪见证了互联网的红利期，我们可以感受到，随着互联网的快速渗透，各类内容产出爆炸式增长，各类内容平台层出不穷。但是越来越多的人觉得互联网不再“纯粹”，其发展已经偏离了互联网建设者最初的理想。

正如万维网之父、图灵奖得主Tim Berners-Lee 曾在演讲中提到，他认为：最开始的互联网像是一个独立于物理世界的乌托邦，是自由开放的大社群。而经过30余年的发展，目前垃圾信息已经充斥了互联网，普通人无法完全控制哪些信息可以进入自己的视野。 

 回到我们一开始的问题“如何将优质内容高效的传递给接受信息的个体？”解决这个问题的首要条件是要存在优质的内容。但是在当今社会，恰恰有很多因素局限了创作者生产真正的“优质内容”。

2. 现有创作者经济的局限 

在目前的互联网平台上，可以把创作者大致的分为两类，即“个人创作者”与“机构创作者”。其中机构创作者绝大多数是传统的公司创作者，虽然红十字会等非营利组织作为机构创作者也会产出内容，但这部分内容涉及的领域较窄，且占比很低，因此不在此文做过多讨论。对于个人创作者和公司创作者而言，两种模式各有其优劣势：

3.DAO创作者为何能改进现有问题

DAO 作为有一定共识的去中心化组织，能够一定程度上解决个人创作者和公司创作者存在的问题。 

首先，DAO参与者不止一人，并且从目前看来，大多数DAO的参与者背景丰富，很可能在日常生活中从事不同的主业（如研究员、运营、商务、律师、学生等等），背景丰富的一群人通过 DAO 组织协作使得创作内容可以在深度和广度方面相较于个人创作者均有显著的提升。 

其次，DAO 参与者拥有一定的共识，一定程度上享有共同的利益，并且这种利益很可能并非金钱意义上的利益，而可能是怀揣改变某领域的宏伟梦想、希望解决某项问题而达成的共识、甚至只是寻求一个能够充分发挥主观能动性、实现自我价值的平台。正因如此，DAO 与公司不同，在创作方面可能更接近于非营利组织，但又比后者更加灵活。哪怕DAO组织中有部分参与者（甚至是DAO发起人）出现了想法变化，不想再继续遵守DAO的共识，DAO依然可以正常运行，最差的情况也不过是社区分裂，原有的DAO依然存在。 

最后，DAO 和 DAO 参与者本身也需要 DAO 创作者经济。一方面，很多DAO 创作者还是希望通过内容变现，或许是作为全职创作者需要经济回报养家糊口、或许是作为兼职创作者赚取一定的收益、或许是仅仅想通过内容变现实现一定的精神成就感。另一方面，DAO 自身也需要创作者经济的加成，或许是充实DAO国库的需求、或许是通过创作的方式增强DAO参与者的协作增强组织凝聚力、或许是通过DAO创作行为增强DAO组织的知名度。

02

DAO创作者面临的问题

虽然DAO创作者在一定程度上能够打破现有创作者经济的局限，但同时也面临着创作者经济的问题：通过去中心化协作的方式产出的、并非以盈利为核心目的的内容，是否能够通过平台变现？

创作者端的问题

DAO创作者相较于个人创作者、公司创作者同样有一定的劣势，具体包括： 

1) 去中心化协作的模式下，创作的效率相对公司创作者更低；

2) 去中心化协作模式下，内容产出质量相对不易控制； 

3)在去中心化协作的情况下，DAO内部对于内容收益的公平划分有一定的难度。 

上述存在于创作者端的问题其实涉及到了人类社会面临了很多年的一项难题，即：“如何兼顾公平与效率？”这个问题同样也是DAO治理核心要解决的问题之一。但鉴于此问题太过宏大，而本文主要仍聚焦于内容平台与创作者经济的问题，因此不再花大量篇幅展开讨论。本文需要重点讨论的是：即便DAO创作者能够克服上述创作者端的内生问题，也依然面临以下来自平台端的外部问题。

内容平台端的问题

1. 所有创作者都受制于平台的分发算法 

绝大多数内容平台，都有一套复杂的推荐与分发算法。创作者作为算法的被动接受方只有两种选择：

1）无视算法的存在，以自己原本的想法进行内容创作。这种做法可以保持内容的纯粹性，但可能因为不迎合算法的喜爱而无法被高效传播； 

2）摸索算法的偏好，并相应修改内容（大改的情形如蹭热点借题发挥；小改的情形如标题党）。这种做法通常可以有效提升内容的传播效率，但可能或多或少的扭曲内容创作者原本的意思。 

2. 平台帮助创作者内容变现是一个长久的难题

DAO 创作者面临着和其他创作者一样的问题，即：内容平台如何帮助创作者变现？目前，创作者经济获取报酬的方式可以根据平台是否起到直接作用而划分为两类：“直接变现”和“间接变现”。 

1）直接变现模式（平台能够直接为创作者带来收益）主要包括： 

 ① 签约费用（多适用于知名创作者，平台邀请其入驻，或一次性、或按创作篇数、或按年支付报酬）

 ② 平台奖励（根据点击量等指标由平台方直接给予的报酬） 

 ③ 平台广告（通常为创作者可以自愿选择将广告链接插入自己的内容中，并根据一定规则获取报酬）

 ④ 付费订阅（读者对全部或部分内容、按月或按篇付费阅读） 

 ⑤ 付费咨询（读者向创作者支付报酬提问）

 ⑥ 读者打赏（读者自愿向创作者支付的报酬）

 2）间接变现模式（平台仅作为“平台”使用，不直接提供变现手段）主要包括： 

 ① 引流销售（内容作为吸引潜在买家的手段，使读者购买创作者提供的商品或服务） 

 ② 外部广告（根据外部广告主的要求在内容中植入软广或硬广，并从广告主处取得报酬） 

 ③ 其他变现手段（创作者将内容仅作为增加自身曝光度的手段，不通过内容而是通过其他手段变现，如参加商业活动等） 

考虑不同的写作目的和预期的变现方式，不同的DAO创作者会选择不同的平台发布内容。经过本次研究，我们认为部分web3内容平台已经提供了超出上述变现模式或一定程度上优化算法机制的功能。

03

DAO创作者常用的内容平台

各类不同的DAO组织倾向于在不同的平台进行创作，例如国人很熟悉的微信公众号、Twitter等。但除了这些常见的创作者也会选择的内容发布平台以外，我们观察到DAO创作者也自发的选择了一些不那么大众的内容平台来发布内容。这些平台包括GitHub、Mirror、Medium、Radicle、Paragraph 等等。本文选取其中最典型的3个平台，探究这些平台的优劣势，分析为什么DAO创作者会选择这些平台，也能够从侧面了解DAO创作者真正需要什么样的内容平台。

GitHub

简介

GitHub 平台最初于2008年上线，该平台创立伊始的定位是“软件源代码托管服务平台”， 是程序员们最熟知的网站之一。2018年6月，微软以75亿美元的价格收购了GitHub。 

GitHub 目前的实际用途早已超出了“代码托管服务平台”的范畴，兼具文字内容平台和社交平台的属性。2022年10月，微软称GitHub的年经常性收入已达10亿美元；2023年1月，GitHub宣布其用户数突破1亿。

内容创作者变现模式

GitHub 提供了内容创作者的直接变现渠道：GitHub Sponsors。该功能于2019年5月推出，允许内容创作者开通赞助（Sponsor）功能以获得报酬。 

如果内容创作者开通了赞助功能，读者可以点击内容创作者的个人信息界面，点击“Sponsor”按钮，并选择赞助的方式（例如每月5或10美元）。在确认赞助之后，读者就会成为这名创作者的赞助人（Sponsoring），在双方的个人信息界面都会展现Sponsors和Sponsoring 的关系。

GitHub Sponsors 对来自个人账户的赞助不收取任何费用，这些赞助完全100%归被赞助者所有；对于组织账户象征性的收取最高6%的费用（主要为信用卡费用和服务处理费用），没有抽成。缺点是不支持大陆地区的银行，因此国内用户需要办理香港或其他地区的银行卡才能收款。 

下图是 GitHub Sponsors 功能首页展示的一位被赞助者：David Nolen，ClojureScript 的首席开发者。David Nolen 在 GitHub 上拥有约 4,200 名关注者，目前拥有34名赞助人，曾拥有50名其他的赞助人。同时，David Nolen也是两名GitHub 用户的赞助人。

平台优劣势

优势： 

1）GitHub 是世界上最大且最活跃的开源社区之一，拥有着极大的用户数和强大的影响力 

2）较之于一般的大众化社区，GitHub用户素质相对较高，内容反馈质量亦相对较高 3）GitHub 对于打赏不抽成的做法是创作者和用户的福音，能够有效激励创作者使用平台发布内容，也使得用户打赏的意愿更强 

劣势： 

1）GitHub 是典型的中心化平台，创作者受制于平台的程度较深 

2）GitHub 仍以代码和技术交流为主，文字写作在该平台不属于主流 

3）GitHub 仅有读者自愿打赏的功能，并无其他直接变现手段，绝大多数创作者使用该平台的目的在于相互交流学习，变现能力弱

Mirror

简介

Mirror 是 a16z crypto 前合伙人 Denis Nazarov 创建的去中心化写作平台。平台于2020年上线，口号是“像往常一样写作，以前所未有的方式发布”。

Mirror 是目前最受web3从业者喜爱的写作平台之一，其创建伊始的目的便是通过区块链技术确保内容永久存储、让内容创作者真正拥有自己内容的所有权。因此其功能设计上面非常贴合DAO创作者的习惯。其为创作者提供的工具包括：发布文章（Entries）、众筹（Crowdfunds）、拍卖（Auctions）、合作贡献分流（Splits）、社区投票（Token Race）等。 

根据similarweb 的数据，Mirror 官网 2023 年 10 月的总访问量为240万次。平均访问时长为2分22秒。

内容创作者变现模式 

由于 Mirror 创建之初的目的之一便是帮助创作者体面的进行内容变现，因此，该平台的内容创作者变现模式较为丰富。具体包括： 

1. 出售NFT 

在 Mirror 上发布的文章，创作者可以选择将其铸造为 NFT，并创建 NFT Collections。创作者可以设置 Collections 的名称、价格、版税比例、发行网络、发行数量、挖矿时间等信息。读者可以在规定的时间内，mint该系列的NFT成为其collectors。 

举例而言：假设一位DAO创作者设置了一个名为“Chauncey Show”的 NFT Collections，该系列包含了这名创作者过往撰写的10篇文章。价格设置为0.1 ETH，版税比例为2%，发行网络为Optimism，发行数量为100个，挖矿时间为2023年12 月1日至2023年12月31日。

那么，他的读者可以在2023年12月内，花费0.1 ETH的价格+ 0.00069 ETH的mint 费用+当时的gas fee，购买到一枚NFT的所属权，而创作者可以收到0.1 ETH 的报酬。假设读者后续转手以0.2 ETH的价格出售了该NFT，那么创作者还可以再收到0.2 × 2% = 0.004 ETH的版税（Royalty fee）。

2. 众筹资金

创作者可以使用Crowdfund功能为项目筹集资金。例如，某个DAO创作者可以基于一本正在构思但尚未完成创作的书发起众筹，并基于这本书发行代币，参与众筹的用户成为这本书的股东。当书稿制作完成时，可以铸造成NFT进行售卖，参与众筹的读者可以按照事先约定好的机制参与收益分成。

3. 合作贡献分流

在一篇内容的创作中，如果某位创作者A使用了另一位创作者B的想法或者信息，或者A和B实际共同创作了某项内容，那么A可以事先和B约定好贡献分配的比例，在内容出售时自动将收益分配给A和B。

平台优劣势

优势： 

1）Mirror 采用Arweave 实现部分内容的去中心化存储，内容安全性高 

2）将内容打包为NFT的形式有助于内容的确权和转卖，大大促进了内容直接变现 

3）Mirror 作为最热门的web3创作平台，引领了文章众筹、合作贡献分流等新鲜玩法，未来也可能推出更多有趣的创作者工具 

劣势： 

1）Mirror 原生的文章分发和导流前端相对不完善，目前没有内部搜索功能，优质内容发现机制相对较弱，只能将发布于 Mirror 的文章链接在其他平台引流（但同时其实也是优点，不会用太强硬的推荐算法绑架创作者、引发流量焦虑） 

2）虽然拥有较为完善的变现功能，但实操层面大多数NFT仍是free mint，并且参与mint的用户数较少，实际变现效果不及市场预期 

3）受制于区块链基础设施的成本，文章内嵌的图片、视频等内容并未上链。

Medium

简介

Medium 是由Blogger 的创始人、推特的联合创始人Evan Williams创办的内容平台，于2012年上线。该平台的定位兼顾专业人士的在线发表需求，也照顾到非专业人士的日常写作需求。 

Medium 虽然也是一个传统的中心化内容平台，但却也受到了许多web3从业者的青睐。该平台的主要特色包括：UI 设计简洁、读者在阅读文章的同时可以进行标记或笔记、写作文章会保留过程版本且可以发送给其他人审阅批注等等。 

根据similarweb 的数据，medium 官网 2023 年 10 月的总访问量为 1.57 亿次。平均访问时长为2分1秒。 

内容创作者变现模式 

Medium 为内容创作者提供的变现模式被称为“付费墙”。这种模式从2016年开始正式实施。具体来讲：Medium上的文章被划分为两种：1）免费文章；2）付费墙内的文章。Medium上的创作者在发布内容时，可以选择是否将文章加入付费墙。

加入付费墙的文章，在预览界面会有一个五角星的图标。如果读者想阅读付费墙内的文章，那么就必须开通Medium会员，费用为每月固定的5美金。假设这名读者在1个月里在Medium上阅读了10篇付费文章，那么10篇文章的作者将会分享5美金的收益。其分配算法是按照读者的阅读时间比例来分配，读者在某篇文章停留的时间越长，则该篇文章的创作者分得的收益比例越高。

平台优劣势

优势： 

1）Medium 是一个很“纯粹”的内容平台，创建的目的便是为阅读篇幅较长的严肃文章而生，平台无广告、软文很少、阅读体验佳 

2）Medium 的内容变现模式相对温和，有利于DAO创作者潜心创作优质长文，厚积薄发 

劣势： 

1）Medium 仍是典型的中心化平台，创作者受制于平台分发算法 

2）Medium 订阅价格相对低廉，且读者阅读新的文章会导致之前被阅读的创作者分配的收益下降，创作者在该平台获得的收益相对较低 

3）Medium 系统只会推荐英语文章，对于其他语言创作者相对不友好

结论

1.DAO创作者经济现状 

根据Linktree 的数据显示，66%的创作者将在线内容创作视为副业，36%的创作者参与内容创作经验不足一年。而在这些早期创作者中，仅有6%的人收入超过 10,000 美元；35%的创作者已实现了变现，但收入不足以维持生计；其余59%的创作者完全没有实现内容变现。 

实际上，无论是传统的个人创作者、公司创作者或是DAO创作者，其实都面临着一样的变现难题。从上文中提及的几家典型内容平台也可以看到，头部DAO组织的关注人数并不算多（必须直面的事实是，DAO目前依然是广大人民眼中的小众话题），想要通过平台直接变现的能力很弱。但随着越来越多DAO的涌现，我们也可以在现实世界中看到DAO组织话语权的增强，越来越多的人愿意去了解什么是DAO，以及倾听DAO的声音。因此，DAO创作者经济仍应主要采用间接变现的模式。鉴于大多数的DAO并没有成熟的“产品”， DAO创作者更应该走出象牙塔，在平台传播优质内容的同时，参与到更多现实世界的活动中来。或许是建立付费社群、或许是撰写付费约稿、或许是参与商业活动、或许是传递行业信息，在大众还没有以一种功利性的眼光看待DAO组织的时候，这些方式也可以在不污染内容本身的前提下实现体面的变现。

2.什么样的内容平台适合DAO创作者

近年来，以 Mirror 为代表的新力量已经在内容平台的搭建方面做出了巨大 创新和努力。经过一段时间的粗浅研究，我本人认为理想中的适合DAO创作者 的内容平台，应该满足以下条件： 

（1）内容归属于作者，并去中心化存储 

（2）不过度依赖于算法分发 

（3）平台有一定的直接变现能力

（4）创作者与创作者之间、创作者与读者之间、读者与读者之间都能形成活跃交互 

我个人认为，Mirror已经初具理想内容平台的雏形，但受制于内容搜索、整 合能力的薄弱，以及担心过度依赖算法推荐造成的“泯然众人”等原因，该平台 的使用体验还称不上极佳。未来，去中心化内容平台如何在建设过程中引入更多 的功能、增强平台使用者的交互、增进创作者收益，但同时又如何规避传统资本 的干扰，才是需要长久关注的问题，值得更多有胆识又有理想的人的努力。 

虽然早在2021年就已经有很多人喊出“DAO才是创作者经济的未来”，但 DAO创作者经济的建设依然路途遥远。
