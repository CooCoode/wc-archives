在 Seed Club 第 2 季的列表中，有一系列探索社交代币应用的前沿项目，十分引人注目。比如 Jump（一个由 b2b 广告代理领导者组成的社区）、Global Coin Research（一个代币化媒体组织）等等。

我作为 Ampled 团队的一员参加了 Seed Club。我们是一个直接支持音乐家的网络平台，由艺术家和工作者拥有的合作组织。在加速器中，我们一直在探索社区代币的实施，作为现有民主合作社的协调层。

我们一直在探索的一些问题包括：

收益：某人如何赚取社区代币？

  • 创建赏金是为了激励社区参与吗？

  • 哪些工作/任务应该得到奖励？

兑换：一旦你有了社区代币，你能用它们做什么？

  • 我们可以用原生社区代币作为拍卖音乐设备的货币吗？

  • 我们可以给代币持有人提供商品折扣吗？

  • 它们可以在其他平台上使用吗？(比如在 Catalog 上竞拍1/1的版本)

访问：代币持有者可以获得哪些特殊的权限？

  • 会员是否可以获得财务和私人频道？

  • 会员可以解锁一个实体共享空间吗？

01

杀手级访问应用：Token 门控的物理空间

Token 门控物理空间的想法并不新鲜，这似乎是关于 Token 使用潜力的长期讨论话题了。现在，围绕 NFT 作为票证的使用和讨论越来越多，像 Unlock 等协议创建的一套工具，就集成了让 NFT 可以提供实用能力的程序。最近，Friends with Benefits 社区举办了一场派对，只有拥有 60 个 $FWB 代币的人才能参加。这是最明显的例子之一，说明代币不仅可以作为字面意义上的门禁，还证明了在 Token 化社区中，个人被共享的那部分信息的价值有多大。

Ampled 的工作人员，从项目开始就在讨论一个共享的物理工作空间/家庭基地的想法。我们很想创造一个空间，可能在纽约市，任何 Ampled 成员都可以使用。它可以是一个有工作或社交空间的办公室，也可以是一个活动空间或共享音乐排练室。我们的想法是，如果你是项目的共同所有者，那么任何属于 Ampled 的空间就都属于你。这是有助于加强所有权和合作主义意义的一种非常直观的方式。

最近，在 Unlock Protocol 的支持下，我们开始尝试实施这个想法——在纽约布鲁克林创建一个所有成员都可以进入的空间。我们正在计划给现有的会员提供代币，从而获得进入这个空间的权限。

这个想法就是为了展示这种将成员从数字空间连接到物理空间的令人兴奋的方式。

The Patch House是这个想法的早期灵感来源。这是一个品牌赞助的空间，乐队在巡演时可以免费住在布鲁克林。

02

制造一个有效的令牌门控智能锁

实验的目标是创建一个 MVP、无代码版本的令牌门控智能锁。最终我找到了一种相对简单的方法，通过 Unlock 协议创建的 NFT 来授予对物理空间访问权限。以下是它的工作原理：

  1. 会员使用 Unlock 购买 NFT 锁。（任何其他 NFT 或社交代币也适用）

  2. 代币通过 Collab.Land 授予对私人 Discord 频道的访问权限。

  3. 仅限会员的私人 Discord 频道有一个机器人，每天发送一个新的 4 位数 PIN 码。

  4. 预编程的密码作为临时密码与智能锁配合使用，可用于当天开门。

这是一个很笨拙和也很简单的解决方案。基本上，代码已经部署完成，计划与智能锁一起使用，而代币门控的 Discord 频道内的机器人只是在每天早上通过查询 airtable 数据库，显示哪个代码可以打开当天的门锁。

03

硬件

为了证明这个概念，我使用了这款 Schlage Encode 智能锁。选择这个锁的原因是它的原生移动应用允许创建预设的临时键盘码。

04

软件

为了实现这个想法，我使用了几个无代码软件工具：

  • Collab.Land（用于控制 Discord 频道的令牌）

  • Airtable（作为存储预编程锁的数据库）

  • Airtable Automation（如果日期=今日，查询数据库并向 Zapier 发送包含相关代码的电子邮件）

  • Zapier（解析电子邮件并将代码发送到 Discord）

  • Discord & Discord webhook（接受并显示代码）

需要说明的是，这个系统有些缺陷，我绝不是无代码自动化专家。但是，我已经用智能锁进行了测试，它确实有效（即使端到端系统可以大大提高效率和优化）。先不谈潜在的改进，它确实有效！这是创建令牌门控物理空间的一种方法。

Token 门控的物理空间有许多可能的应用。可能包括：

  • 协同工作空间

  • 物质图书馆（例如：Library of Things & Synth Library NYC )

  • 社交俱乐部

  • 演讲厅

  • 会议空间

  • 私人酒吧或餐厅

  • 共享的练习/排练空间
