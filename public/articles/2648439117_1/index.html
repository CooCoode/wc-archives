SeeDAO支持｜9月相约新加坡！Rebase HackerHouse in Singapore

作者: RHH 发布时间: 2023-08-29 13:39:36
9月11日-15日，我们以 Building from DAO to Network Polis 为主题举办 Rebase Hacker House，邀请各位开发者共赴花园城市新加坡，开启一次4天3夜的共居共创之旅。

在近年来的组织形态变化趋势中，DAO逐渐成为组织管理研究者心目中的组织3.0。DAO通过相似的理念、共同的目标将大家聚集起来，采用去中心化的透明自治方式获得了组织驱动的持久动力，通过大量采用自动化的工具，使人从重复繁琐的忙碌转向更具创造性的工作。

如果说 DAO 是承载我们某种特定生产关系的新型组织形式，那么众多的DAO将融合交流，形成新的组织形态—— Network Polis。Network Polis 将涵盖我们生活的方方面面，它是一座构建在去中心化网络上的虚拟世界里的理想城邦，我们会以一种全新的关系生活在其中。

实践者们为新的组织形式更好地实现其去中心化的治理和信任的目标做出了各种各样的努力。尽管这其中不乏创新，例如 MakerDAO 结合 AI 做 DAO 的治理，Zuzalu 使用 zk 技术给予市民可验证的身份，DAOstar则提出了 EIP-4824 以增强不同 DAO 之间的可发现性、可读性和互操性。

而实际上，更普遍的情况是，很多应用只不过是 Web2 应用披上了区块链技术的外衣，像Web3版的聊天软件等，且用户体验还远远比不上Web2本身的应用。因此大部分 DAO 组织仍然选择采用Web2的工具进行治理，而且缺乏自动化，成员困在大量重复的人工劳动中，这很大程度制约着 DAO 内的创新以及 DAO 本身的发展。

不管是 DAO 还是 Network Polis，都尚处于开端，未来还有很长的路要走。新型组织通过区块链技术得以更客观地留存其历史，而它们的未来则依赖于更繁荣的基础设施和更成熟的工具，来实现更高效更自动化的治理、更丝滑的协作，从而实现规模化的发展，真正地成为以人为目的的新型组织形式。

Rebase 社区一直致力于汇聚开发者的力量，相信创新来自于思维的碰撞、不同观念的沟通。因此，在Token2049 召开之际，我们以 Building from DAO to Network Polis 为主题举办 Rebase Hacker House，邀请各位开发者共赴花园城市新加坡，开启一次4天3夜的共居共创之旅。通过这次 Hacker House ，我们希望为 DAO 和 Network Polis 建设底层基础设施及生态应用，吸引越来越多的人参与其中，透过实践从各种概念和构想的迷雾中，看清我们期望中那个理想世界的具体模样。

关于HackerHouse 

HackerHouse是一种短期共居活动，针对科技行业人士，特别是独立开发者、创业者和数字游民。活动提供共享的居住、办公和社交空间，致力于行业创新，鼓励合作和资源共享，推动技术发展。每个HackerHouse有独立的研究主题，参与者在活动期间结识志同道合的朋友，收集想法并推动实施，经闭门开发后，组织线下Meet Up，促进项目间交流，并通过Demo Day展示成果。

关于 Rebase

Rebase 社区是由中国的一群 Web3 开发者们基于业余爱好建立的非营利开发者社区。旨在链接 Web3 开发者，建立沟通交流桥梁，助力极客促进科技创新，推动社会进步。

目前，Rebase 社区有几个专注于不同领域的工作小组。Web3极客日报，Rebase大学，Rebase Hackathon，Who-Is-Hiring，Event-Pool，Rebase D. Ventures 和 Web3Camp 等。

如果你有兴趣参与其中或希望建立新的研究方向和领域，可以通过 Rebase 社区公众号同我们取得联系！
