Network State 发布两年, 进展得怎么样了？加密人文系列1

作者: 大地之子定慧 发布时间: 2024-09-13 09:12:11
去那新秩序诞生之地
Crypto 的未来在哪里？当 web3 的梦想破灭，行业回归赌场，我们的未来又在哪里？
2022年7月，Balaji 发布了《Network State》，描绘了一种人类社会与加密技术深度结合的可能：
Network State 是一种高度团结的在线社区，拥有集体行动的能力、独特的价值主张/道德创新、公认的创始人。通过众筹获得分散在全球的领土、用加密货币捍卫自身主权，用智能合约约束的共识政府。通过普查在证明有足够多的人口、收入、不动产的基础上，进而获得一定程度的外交承认。
当越来越多的数字游民在全球流散，血统、民族、家乡被解构的今天，我们需要团体、联结与精神共鸣的同伴。而加密技术与诞生也使得我们有了新的治理工具。在这本书发布的随后两年中，全球大大小小上百个与这个愿景相关的项目开始形成网络，在高度金融化的加密世界中显得格格不入。
最令人瞩目的应当时 2023 在黑山诞生，并在短短一年内遍布全球十几个城市的 Pop-up City 快闪城市 : Zuzalu。人们为了共同的愿景来到某座城市展开为期 1-2 个月的共居，在高频交流与链接中展开共创。
除此之外，还有许多正在涌现的方向：例如承保一个海岛作为前沿科技的实验基地、将全球数字游民社区接入一套会员体系的网络联盟、与小国合作为数字游民颁发链上身份 ID、为数字社会设计的抗审查/隐私公链，以及通过第三方机构将链上共识在现实世界映射的房产 RWA 实验……
在翻阅了上百个项目后，我整理出了一份《Network State  List》，并在这篇文章中分享自己的一点观察：
网络国家，发展得到底怎么样？
 01 

炒叙事的都死了，真要搞还得靠资本+科技公司+岛国联合

2022年初，去中心化治理和DAO 工具平台Aragon 联合创始人Luis Cuende 在社交媒体上发文宣布启动Nation3 DAO，旨在建立数字国家。其代币（NATION）价格从510 美元飙升到12749 美元，最高涨幅2399%。
Nation3 希望对“国家”的组织方式进行范式革命。我们经历过集权，也正在经历被窄化与阉割过的民主。而 Nation 希望实现基于加密协议构建的社区之网，由无数独立自治的“城邦”相互链接创造的未来。
代币机制上，Ntion3 引入了一些经典的 Defi 玩法，例如质押 token 获得治理权、铸造创世NFT等一系列权益。
然而在两年后的今天，我们看到的是死亡螺旋：
社会实验并不简单，然而总有喜欢用叙事来忽悠人的 Defi 和公链项目，强行用 Network State 为自己没用的基建/经济模式加戏，而用户量和代币价格一塌糊涂（例如 Jur、Epoch Island）。
然而令人意外的，2022 年另一个曾在圈内一度火热的项目 Satoshi Island 中本聪岛竟然没有跑路。
中本聪岛是一个 3200 万平方英尺的私人岛屿，位于热带天堂瓦努阿图。该岛私人所有，旨在成为加密专业人士和爱好者的家园，建设世界加密之都。经过多年的准备，瓦努阿图总理和财政部长的绿灯和所有批准后，Satoshi Island 在 2022 年发行了 5000 多张公民 NFT 和近 2万张土地 NFT 进行募资，之后便逐渐没有了消息。
相比重资建设一个小岛，跑路似乎是更好的选择。
然而就在2023年8月，其代币上架 OKX；10月，岛上第一批模块化房屋在中国的代工厂被建造，并在今年3月运输到了岛上。
随着利好消息与牛市的到来，跌下去的币价也在今年迎来一波 2-18 倍涨幅。相比归零后一点起伏都没有的 Nation3 和 Bankless DAO，已经非常令人惊喜了。
在美洲，去海岛上搞社会实验的案例，同样在大资本的加持下发生。
美国后花园加勒比海的洪都拉斯，就在2009年通过了一项自治区法案，为外国投资者开放前所未有的立法、执政、司法和经济自主权，目标是在2025年吸引至少5亿美元的外国直接投资，在本地创造至少一万个就业岗位。
截止目前已经有 3 个项目获批，其中一个就是源自硅谷背景的 Próspera —— 在一罗阿坦岛上建立起的 Start up Society。
Próspera 的建立得到了不少硅谷大佬的青睐，例如一家生物科技公司 Minicircle 在美国本土苦于基因疗法无法得到食品和药物管理局（FDA）的批准，于是就在这里开始了人体注射实验。他们会将一小段新的基因组注射到人体内，并测验后续安全性。目前已经有 300 多人来到  Próspera 接受了该疗法。Web3 世界中的 Vita DAO 也来到 Próspera  举办线下共居的 Pop-up City：Vitalia，联合生物科技行业尖端力量研究如何“将死亡变成一种选择”。
Vitalia 现场
虽然这些大资本投入的项目并没有像很多 web3 中的 DAO 一样有着开放开源的社区精神，但至少他们在认真建设，去将这份愿景变成现实。
相比将这些小岛视为 Network State，我更觉得这是无数个 Network State 在现实世界的投影。例如 Próspera 可以在生物黑客的标签外，邀约其他另外 20 个网络社区带着彼此各不相同的愿景来到这里建立线下据点，以挂牌加盟的方式将自身属性无限展延。
 02 

政客的游戏与人民的国土：为什么要把 Network 变成 State？

Balaji 在其书中非常强调 Network State 在价值主张（道德）上的创新带来的内部的凝聚力，例如“生酮” 或“纯素”饮食爱好者组建的小镇中，所有的人都会自动以此为标准，你无需顾虑走进任何一家餐厅都会吃到你想吃的食物。
然而这种组织方式并非加密世界独享。
以宗教为例，遍布全球的教堂、寺院、道场不正是一个典型的 Network State 吗？信徒共享某种愿景+世界观，在全球建立据点形成网络。欧洲中世纪流传下来的马其他骑士团，虽然如今在没有任何一块领土的情况，拥有受国际认可的护照（无领土主权实体）。流散千年的犹太人，也在以色列成功建国。
Cabin DAO 尝试链接全球具有加密文化的数字游民社区，形成网络与共同体，然而这也并不新鲜。Couchsurfing 早已链接全球无数沙发客与房屋主，形成了自身独特的关系网与社区文化。
星巴克更是带着“尝试创造人与人链接的无限可能”的追求，通过提供咖啡于“第三空间”，把连锁店开到了全世界，人们自发形成了去星巴克办公、谈事的文化。他们甚至发行了自己的会员 NFT！而为什么没有人会认可星巴克也是 Network State？
在原书中，Balajia 仔细分析了为什么缺少一些特征，网络国家就行不通。
例如社交网络和价值主张上的创新缺少任意其一，就无法形成足够强的共同体意识（新“民族”的诞生），而缺少共同体意识与目标就会失去行行动力。组织在一起的人们如果不用加密货币捍卫自己的权益就会很容易被主权国家冻结资产，如果不众筹领土就不会被传统世界认真对待，以此类推。
正因实现 Network State 很不容易、有无数细节一旦疏忽就可能失败，因此 Balaji 认为 Netwrok State 不应简简单单是一个网络社区全球建据点，而需要具备道德创新、创始人、人口普查、加密货币等十几条达成变革的特质。
但我们不得不问一句：Why？
如果我加入了一个社团，一个 Network，为什么一定要成为一个 State？
如果我的创业公司在美国因为 FDA 的限制不能做人体基因相关的前沿科研，那我可以去其他支持的城市例如 Próspera，而非花费时间筹建一个“反 FDA联盟” 的网络社区；
如果我是生酮或者纯素饮食的忠实粉丝，我可以去那些支持相关饮食习惯的餐厅或地区，而非花钱众筹一块土地然后把家搬过去。
如果我是背景理想的 web3 行业从业者，我可以穿梭在全球不同的数字游民社区/Pop-up City，专注于开发 Public Goods，而非把时间都花在与建筑材料和施工队痛苦的打交道上，痛苦的开垦入地。
实际上，上一轮牛市众筹土地、售卖 NFT 公民权的 City DAO，至今没有去开垦那片荒芜的土地，并在今年7月左右关停退款。
所以，我们生活在现代世界，我们有选择，所以为什么一定是 Network State？为什么我们要把自己身处的 Network 变成 State？
Balaji 在书中给出的答案是：
相比改革旧国家，启动一个新的，也可能比改革你的国家更容易，我们想要一个没沉重的包袱的新开始。特别是对于某些类型的技术——特别是像延长寿命这样的变革性生物技术——我们需要新的司法管辖区，它们需要具有根本不同的风险承受度，以及所有选择加入的人都要明确同意。
对于技术专家而言，如果我们建立一个人人都支持技术创新的社会将会为你的实践带来解放。
天主教会将日心说的支持者烧死在火刑柱上，航天飞机就不可能被发明；苏联禁止影印机，互联网就不会被允许出现；陈旧的出租车规定给 Uber、Lyft、Grab 和 Didi 等公司造成了百亿美元的损失，金融规定阻碍了比特币和以太坊万亿美元的市值…….而这一切有机会被一个更加支持创新的社会系统所解决。
当然，网络国家并不适合每个技术专家。如果你主要关心的是编译器或编程语言，你可以在目前的体制下过得很好。如果你只想在大科技公司获得稳定的薪水，网络国家也不适合你。但是，如果你想要加速物理世界的创新，你终于有了一个答案。
而对于政治进步人士：
如果你是一个年轻的政治家，也许你不想等到 70 岁时才可以交纳入会费，以留下自己的印记。
如果你是一个社区组织者，网络联盟会给你一个数字社区来组织，虽然有时可能会反对国家和公司，但能为个人成员的开源项目、企业和咨询服务带来好处。
如果你是一个政策专家，网络国家允许你对政策进行道德实验，选择加入的参与者和你一样对治理创新感兴趣。你可以试验数字民主、新的政府形式，以及任何你认为有趣的东西。
如果你是一个理想主义者，网络国家可以让 19 世纪中期美国的自愿公社重现，在那里，人们可以选择加入，建立他们自己的乌托邦愿景。
如果你是一个无政府主义者，网络联盟提供了一个愿景，你可以在没有传统治理和没有胁迫的情况下进行横向合作。
如果你是一个城市规划师，网络社会可以让你获得支持，积累资金，众筹你的美好愿景。
简而言之，无论你想尝试改革，还是创立全新的政府形式，网络联盟、网络社会和网络国家这些概念，总有一款适合你。
其实不难看出，Balaji 所尝试说服的读者是那群天然有着强烈变革意图政治家、理想主义者以及挑战社会边界的科技骇客——然而这一切似乎离我们过于遥远。
实际上已经有这样的团体在实践了，例如打出“国家加速主义”旗号的 PRAXIS，他们的口号是“局部降速终将失败，因为无人可以阻拦技术在世界范围内蓬勃发展”
“Going slow locally is unwise because going slow globally is impossible.”
目前，他们声称自己已有拥有了 2000+成员、140 多家企业加盟，总募资超过百亿，其中就有做人体冷冻的公司与其成为合作方，提供后续的保存、运输、维护工作。
目前在推特上可以搜到不少万粉大V 晒出他们拿到的金属签证，证明自己是 PRAXIS 的一员。
然而这一切似乎离我们过于遥远。
如果从演化的视角来看，如果 Network State 所预测的是人类组织方式演化的优质方案，那一定会涌现出真正的 Network State。
而作为人民，我想我们大可把 Network State 的叙事抛在脑后，我们要做的不是奔向那个“正确”的方向，而是更好地活在这个世界。
 03 

黑客松 + 共居 = 快闪城市 ≈ The MVP of Network State ？

如果跳出硅谷大佬的圈子，我一直好奇网络国家会在我的身边以怎样的方式落地。
是通过 DAO 吗？先形成高度组织化的网络社会，找到自己的产业和利润，并搭建无数线下实体据点吗？——似乎并没有，这个赛道都快要被行业证伪了，因为除了 Defi 和开发者社区的场景外，DAO普遍政治早熟，只会分钱不会赚钱，币价崩了就没了。
或是通过线下空间联盟？线下数字游民社区之间互相链接，加入一个个不同文化的网络国家，组建网络联盟吗？——似乎也不是，因为每个线下据点因为人群的差异，往往都有自身独特的市场、文化与受众。也很难有数字游民有在数十个线下社区之间穿梭的需求。
除此之外，是否有最小可行的方式能展开 Network State 的落地实践？
在去年3个月在 Zuzalu 的共居结束后，Web3 Pop-up City 如蒲公英一般在全球开花。
相比一般持续一周的线下 Hackathon，Pop-up City 一般持续一个月之上；相比人数规模一般在 20 人以内的 HackerHouse 共居，Pop-up City 又会大到有几十上百人才足够称之为一个 City。而 10-20 万就能办一场的较低预算，也为很多团体降低了入局的门槛。
目前在 Zuzalu 的首页，我们可以看到今年有 19 场 Pop-up City 发生在加州、柏林、阿根廷、泰国、印度、格鲁吉亚、瑞士等地。
今年下半年，因为 DevCon 将在曼谷举办，有至少5场 Pop-up City 作为周边活动在 9-12 月在周边发生。

结语：去那新秩序诞生之地

Balaji 在书中有一个假设：从0到1 建一个新国度，比改变现有的国家更容易。
然而真的是这样吗？对于那些手握金钱与权力的大佬来说或许是的，然而在一些特殊的历史时期，改变旧秩序的难度可能会大大降低。
二战后，大量民族国家独立，战败国在被殖民中学习民主与现代化，联合国与布雷顿森林体系出现。在上个世纪末，东亚与互联网快速发展、中国改革开放并加入世贸，以及在 08 年金融海啸的灰烬中诞生的 BTC…….
在那些百废待兴的时刻，在那些增量市场快速发展的时刻，在那些人们下决心希望改变的时刻，正是人类的变革闪耀时。
或许今天我们会遇见二战后整个世界洗牌重建的画面，而局部的变革则不断在大国博弈间发生。
例如乌克兰战火有望平息时的战后重建、中美贸易战开打后产业链从中国向外的转移、阿根廷的经济改革，以及我们常常听到的用加密货币系统对银行系统稀烂地区的支持（包括将 BTC 纳入法定货币的萨瓦尔多、中非共和国）。当然也有我们一直在等待的，天朝的下一轮转型。

此时此刻，已经有这样的变革正在发生：

通过 Sun 老师的朋友圈，我了解到在众多力量的合作下，以太坊建设者在阿根廷申请下了一片“加密货币经济特区”，实践区块链与现实世界的结合，包括不限于：

  • 数十家餐厅、酒吧支持虚拟货币，短短两周产生1450笔交易（截止8.21）

  • Solidity 和 Ethereum 成功进入高中，成为教学课程的一部分

  • ZK 身份证项目首日上线就有 700 名用户开通使用，还在持续增加

Web3 建设者们正在尝试从一片社会试验田中收集真实的数据，带着真实的数据获得阿根廷政府的进一步支持，并一次为案例去与更多的国家、地区商榷。这并非一路顺遂，无数脏活累活蕴含其中，然而也充满希望。据说，相关人员正在尝试与政府商议为加入经济特区建设的国际区块链技术人员发放特别签证，并有拿到阿根廷护照的机会。

在孤阅葛旭船长的启发下，我发现其中一个离我们比较近的可能是墨西哥：
正如上世纪末日本制造业企业通过出海“再造一个日本”，中国也在经历着类似的变迁。贸易战后，美国正在自己能掌控的地区重建产业链，例如墨西哥和拉美。按目前政策，一辆汽车75%的部件在墨西哥生产，那么它对美就是0关税。靠着这一优势，除了义乌商品的涌入，已经有不少中国企业开始在墨西哥建厂，把原材料、零部件跨越大洋运过去组装，承接北美的需求，华人商会也随之形成。华人老板穿梭在中墨两国间，流转物资人力、传递信息，当然还有我们更熟悉的，交换金钱。
我们是否可以带着 web3 的开发能力与资本力量，卷入贸易与博弈的现场，参与新秩序的搭建？

另一个数据也在佐证这种可能：

中国互联网渗透率 77.5%，墨西哥有 80%，两者不相上下。然而，中国银行账户拥有率超过95%，墨西哥只有40%。据说，近期墨西哥融资规模最大的4家企业均为电子金融。Crypto 是否有机会卷入其中？

于此同时，Network State 的创始人 Balaji 先生也没有闲着。

就在8月17日，Web3 顶级投资人 Balaji 正式发布了 The Network School，这是一所聚焦科技创新的教育实验性大学，坐落在新加坡附近的美丽岛屿上，致力于为全球资源有限的人才提供发展机会，优先录取亚非拉地区的青年才俊，无申请年龄限制。关注学习（Learn）、锻炼（Burn）、赚钱（Earn）和娱乐（Fun）。首个学期将在今年 9.23-12.23 举办，为期3个月。每天将会发放 1000 美金悬赏开源软件开发与 Ai 创作。

他在发布信中写道：“我的父亲出生在一个极度贫困的国家，尽管没什么背景，但只要有机会他就能做出一番事业、去实现他自己。这就是我们所说的 Dark Talent：身处边缘未被主流世界注意、身怀天赋的青年才俊。十多年来，我一直在思考如何让更多处于类似境地的人也能拥有同样的机会、如何赋能全球的 Dark Talent.......

Network School 适合的是印度工程师和非洲创业者，适合来自美国中西部和中东地区的建设者，适合中国的自由派和拉丁美洲的自由主义者，适合东南亚崛起的技术人才，以及欧洲仍在坚持的投资人。”

Network School 是我为年轻时的自己打造的“产品”——那个有抱负的年轻工程师。这是我梦想中的社区：一个技术资本主义的大学小镇，一个全球人都负担得起且真正以实力为王的斯坦福2.0！“

完整的发布信可见：Web3×教育创变：一所创新大学正在诞生！
