SeeDAO | 想要发行个人专属 NFT？以下五个平台帮你轻松实现！

作者: 翻译公会 xiaobao 发布时间: 2022-11-09 15:40:33
分类: #NFT, 创作者, 无代码平台, 智能合约工具, NFT工具
快来看看
如果社交平台上的内容创建者已经围绕其内容建立了社区，并且通过销售课程进一步进行了内容和服务的生产，那么，他们也能够从 NFT 背后的底层技术中受益。
要做到这一点，需要创建 NFT 作品集，并将其数字产品作为实用程序提供给购买 NFT 的人。由于 NFT 是可编程的智能合约，因此，每当一个用户将 NFT 出售给另一个用户时，创作者就可以按百分比为自己分配版税。
NFT 版税可以为这些创作者提供源源不断的收入。NFT 只是达到目的的一种手段，因此，如果需要复杂的工具或专业知识才能制作，就不会给任何人带来好处。
因此，NFT 领域出现了一些无代码平台，其唯一目的是让任何人都能轻松生成和出售他们的 NFT。
完全不懂编码，或只有很少编码经验的创作者可以利用这些平台来构建 NFT 作品集。
发布 NFT 的整个过程就在这 3 个简单的步骤中：
  • 生成艺术作品或作品集和元数据。元数据是作品及其所代表的内容的文件描述。
  • 将艺术作品和元数据上传到 IPFS。IPFS 是一个去中心化的文件存储系统，用于共享和存储文件，（这会生成一个 URL，它是你上传的数据的唯一标识符）这个 URL 将被添加到你的智能合约中。
  • 生成智能合约并将其部署到区块链。
智能合约是一组代码，被外部账户的交易触发时自动执行。它们用于设定你的 NFT 的销售规则。
这些平台的不同之处在于：在无需借助任何外部工具或平台的情况下，每个平台能让你对上述各个步骤做多大程度的操作。
作品生成器：一个作品生成工具，使用提供的图层（背景、特征和符号）来创建作品。
图层：图层是用于生成作品的若干图像，它们是一系列元素和部件，将会合成到一起制作成你的最终作品。
假设你想要生成一个青蛙作品集，那么你的图层将会包含以下内容：

  • 一个基础图层（蛙身）

  • 背景颜色

  • 特征（眼睛、鼻子和嘴巴）

  • 任何你想加到作品中的配饰，比如帽子、太阳镜、手杖等

任何图像编辑软件都可以用来创建图层，例如 Photoshop。每一层中，制作上的变化数量将决定作品的大小。
在上面的例子中，如果我们为每个图层（基础图层、背景、眼睛、鼻子、嘴巴和配饰）创建五个变体，就会生成一个超过 10,000 张图像的作品集。它是将所有层中的变化数量相乘计算出来的。在我们的例子中，它就是：5*5*5*5*5 = 15,625。
智能合约生成器，是用于编译智能合约并将其部署在区块链上的工具。
部署智能合约仅仅意味着将编译后的智能合约代码，以不指定任何接收人的以太坊交易的方式发送出去，一经部署，该合约就会获得一个区块链上的以太坊地址。
一个智能合约就是一个代码集合，位于以太坊区块链上的特定地址；它是以太坊账户的一种类型，类似于一个用户的账户。但不同之处在于：用户账户完全由用户控制；智能合约账户则由其自身的代码控制，而该代码则由来自另一个账户的交易触发。
「另一个账户」可以是某个购买你的 NFT 的人的账户。因此，这些合约生成器允许你设定规则（代码）。这些规则包括：你的 NFT 以何种方式购买、出售或转让；当 NFT 在二级市场出售时，谁可以获得版税；以及此人可以获得售价的百分之几。
智能合约将包含与你的 NFT 相关的信息，包括 IPFS 存储地址。只有当智能合约完成编译和部署之后，你才能在区块链上发布你的 NFT（铸币/铸造）。
以下这些平台可让你轻松生成作品和智能合约，并将你的服务和产品作为实用程序出售。

GENIFTY

Genify 平台是一个用来生成 10,000 个 NFT 的工具。按照以下步骤，你能够生成 NFT 作品集、智能合约并完成铸造：
  • 上传包含图层的文件或逐个添加图层。
  • 为任何层设置稀有度；这与特定层在集合中出现的频率有关。然后一键生成你的作品集，元数据与 NFT 作品集一起生成，你可以将其上传到 IPFS，下一步是生成智能合约。
  • 通过输入编译你的智能合约；NFT 的名称和符号、最大供应量、每个用户的最大铸造量、部署智能合约后可供铸造的 NFT 数量以及 NFT 的售价。
  • 但是，你不能直接在平台上将智能合约部署到区块链。你可以依赖外部工具如 remix.eth，它可以让你进行智能合约的部署，以及 NFT 的铸造。

NFT 艺术生成器

NFT 艺术生成器平台允许你上传图层、生成作品集与智能合约，以及铸造 NFT。步骤如下：

  • 创建你想要的图层，并将用 Photoshop 等图像编辑工具制作的图像上传到这些图层中；为任何图层设置稀有度。

  • 一键生成你的作品集和元数据。

  • 为你的作品集创建智能合约；首先决定要在哪个网络上发布。该平台支持以太坊和更便宜的选项，如 Polygon 网络。

  • 选择该平台提供的智能合约模块之一；平台提供具有不同复杂性的不同智能合约模块，这可能会影响在区块链上部署的成本。不同的模块包含平台提供的四种 NFT 铸造合约中的一种或多种，铸造合约包括：定期铸造、空投、现在铸造稍后显示，和白名单。

  • 选择任何一个合约模块时，你都可以输入作品集信息，并将图像和元数据上传到 IPFS。智能合约设置完成后，你可以轻松地将其从该平台部署到区块链。

  • 合约部署后，铸造按钮就可以使用了；你可以将其嵌入到任何网站上供他人铸造 NFT。

Moonwalk

Moonwalk 平台为品牌和创作者制作生成性 NFT 作品集提供便利，该平台同时推出了一个名为 moonwalk 钱包的网络应用程序，用于销售他们的 NFT，吸引社区用户，并将以实用程序的方式提供其服务和产品。
他们使所有创作者都可以轻松地使用 NFT 构建独特的社区体验，而无需编写代码。所有这些都由 Moonwalk 团队处理。

ZERO CODE NFT

Zero code NFT 是一个无代码智能合约和铸造页面构建器。
Zero code NFT 只是一个 NFT 智能合约平台，因此，你只能使用外部平台来生成作品和元数据，并将其上传到 IPFS。
该平台支持多个网络，包括以太坊、Solana、Polygon 和 Avalanche。
你可以通过输入一些常规的信息来编译智能合约，例如指向图像和元数据的 IPFS URL、作品集大小、名称、价格、符号、作品集描述和项目 URL，然后你可以设置的功能有：延迟显示、白名单等。
最后一步是一键部署智能合约。这个平台的一个优势是，你可以从这里部署一个与你的智能合约相关联的铸造网站，任何人都可以通过该网站铸造你的 NFT。

UNLOCK

现在，如果你已经创建了 NFT，就可以将它们用作访问网站上付费内容的钥匙，或 Discord 、Telegram 等平台上的在线社区的访问密钥。
解锁协议（The unlock protocol）允许你创建一个锁定合约，并将它映射到你的 NFT 作品集的合约上，从而使你的 NFT 成为打开这些锁的钥匙。
你可以将这些锁放在网站或平台上，只有持有你的作品集中任何 NFT 密钥的用户才能获得访问权限。
以上内容只介绍了一部分无编码 NFT 平台。市面上还存在很多类似平台，并且正在构建更多的让 NFT 生成更简单高效的平台。此外，大多数平台仍处于 alpha 和 beta 阶段，随着时间的推移，这些平台将添加更多功能。
