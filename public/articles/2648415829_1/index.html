GEM：最流行的 NFT 交易聚合器

作者: 产品公会 Will 发布时间: 2022-04-04 11:39:36
上线三个月总交易额达4.1亿美金。

NFT 聚合器是近期 NFT 热潮中出现的一种新产品。它们允许收藏家在一次交易中批量买卖他们的 NFT。

GEM 是目前最棒的 NFT 市场聚合平台。

NFT 生态痛点

01

◦  购买 NFT 的主要问题之一在于，限制了用户一次只能购买一个 NFT。如果想要大量购买一个优质项目的 NFT，用户必须逐个资产进行购买，并进行单独的购买交易。

◦  用户不得不执行多个交易，消耗大量 gas，速度也很慢。这意味着其他人可以在你逐个购买时，购买你的目标 NFT。

◦  人们应该高效地一次购买多个 NFT、人们应该能够一次在多个平台上购买多个 NFT，因为流动性现在已经开始扩散到其他市场，如 Rarible、X2Y2、NFTX 和 LooksRare。

NFT 聚合器的出现解决了这个问题。

GEM 的主要优势

02

◦  GEM 直接集合了多个平台 (其中包括 OpenSea、Rarible、LooksRare、X2Y2、NFTX 和 NFT20) 上的 NFT 挂单。其最终目标是整合所有 NFT 市场的挂单，用户再不需要平台间反复横跳的比较价格；

◦  NFT 购买的用户体验变得更接近网购，有了购物车的功能，让用户可以一次性购买多个 NFT；

◦  比在交易所一个一个地购买省钱，在 GEM 上批量购买 NFT 最高可省 33% 的手续费；

◦  GEM 接受所有的 ERC20 代币作为支付币，用户可以一次性支付多种货币，极大增加了支付灵活性；

◦  GEM 会提供平台上 NFT 的 Dune Analytics 的数据分析，用户可以直接在 GEM 平台上查看关键数据（销售额、地板价、买卖活动、持有分布等）。

GEM 使用体验

03

GEM 界面干净简单，信息框架清晰，用户不太需要学习就能上手，功能都出现在该出现的地方。目前，GEM 还处于 beta 阶段，一些功能还是缺失的，比如收藏夹和查看单个 NFT 交易历史的功能。

◦  GEM 最大的亮点是批量购买 NFT，还很贴心地提示了本次交易省下的金额。虽然没有钱买 Azuki，但是并不妨碍把我喜欢的 Azuki 甩进购物车。

◦  同样，在 GEM 上批量挂单也会更便宜，方便用户把手里的项目批量卖出去。

◦  以 Azuki 为例，GEM 用到了 Dune Analytics 的数据，给用户提供了比 OpenSea 更多方位的 NFT 数据。

◦  用户结账时，可以使用多种 ERC20 货币一起结算，如果你的 eth 不够多，可以添加其他的代币当作支付手段。(Fig.3)

◦  下面两张图显示了在 GEM 上的一笔交易记录。用户花了3亿4千万个 SOS 代币买了 8个 NFT。GEM 直接自动通过 Sushiswap 把 SOS 换成 WETH，省下了不少的 gas。

GEM 的竞争对手

04

目前，GEM 与很多交易所是合作关系。但 GEM 还是有一个竞争对手：Genie.xyz。

Genie 也是 NFT 聚合器，和 GEM 构成竞争。两个网站的的 UI，不能说高度类似，只能说一摸一样。带着浓烈的好奇心，结合数据和社交动态，总结出了几点主要的区别：

◦  相比于 Genie，GEM 平台上的资源要好得多。GEM 拥有 OpenSea、LooksRare、X2Y2 和 NFTX 的数据；而 Genie 只有 OS、Rarible 和 NFTX。（仅这一点上，GEM 就实现了降维打击）

◦  GEM 提供数据分析，方便用户分析项目。Genie 没有类似功能。

◦  GEM 还有 FLASHBOT（防夹工具）功能，可以防止用户因为被恶意刷单之后，交易失败矿工费被扣除的情况。

◦  根据 Discord 社区反映：Genie 的数据很慢，数据过时，导致用户体验很糟糕，很多人都是原先的 Genie 用户，渐渐都移步到了 GEM 社区。

◦  可以感受一下两者真假美猴王般的画风。（本人亲测，Genie 的网站确实不好用，比 GEM 要慢）

GEM 数据分析

05

◦  截止4月1日，GEM过去24小时促成了2300笔交易，交易额达到1400万美金。GEM 上线之后3个月达成总交易额4.1亿美金，非常杰出的成绩。

◦  GEM 和 Genie 两个平台的交易数据对比也很说明问题：自 GEM 于2022年1月末上线，交易额在3月之后实现赶超，Genie 显颓势（Fig.8）。

GEM 社区&平台规划

06

◦  GEM 的社区良性讨论居多：社区成员一直在给开发者提供需要的功能，开发进度也透明。做到了 build 用户需要的 NFT 平台。

◦  GEM 的工程师 VASA 也一直在推特上更新平台进展和回答社区问题。

GEM 的生态合作

07

目前，GEM 还没有和资本达成正式投资，但是已经和多个 NFT 平台及项目方达成合作。

此外，小交易所目前和 GEM 的各种合作，确实有种快要和 Opensea 平分秋色的感觉。

◦  X2Y2 和 GEM 合作，取消市场费 (marketplace fee)，并且退还用户之前在 GEM 上支付的 gas；

◦  GEM 和 LooksRare 达成合作，Gem 将能自动帮助用户认领 LOOKS 奖励，出售 WETH 奖励，并兑换为 LOOKS 进行再质押获得复利，并且用户无需支付 Gas 费；

◦  FloorDAO 和 GEM 合作，为 GEM 添加了5百万美金的 CryptoPunk金库流动性；

◦  受欢迎的 NFT 属性调查插件——Trait Sniper 也和 GEM 达成合作。

GEM 发币之后的玩法想象力十足：如果团队可以把 GEM 代币和平台的交易流程挂钩，每天 GEM 的交易额都是肉眼可见增长；后期大量的交易额会对 GEM 代币产生非常正面的赋能，这个项目非常有潜力。

GEM 目前支持全部主流 NFT 平台，以后肯定还会继续增加。高质量的用户体验和优秀的、以用户为中心的功能让 GEM 在 NFT 生态圈中有着得天独厚的优势；以后很可能其他交易所都会成为 GEM 的库房，GEM 把握流量入口。除非各个 NFT 交易所推出独特的功能，不然，大家首选入口可能都是 GEM 这样的聚合器。

不得不说，GEM 的出现确实是 NFT 生态的一颗宝石，闪耀着更多可能性的光芒。
