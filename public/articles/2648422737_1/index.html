NFT 会员机制打法：面向品牌和用户建设的指南

译者的话

基于 NFT 的会员机制为品牌和用户构架起全新的关系纽带，它通过打造稀缺性、激励性和流动性，让品牌和用户都能为其赋予用户身份 NFT 持续增长的价值，并以无感的支付方式实现非加密用户丝滑体验。

几十年来，专属会员机制一直是品牌与最活跃的客户之间建立深度关系的方式。许多企业对自己的会员机制策略都至少会进行部分定制；然而，大多数机制则遵循以下两种基本模式中的一种：

付费会员：用户通常每月或每年支付固定数额的美元。相应地，他们会获得预设好的会员专属权益包。这些权益包括活动门票、访问优质内容的许可、产品折扣等等。这是时下最流行的会员模式，像 SoHo House 这样的俱乐部、Costco 之类的零售商、《纽约时报》等出版媒体都采用该模式。

免费会员：用户不需要为会员身份支付任何费用。相反，注册之后，基于个人购买活动或相关标准，用户获得不同层级的权益。这种会员模式通常不是被用于为社区设立门槛，而是用来养成客户忠诚度，如 Walgreens 的 myWalgreens。

这些传统模式本身没有什么问题，随着非同质化代币（NFT）和相关 Web3 技术 的引入，会员机制也正在获得它所亟需的升级。而这些变化正同时为企业和 NFT 持有者释放新的增长机会和收入激励。

如果你一直在考虑把 NFT 纳入你的业务、或是打算自己加入一个基于 NFT 的项目，这篇文章包含了你所需要知道的一切。

什么是基于 NFT 的会员机制？

简而言之，基于 NFT 的会员机制，是将 NFT 作为访问秘钥来解锁多种服务和奖励的会员专属机制。它也被称为＂NFT门槛＂，这些会员机制使用区块链技术验证 NFT 的所有权，并授予持有者各种会员专属的权益。

与传统的会员机制一样，基于 NFT 的会员机制也能采取许多不同形式。例如，NFT 所有权可以解锁像 BAYC 这样的专属网络社区，为用户提供 LinksDAO 之类的 DAO 的投票权，或是获准进入像 Gary Vee 的 Flyfish 俱乐部 这种私人餐饮俱乐部，体验烹饪的乐趣。

基于 NFT 的会员机制所提供的此类权益，会因品牌的核心产品不同而有所区别。例如，OneOf 的 OnePass 是一个由 2600 个生成性艺术 NFT 组成的集合，OnePass NFT 给持有者提供了一整套与其商业模式相对应的线上和IRL权益。持有者可以提前获得 OneOf NFT 的投放、受邀参加 OneOf 的虚拟和现场活动、进入 OneOf 的私密 Discord 频道，等等。另外也有不同层级的会员身份，以满足不同买家的需求。比如，拥有 Titanium 会员身份 NFT，用户可以在一年内获得 1160 万个餐饮、娱乐、旅游和零售合作商的额外权益。

为什么选择基于 NFT 的会员机制？

基于 NFT 的会员机制比传统会员机制更有利，其原因有很多。首先，由于利用了区块链技术，会员身份 NFT 无法伪造，也不可能进行复制。这有助于品牌防范欺诈，个人用户也不用担心实体卡遗失或忘记登录信息。

也就是说，基于 NFT 的会员机制，其主要好处在于促进了品牌和持有者之间的双向合作。借由稀缺性、激励性和流动性，会员 NFT 确保了品牌和持有者与项目未来的成功步调一致。

接下来让我们深入地看一看，前文提到的各项——稀缺性、激励性和流动性——如何在 NFT 会员机制中发挥服务品牌和持有者的作用。

1. 稀缺性与会员身份 NFT

在接受 nft now 采访时，OneOf 联合创始人兼首席执行官 Lin Dai 解释说，有了基于 NFT 的会员机制，企业就能更好地＂打磨社区，并为其最忠诚的客户提供超级服务＂。简而言之，与其采用＂一招通吃＂的方法，通过 NFT 对会员身份的数量进行限制并创造出稀缺性，品牌能够更好地开展规划，专注于如何为核心持有者提供卓越的服务体验。

值得注意的是，持有者并非是唯一从中受益的人。Dai 解释说，通过提供能产生更深层次共鸣的权益，品牌会＂建立一个超级活跃的购买者构成的社区＂，他们是＂品牌的忠实拥趸和代言人＂。

这与许多其他传统的会员机制有着根本的区别，后者无法对能够达到某种身份地位的会员数量进行限制。例如，我们来看看航空公司的奖励机制。会员资格不限量导致过多会员争抢数量有限的权益，或试图同时享用他们的奖励，比如在热门航线上搜寻座位。

2. 对持有者和品牌的双向激励

在典型的会员机制中，会员证的经济价值不会随着时间的推移而增加。但对 NFT 来说，情况则完全不同，这就创造了一个全新的态势。有史以来第一次，会员机制的价值 —— 以及个人所持有的会员证的价值 —— 不再完全由品牌方来决定。反而，它成了双向的。

由于会员身份 NFT 本身就具有货币价值，所以随着时间推移去提高会员身份的价格，成了持有者和品牌共同的动机。

＂会员身份 NFT 的价值，同时取决于品牌对提供效用和权益的承诺，以及会员为推广品牌所付出的共同努力，这就创造出了一种真正的合作伙伴关系，＂ Dai 解释说。对于传统的会员机制，企业可能会变得自满，并逐渐削减会员权益。同样，用户可能会流向提供类似权益的其它品牌。然而，有了 NFT 会员机制，双方都有动力去积极寻找提升品牌品质的新途径，进而也提升了代表会员身份的 NFT 的价值。

3. 流动性：出售会员身份 NFT

或许，将传统会员机制和基于 NFT 的会员机制区分开的最重要差异在于，会员身份 NFT 能够在公开市场上进行交易。如果用户试图提前退出，许多传统会员机制会向其收取几个月的费用。更过分的是，品牌有时会迫使用户在一系列网页中完成确认步骤，让退出的过程极其困难。

基于 NFT 的会员机制则为持有者提供了灵活性。

如果生活方式或需求发生了改变，你可以轻松地在任何自己喜欢的市场上转售会员身份 NFT —— 并且真的会因此赚到钱。Dai 说：＂身为会员期间，你不仅享受了多年的权益和福利，你的影响力，连同社区的影响力，现在都反映在了会员身份 NFT 的价值中，当转售 NFT 并退出该机制时，你又再次从中获得了回报。"

与此同时，由于智能合约中可以嵌入二级版税结构，在会员身份每次转售时，品牌都可以获得预先确定的版税比例。并且，这也进一步强化了激励机制的一致性：会员身份 NFT 的增值，既可以让当前的持有者在转售时获利，也可以让品牌从二级版税中赚到更多钱。

在某些情况下，比如 Vaynerchuk 的 Flyfish 俱乐部，代币持有者还可以将其会员身份租给其他人一段时间。对于持有者来说，这使得他们可以把会员身份当作一种收入的来源，同时还能时不时地享受一些特权。而这对于那些考虑购买 NFT 并加入该会员机制的人来说，也是给了他们一次尝试的机会。

这种模式的成功是显而易见的。以 Flyfish 俱乐部为例。1500 个 NFT 在几分钟内全部售罄。其 NFT 的初始价格约为 2.5 以太坊。而目前的转售价值几乎是这个数字的翻倍。

且慢!  好事多磨

遗憾的是，对大多数人来说，获得传统的会员身份要快得多，也容易得多，因为他们不需要克服任何技术障碍。为了使基于 NFT 的会员机制持续加速普及，企业必须投入资金和精力来改善入门体验。而对于目前正在考虑采用基于 NFT 会员机制的品牌来说，尽可能地减少非加密用户的学习成本则至关重要 —— 理想情况下，要达到用户与 Web3 技术无感交互的程度。

为此，找到正确的流程来促进用户引入是关键。Dai 指出，这是 OneOf 在创建 OnePass 时考虑的主要因素之一，也是他们最重要的价值主张之一。用户只需一个电子邮件和一个电话号码即可完成注册。当涉购买 NFT 时，可以用任何主流信用卡来完成支付，所以用户无需经历购买加密货币的麻烦操作。

结果显而易见。通过优先考虑易用性和用户体验，OneOf 已迅速成长为非原生加密用户中的顶级 NFT 平台之一。

为了效仿这些成功案例，考虑采用基于 NFT 的会员机制的品牌，在其网站和平台上实施该机制时，需要仔细审查各种选项，找到那些提供最丝滑的入门体验的选项。比如 有许多支付提供商 无需使用加密货币就能让个人完成支付。
