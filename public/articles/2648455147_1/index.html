华人 Web3 浪潮。

伴随着 Web3 逐渐成为一种趋势，越来越多的华人纷纷投身于这股浪潮之中，积极参与相关建设。

根据Folius Ventures的研究，中国互联网精英群体正成为Web3领域华人创业者的有力支持。据预测，在未来三年内，每周将有1-2个顶尖团队加入Web3创业浪潮。

相较于中国Web2的限制性环境，Web3的迅猛发展成为这些人才跨界探索的重要动力。类比于过去的A股、H股和纳斯达克，Web3目前正处于快速发展的阶段，为有创业意愿和技术能力的人提供了巨大的机遇和潜力。

华人在这场新浪潮中扮演着相当重要的角色。他们凭借着自身的智慧和努力，积极参与到各种项目中，为 Web3 的发展贡献着力量。他们有的是创新者，带来了新的理念和技术；有的是建设者，用实际行动推动着 Web3 的前进；还有的是传播者，让更多人了解和关注 Web3。

#01

Element 

NFT Marketplace 

●

Twitter：https://twitter.com/Element_Market

中文Twitter：https://twitter.com/Element_Chinese

官网：https://element.market/ethereum

团队：

创始人：

王峰是NFT市场Element、加密媒体 MarsBit News创始人，曾担任蓝港在线创始人、董事长兼CEO，北京大学工商管理硕士。1999年至2007年之间，担任金山软件公司高级副总裁。2010年参与创办极客帮技术天使基金。2014年率领从事游戏研发制作的蓝港互动在香港上市。2018年发起火星财经和共识实验室。

投资方：

2021年5月，NFT资产聚合拍卖平台Element（加密元素）宣布累计获得1150万美金天使轮注资，本轮估值5000万美金，由SIG和Dragonfly Capital领投。红杉中国、Draper Dragon、INCE Capital等投资。

概况：

简介：

Element 是一个多链聚合的 NFT 市场，旨在建立一个社区驱动的一体化市场，具有最佳的交易体验和最低的准入门槛。Element 为合作伙伴提供独家页面、收藏认证和 launchpad服务，以促进项目开发和参与。

公链部署：

目前 Element 支持 BNB Chain、Ethereum、Polygon、Avalanche。正在努力将其部署在更多的区块链上。

项目亮点：

首个社区驱动的聚合交易市场，省钱省时间。为用户提供最好的流动性、最低的交易成本、最高效的交易功能。

🤝 优化的智能合约 + 批量挂单及取消，最高节省 49% Gas费

👍 聚合主流市场流动性 + 跨市场挂单，真正的流动性

😄 业界最低交易费 + 交易奖励，带给你最低交易成本

⚡️ 实时版税 - 创作者在每一笔销售完成后可以立刻获得版税，无需等待

🎉强大的工具集，合集Offer + 混合支付/稀有度排名

🔘紧跟市场热点行业趋势，不断推陈出新升级优化产品，带给用户更好的使用体验，比如以下这些：

🟠多市场挂单，目前支持 Element、OpenSea、LooksRare、X2Y2 等交易市场。

🔵超强的数据分析功能，合集页增加Sell Wall高级图表，散点图等功能。

行业竞品：

一直以来，NFT总给人以流动性差的印象。不少人体验过将自己的NFT挂单到OpenSea，却几个星期都没有人购买的情形。最后只能被迫降价挂单，甚至以低于地板价的价格出售，以期望尽快成交。同时，OpenSea 也因为中心化饱受诟病，这也给了其他 NFT 交易市场一些发展空间。

X2Y2、LooksRare 等 NFT 交易市场的出现，虽然解决了中心化问题，甚至通过激励措施吸引了不少用户，但是从本质上来说，NFT的流动性问题并未解决。Element Market 是一个 NFT聚合交易市场，通过整合多家NFT交易平台的挂单，使市场中等待交易的NFT得到更多曝光量，从而提升了NFT 在市场中交易成功的概率，并将NFT市场推向了聚合交易的时代。

经济模型：

Element市场推出Element DAO，将推出ELE治理代币。

2024年3月4日，Element市场宣布正式推出Element DAO，Element DAO 未来将行使 Element 协议的使用权和决策权。

Element DAO将专注于革新Web3 DApp服务，包括NFT交易服务。早期核心贡献者，特别是Element NFT市场正在将其领先的 NFT 技术开源给 DAO。这包括多链 NFT 市场、ElementEx 协议和创新的 NFTaaS（NFT 即服务）技术堆栈。

Element DAO将推出治理代币$ELE，并将空投主要分配给使用其服务的社区。未来发展和使用将由$ELE社区管理和监督，即将发布路线图公告。

#03

Unipass 

●

项目播客：https://medium.com/unipass

官网：https://unipass.id/

推特：https://twitter.com/UniPassWallet

团队：

创始人是知县，目前 UniPass 团队总共 10 人，绝大多数都参与到研发工作中，项目也只是基本结束了种子轮融资。团队希望走长期路线，目前的任务先做好产品和应用场景，并不打算开展对外宣传的工作，因此人们很少听到 UniPass 的声音。

概况：

简介：

UniPass 是多链统一加密身份应用，提供基于电子邮件的非托管社交恢复Wallet解决方案。可以通过无密钥的方式控制用户的加密身份。还可以通过加密方式验证多链地址甚至社交帐户。

在前端功能上，UniPass 与 Web3Auth 类似，都有提供社交认证、社交恢复Wallet等功能。

Web3 身份聚合：基于密码学实现多链多账户的聚合。通过这种聚合方式，可以将用户在不同链上不同地址的行为聚合到一个身份 ID 中，实现信任的传递。简单说就是可以最大程度提升用户的链上行为评分。

Web2 身份验证：在智能合约中提供验证电子邮件地址、Twitter 账户、Discord 账户的能力，从而在 Web3 中原生提供 Web2 身份信息。

单点登录与访问门户：通过使用 UniPass id，进行多应用的统一登录，并为用户提供统一的 Web3 门户导航。

项目亮点：

我们将UniPass与Torus和Magic进行对比，从三个方面挖掘项目亮点。

利用邮箱进行身份认证和密钥管理方面:

  • Torus:

基于OAuth进行用户认证，支持多种社交平台如Discord和微信。

采用“门前签名”方式，认证后进入区块链流程。

第一版产品与邮箱关系不大。

第二版“Open Login”功能支持邮件作为2FA。

  • Magic:

将邮箱作为主要的登录和签名方式。

用户输入邮箱后，收到邮件链接进行身份验证。

将邮件视为与用户的隐私通信渠道。

理论上也可使用微信服务号发送验证消息。

  • UniPass:

与前两者不同，采用密码学角度。

创新性地将50年历史的DKIM反垃圾邮件协议应用于Web3。

在区块链上直接验证邮件，实现“社交恢复”功能。

将古老的Web1协议带入Web3世界，横跨两个维度。

因此，Torus和Magic更多停留在Web2.0的邮件认证维度，而UniPass则将邮件置于Web3.0的密码学层面，利用DKIM在链上验证邮件归属权，实现创新的“社交恢复”功能。

密钥托管程度：

  • Torus:

采用9个节点的门限签名方案。

每次随机选择5个节点获取密钥分片，在本地重组解锁密钥。

部分去中心化，无需完全托管密钥。

  • Magic:

依赖亚马逊的KMS(Key Management Service)密钥管理服务。

关闭了对KMS的访问权限，无法直接访问。

需持续为KMS服务付费，否则服务中断，无法解锁密钥。

Torus和Magic的盈利模式是对活跃用户收取类似包月费用，说明用户依赖于它们的服务。

  • UniPass:

正式版本将采用本地密钥存储，密钥不可导出。

前端可独立运行，无需后端服务支持，可放在IPFS等开源平台。

目前使用PWA技术，可在桌面缓存类App，支持离线使用。

通过邮件实现“社交恢复”功能，不依赖UniPass后端。

因此，Torus部分去中心化，Magic完全依赖亚马逊服务，而UniPass则采用本地密钥存储和前端独立运行的架构，提高了自主性和可用性。

环境安全方面：

  • Torus:

采用较为安全的方式，密钥仅在会话期间存储于Session Storage中。

每次访问时解锁一次密钥，关闭访问后直接释放密钥。

相比早期网页钱包将私钥存储于Local Storage，安全性更高。

  • Magic:

存在依赖亚马逊服务的风险。

如果用户邮箱丢失，资产可能会遭受损失。

实验证明，即使在新IP下登录，Magic也能成功访问账户。

  • UniPass:

将用户邮箱视为高度隐私信息。

大量使用零知识证明技术，抹去邮件中的邮箱信息。

在区块链上验证邮件归属权，而非存储邮箱。

花费大量精力保护用户隐私和安全性。

因此，Torus采取会话密钥存储确保安全；Magic存在邮箱丢失风险；而UniPass高度重视用户隐私保护，通过创新技术实现邮箱信息脱敏。

总的来说，华人在Web3领域面临挑战与机遇并存的局面：

文化和认知差异：Web3是一个全球性的领域，不同文化背景的人可能对Web3的理解和应用存在差异。华人需要更好地理解和适应国际社区的思维方式和价值观。

资源和市场份额竞争：Web3领域的投资和发展主要集中在美国和欧洲，华人可能在初期面临资金、人才等资源的竞争压力。

监管环境:不同国家和地区对Web3的监管政策存在差异，华人需要密切关注并适应当地的监管环境。华人需要在合规的前提下寻找发展机会。

但展望未来是充满希望的，随着技术的不断进步和经验的积累，华人在 Web3 领域有望取得更大的成就，为中国区块链行业的发展带来更多积极的影响，并且在全球范围内发挥更重要的作用。
