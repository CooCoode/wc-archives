Web3 和 NFT 中的匿名性问题

作者: 翻译公会 yu zhu 发布时间: 2022-07-04 17:34:30
分类: 身份, 匿名, 隐私
保持匿名的好处和危害。

很多人都以匿名身份来参与 web3 和 NFT 世界，保持匿名有哪些好处，又有哪些危害呢？本文采访了四位不同程度匿名的 web3 社区成员，从正反两个方面讲述了他们对这个问题的理解。

”

Web3 的核心精神是去中心化所有权和隐私，所以匿名和假身份在 Web3 领域也非常普遍。以比特币的创造者中本聪为榜样，很多加密社区中的人都在以多个虚拟身份活动。有些人是想保护隐私，有些人是希望通过匿名来获得更多的行动自主权，还有些人认为这是融入加密文化圈子的有趣方式。

但不是所有人都这么想。在 NFT 领域，化名和匿名，正在引起新一轮的争议。

01.

化名 vs. 匿名

首先，我们要来辨析化名与匿名之间的不同。化名是以一个可以被识别，但是无法与你线下真实身份关联的线上身份。NFT 社区中的很多成员都选择使用 ENS 地址和 PFP 图片头像作为身份，这样就是以化名活动。

现在的区块链就是完全基于化名的。用户的链上身份是用他们的钱包公钥，由一串字母和数字组成。但是从技术角度来看，可以跟踪到他们钱包关联的交易，再把他们在各处公开的个人信息拼凑到一起，所以偶尔你可以从这些信息中察觉出账号背后的使用者。

而匿名性，则是你完全无法跟踪或者识别真正使用者的任何信息。举个例子，比如在博客上的访客评论，如果不使用 IP 地址或者后端的 GPS 数据，你完全无法知道谁发了这条评论。

02.

可追责的重要性

2022年2月，BuzzFeed 的记者 Katie Notopoulos 挖掘出了无聊猿俱乐部的四位使用化名的创始人的身份信息。虽然这些创始人们在推特上使用化名，但是他们在公开发行的商业文档中披露了真实的姓名。用这些姓名再稍微挖掘一下，就可以发现他们的真实身份了，这也是几位创始人可以预见到的。面对公众的反对，Notopoulos 坚持，像无聊猿这么大市值的项目，如果创始人还保持匿名，就会减少了创始人的责任。

在某种程度上她是对的。

最近，使用化名 ZAGABOND 的 Azuki 联合创始人，在 Mirror 上发表了一篇题为《A Builder’s Journey》的文章。发这篇文章的本意是为了在熊市坚定信心，但 ZAGABOND 在文章中自曝了在创建 Azuki 之前，曾经参与过三个失败，甚至被认为是跑路的产品：CryptoZunks，Tendies 和 Phunks。这迅速引起了众怒，NFT 圈内很快展开了激烈讨论，谴责他诈骗钱财、进行性别引诱，以及其他不诚实的行为。在 24 小时之内，Azuki 的地板价下跌了约 2/3。

ZAGABOND 与 Andrew Wang 在推特空间进行了一场对话，并大谈产品-市场的不匹配，和创作者-消费者的期望脱钩，并拒绝承认他的过往污点。

“作为一个创作者，我觉得创作者的愿景和消费者的期望存在着脱节。从创作者视角来看，只要创建项目，然后按照承诺完成交付就可以了；但是消费者会期望创始团队永远为这个项目工作下去……就因为创建了一个项目，我就欠社区一辈子了吗？”

他的回应进一步质疑了匿名创始人之间的动机不一致。

任何创新都可以被好人和坏人利用。但是很讽刺的是，由于 Web3 的隐私性，在以去信任化为前提的区块链空间里，比其他地方更需要信任。因为 Web3 领域的去中介化，我们现在需要对我们的交易对手方有更多的信任。同样。我们对那些可以影响我们在 Web3 领域购买行为的人，也需要更多的信任。但不幸的是，在某些情况下，坏人使用匿名作为掩护，在Web3 领域进行着黑客破坏、卷钱跑路、内幕交易、操控股价等种种罪行。

但是替代身份不一定都是恶意的。匿名身份可以是一种自我表达，也可以是一种心理上的“重新出发”，让人们在线上“成为”一个不同于真实世界的，全新的自己。边缘群体可以通过使用假身份来远离偏见和骚扰，同时获得一些他们本来用真实身份难以获得的机会。

Coinbase 前 CTO Balaji Srinivasan 在 Cryptocurrency and Hayek 大会上发表的演讲中，提到了另一个关键的好处。

化名经济是“孔武有力的古典自由主义” 能够在今天的信息环境中存活下来的重要基础。比起让人们超越性别和种族来思考问题，或者停止歧视，不如把这些信息完全隐藏起来，只留下图片小头像和完全虚拟的身份。

对于创意工作，匿名性更是为作品增加了一层价值，匿名带来的神秘性会吸引来更多的关注。如果 Bansky （一位匿名的英国涂鸦艺术家）是用真实身份进行创作的，他的作品还能卖这么贵吗？也许会，但我们永远不会知道结果了。

我们采访了一些 NFT 社区成员，其中有使用化名的，也有实名的。来听听他们在这个最具话题度的问题上的观点吧：

匿名性对于 NFT 和 Web3 来说是好事还是坏事？

 Rebecca Orlov，Epic Playdate 创始人

 REBECCA ORLOV 的 Twitter NFT 头像：BOSS BEAUTIES 

匿名性通常是正面的，带来生机的。它提供了一个无偏见的、更坦诚的空间，让消费者可以放下他们的各种滤镜。话虽如此，过快的发展、错位的社区、一些 NFT 项目进行虚假炒作，也可能会导致情况很快变糟糕。用户可能会陷入困境，不做研究，或不按自己的内心，作出一些情绪化的决定，使得自己陷入财务困境。

虽然现在女性加密货币持有者只占总用户数的 26%，但是女性要更多地参与到 Web3 的大潮中来。这很快就会发生。我们要用可见性来转变并最终消性别偏见，这也是 Epic Playdate 和 MetaMamas 一直为之奋斗的目标。有大量价值巨大的，有影响力的女性引领的项目出现在 Web3 的各个领域：NFT、元宇宙、DAO、D2A。女性应该站出来，展示她们的真实身份。女性应该被看见！

从 2022 年 1 月以来，我个人投资了 30 多个NFT 项目，所有项目的创始人身份都是公开的。我有意地这样的选择项目，因为我认为，投资是需要责任制和透明度的。这些使命驱动的 NFT 社区里充满了积极的对话，清晰的行动计划和线路图，还有快速成长的专业团队。以 Boss Beauties 为例，这是我持有的 NFT 中最喜欢的项目之一。它的创始人 Lisa Mayer 不但公开了自己的身份，还带着她可爱的宝宝参加了上周末的 VeeCon 会议，她一直在履行其 NFT 项目所提倡的使命，即"帮助女性接触到技术、领导力和创造力的前沿”。当我知道这个项目背后的是这样的人在领导，会让我愿意投入更多的时间和资金，并且这样的项目更值得信赖。

当听到各种项目从他们的社区卷钱跑路，这对那些真正想要创造些有意义的东西的人来说是非常失望的。我总体持乐观态度，从二八定律的观点来看，多数人和团队仍然很认真地想创造一些不可思议的项目，甚至有着百年规划。

 Hotshave, Nuclear Nerds 创始人

 HOTSHAVE 的 Twitter NFT 头像：NUCLEAR NERDS

就我个人而言，使用化名和匿名，主要是为了开心或者融入群体。我自己有一个独特的使用网名，对我来说，这是一个我已经接受的独特人格，给它更多的自我支配权会很有趣。我会想要在加密圈子里使用这样一个网名。

在加密领域之外，我从事过多年广告制作、故事讲述方面的工作，也算是小有名气了。所以在一开始，我们团队就曾表示过：如果你想知道我们是谁，可以很简单地在网上找到答案。我们公布了大量我们过去的作品，我们合作过的品牌，还有我们做过的项目。简单 google 一下就知道我们是谁了。当我们使用化名的时候，我们并不是真的想隐藏真实身份。

作为 Nuclear Nerds 的创始人，我们虽然使用化名，但是我们一直尽力与社区保持良好的沟通，比如通过双周的公告和每周的推特空间活动。我们一直在努力履行我们的承诺。我们也在 Discord 和推特上很活跃，在我看来，想要联系到我们一点也不难。

我个人是非常希望我们的团队成员和用户都能够感到安全。对团队成员来说，是在身份泄露以后，没有人会找上门；对用户来说，是参与的项目不会跑路，创始人关心社区和项目的成败。从根本上来说，要做到这一点，需要去建立个人品牌或者项目品牌，这需要时间。

 Mealsandeals.eth, NFT 收藏者交易者

 MEALSANDEALS.ETH 的 Twitter NFT 头像: BAYC 

我个人使用化名的历史要追溯到我进入 Web3 之前，我搞过一个关于时尚和食物领域的 Instagram 号：Mealsandeals。在这之后，无论是在推特上，还是在 Web3 领域，使用化名就很顺理成章了。当然，使用化名确实让我在线下约见他人时更困难，有时候还需要在见面前先给别人发张尴尬的自拍。但使用化名让我在面对搞社会工程的，试图模仿我的，还有其他不怀好意的人面前，感觉更安全。

匿名和化名在 NFT 领域确实是有用处的。首先是保护自己，Web3 现在像是狂野西部，如果你使用化名，至少黑客会更难对你进行钓鱼攻击或者诈骗。匿名性同样意味着，人们是仅仅通过我的内容来评价我，而不是我长得好不好看，或者我跟哪位大佬熟不熟。但是另一方面，如果你是一个作恶者，那么匿名性会让你变得更危险。通过匿名，诈骗者可以一次次地骗走大量钱财，因为匿名性让人很难对此追责。区块链的魅力就在于每个人的行为都被永久记录在链上。非常感谢像 zachxbt 的业界良心，他们不断追踪链上的行为数据，并帮忙标记出哪些账户是危险的，我们可以用这样的信息来保护自己的安全。

 RareShoe – Rare Shoe 联合创始人 

 RARE SHOE 的 Twitter NFT 头像: MUTANT APE YACHT CLUB 

在 Rare Shoe 团队里，我们不觉得匿名是件好事。所以我们选择 100% 实名上网。我们希望，通过让持有者知道我们是谁，我们想在未来做什么事情，这样的沟通可以让他们有些安全感。成为其他社区的一部分，而不仅限于我们正在建设的社区，这一点是非常重要的，这就是为什么我们有 PFP 化身，而社区也很欣赏这一点。
