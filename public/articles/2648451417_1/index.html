SeeDAO 发起人。SeeDAO 积极打造数字城邦，在全球各城市拓展基地，基于区块链目前一个以追求「美好生活」为目的的数字城邦，进行着十个城市的在地建设、搭建全球移动网络。近期，SeeDAO 在清迈生态发布会宣布:完成从 DAO 到 Polis 的转型，将建设数字城邦自主世界。

曾获得美国广播电视最高奖项皮博迪奖（皮博迪奖），曾走遍三十多个国家，在美国毕业后进军电视工业如 Neftlix 和 Vice 制作节目，持续创作，亦策划主持播客「行星酒馆」关心离散社群，分享有关路上风景和哲思。

Host 洁平

这是一个数字游民专场，当我们成为一个移动的人，游历四海，是否有思考过「公共性」，如何与在地连接？做个负责任的数字游民，甚至牧养地方。

Q1.

请分享一下在路上意想不到的困难/收获

白鱼

这两年走了美国洛杉矶、土耳其、中东卡塔尔和迪拜、黑山、香港，最近来到清迈。

在路上，最大的担忧和不便是我有很多的书没地方放。数字游民还是要有个 base 的地方，其它再到处游走。如果一直在路上漂泊，会感觉没有根。

作为游民，我没有跟在地发生很强的连接。之前一直对在地这个话题比较无感，直到出了一篇文章把数字游民比喻成类似于一种瘟疫，我开始去反思原来偏自由主义。

因为在路上的生活比较简单，甚至过于悬浮。没有景点打卡，专程去吃当地美食。比如，我在成都，没有去成都城墙，也没有去吃它的小吃；这次来清迈也是。

林东尼

去年夏天，我的人生陡然转向：先是辞掉了体面的工作，又在疫情割裂世界三年之后，第一次离开北美，回到亚洲，从香港、台湾，再到越南、泰国。

决定离开是明白了想在美国留下来，要把自己工具人化；美国的价值体系它不是我所需要想要的，于是决然割断名利，NOT IN A AMERICA.

「世界那么大，我要找个令自己开心的方式活着。」

我想找一个反面，节奏慢一点，东西好吃一点，chill 一点，对 LGBTQ 友善一点。

终于，我遇到了清迈。在这里感受到自在、舒展的状态，那是被压抑许久后的自我。

每天有一个时刻，会让我觉得很开心、很自如。比如日落时分，在自己的小院子看到水流淌下来的形态。也许这很寻常，但我却感觉有点像在和清迈谈恋爱。或许 爱上一个地方，是因为爱上在这个地方的自己。

Zik

疫情期间我在日本京都，它也希望你安安静静地待在一个地方，不要移动。

之后这大半年机票的目的地，大部分是和工作有关，目前我的生活跟着工作走。

到黑山、土耳其参加参加一些活动大会，去新加坡等第三国办理续签。

我的离开比较简单，想要和一个新地方融合的感觉反而是很难，短期的停留很难快速地了解当地。

印象深刻的有融合感，在京都的一个在地社区。我跟着大家该种田时种田，一起吃饭喝酒吃肉，感受到了对人的文化、当地生活状态在地的连接感。今年我在清迈生活的时间渐渐多了，也慢慢地有了这种感觉。

Q2.

聊聊最困难的经验

或者，如何发现自己、落实回馈

真正走进世界？

白鱼

一路从农村、小镇、城市、出国 循序渐进，思想上也一直比较充盈。

有时候喜欢在长江边坐坐，喜欢大江大河。

对我来说：具体的事情很难，比如很难和家人解释你在做些什么。

我们有一个想象的共同体，清晰内心归属在哪里。

我是一个很有契约精神的人，知道要对 SeeDAO 负责。但如果关于对在地的回馈，感觉是批判我的，目前还不清楚在地的时候对谁负责。

书店是我精神的寄托。喜欢看各地独立书店的选书，找精神上的认同。我在土耳其 Web3 咖啡店看到 NFT 的那一刻很激动。

Host 洁平

没关系，诚实比较重要。

林东尼

我是位性小众，从小就知道这个地方不是我的，那个地方也不是我的。

不停在出走，觉得自己和他人不一样。在拉扯和坦诚中，父亲对我说：「我们都是善良的人，我们都愿意付出爱， 这就是一样的了。」

Zik

重要是你想要的状态是什么？找到安心移动的点，一份在地的融入，才能让你坚持下去。让它更 organic 有机地发生。

也许我们在聊离散的前提，是有家或一个地方可以回去。而关于家在哪里，可能要用一生来回答。
