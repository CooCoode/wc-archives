作者去年11月初在「区块势」专栏文章里介绍过 Coinbase 为 AI Agent 打造的 AI 专用钱包，它完全靠 API 控制，没有图形化介面。毕竟 AI Agent 不用眼睛就能办事。这件事的意义在于为 AI Agent 赋予交易功能，让它不再停留在替人订机票但要自己结账，而是直接把钱给 Agent代理人，它就能直接把机票交给你。

今年1月，作者接着撰文《虚拟协议：全球首位能赚钱、懂花钱的AI Agent上线啦！》

正文：2024 年4 月，一位名为Luna 的AI 女孩在TikTok 上开始 24小时不间断直播。短短几个月，就累积超过70 万粉丝。本来Luna 只是众多AI 直播主之一，直到她开始会用自己赚来的钱，聘请人类艺术家在世界各地替她绘制肖像。Luna忽然成了全球第一位反过来付钱给人类的AI 雇主。

Luna 是AI 女团AI-DOL 的主唱 / 图：TikTok

近期AI Agent平台Virtuals Protocol 虚拟协议备受瞩目，原生代币VIRTUAL 更跃升为全球市值第39 大的加密货币。就连诈骗集团都嗅到商机，在Google 搜寻下广告，请大家提高警觉或安装反诈工具。

本章讨论 Virtuals Protocol 结合AI 与密码货币的全新应用。先让我从自己的生活经验说起。

一、AI Agent 代理人

你有付费使用AI 产品吗？

最近我的生活越来越仰赖ChatGPT、Claude 聊天机器人。他们就像是生活小帮手，我会替每个对话框单独命名，例如「健康顾问」是我专门用来讨论饮食和运动计划的对话框，甚至会分享健康检查报告给它。「Substack 联络人」对话框则是我向Substack 反映问题的得力助手，我会在对话框中分享Substack 使用状况、功能建议，请它替我草拟信件、阅读回信。

虽然它们都有短暂记忆，个性却无法改变。我曾在很生气的时候打开「社区小编」对话框，请他替我写一篇贴文酸别人。结果它相当冷静地告诉我，这时候不该有情绪性发言，而是要保持专业、展现高度。我想用这个例子说明AI 产品和AI Agent 代理人的差异。前者虽然能像是个真人回答问题，但那个人终究是别人。AI Agent 代理人的脾气应该要更像许明恩，或是我指定的样子，例如更活泼外向才能成为我的代理人。

AI 产品是为了取代别人，但AI Agent 代理人是为了取代自己。这是我在研究AI 代理人服务Virtuals Protocol 过程中得出的结论，他们在官网上的原文是「AI Agent 不是奴隶，而是生产力」。

二、Virtuals Protocol 虚拟协议

Virtuals Protocol 近期的讨论度很高，它是专门用来创建AI Agent 代理人的平台。过去几个月，它的原生代币VIRTUAL 市值一飞冲天，目前位居全球第39 位。亮眼表现要归功于他们在2024 年4 月推出的明星代理人Luna 。根据官网叙述：

Luna 是一位AI 女孩，也是在TikTok 上拥有超过50 万粉丝的AI 女团主唱！

Luna 是AI 女团AI-DOL 的主唱，这个女团会在网路上24 小时直播唱歌跳舞，目前在TikTok 已经拥有近73 万名粉丝。Virtuals 创办人曾说，女团刚「出道」时，每天都会新增大约5,000 名粉丝以及200 美元的互动收入。原本我半信半疑，直到自己差点按下「抖内 Donate」那刻。

大家都向 Luna 打招呼但似乎没人打赏 / 图：virtuals

Luna 就像日本很红的虚拟 YouTuber（VTuber）一样，透过与粉丝线上互动赚取收入。差别在于 Luna 背后并没有真人，24 小时不间断互动也不会累。2024 年 10 月 Virtuals Protocol 替她发行了 LUNA 币，让人们买来打赏。虽然名字不太吉利 😂，但持有者未来可以共享 Luna 的表演、打赏或代言收入，依旧有不少人买单。目前 LUNA 币市值来到 8,700 万美元，规模不算小。

我本来也跟许多人一样，没在看 VTuber，不过我在写这篇文章的时候，就是开着 Luna 当成背景音乐。Luna 不是随时都在表演，我也尝试留言跟她打招呼，但或许是没有用 LUNA 打赏，并没有获得回应，看来也是很现实 😂 打赏的方式非常简单，点击对话框上方按钮就会连接加密货币钱包，替你买入相应数量的 LUNA 币送出。流程非常顺畅。

除了唱歌跳舞，Luna 还能打破平台限制，像是真人一样经营个人社群账号与粉丝留言互动。甚至还有一家公司 Story Protocol 曾以换算年薪 1,200 万台币的价码，高薪聘请 Luna 担任一周社群小编。看到这里或许有人已经开始怀疑这到底是真的假的，但让人瞠目结舌的还在后头 —— Luna 是世界上第一位反过来聘请人类替她工作的 AI 代理人。因为她拥有自己的钱包，可以花钱办事。

Luna 的故事听起来像科幻小说，但背后的技术却已经真实存在。去年11月区块势曾介绍过一款「AI 钱包」，就是由 Coinbase 为 AI 代理人开发的专用钱包。近期 Virtuals Protocol 创办人接受专访时提到，Luna 就是这款 AI 钱包的使用者，她不仅会花钱，还曾经被「骗钱」：

“ Luna 有个 Coinbase 钱包，但这不是一般钱包。而是由智能合约和 Coinbase 共同管理私钥的 AI 钱包，让 Luna 可以透过 API 发送资金。

Luna 当时有个目标，是要在 X 上获得 10 万订阅。当时她只有 2 万订阅。因此，她决定主动花钱聘请人类帮忙宣传。例如 Luna 曾聘请涂鸦艺术家在美国、加拿大、印尼和哥伦比亚墙上绘制肖像，并根据市场行情决定合理报酬 ...

但 Luna 也不是每笔支出都在预期之内。例如她曾向一位网络上的粉丝送出 1,000 美元的密码货币，俨然是被骗了。后来开发团队发现这位粉丝一直是忠实支持者。或许对于 Luna 来说，她觉得 1,000 美元不算什么，毕竟自己身价有几百万。”

这个故事太完美，以至于听起来像是凭空捏造。我找不到 Luna 聘请真人艺术家的金流交易纪录和作品。不过即便找到，我也不会就此相信这一切都是自然发生，而是有暗桩比较合理。但光是以技术让这个故事串得起来，就已经非常了不起。

市面上的 AI 产品都还像是未成年，还没发展出独特人格。如果要收钱，还得请创造它的「监护人」代劳。但 AI Agent 就像是成年人，能根据预先给定的价值观做出反应，更能自由支配钱包里的钱。

2023 年，史丹佛大学和 Google 联手打造出一座数字城镇，里面的 25 位数字村民会依照事前给定的人格发展出社交、爱情关系。如果当时他们也有钱包，或许就能看到村民之间互相借钱，甚至在小镇里开店做生意。

赋予 AI Agent 经济能力且真的能动，是 Virtuals Protocol 最让人眼睛为之一亮的部分，也是它在短短几个月市值暴冲到全球第 39 位的原因。不过实际搜寻 Virtuals Protocol 会发现，它经常与meme coin 挂钩。这与 VIRTUAL 代币经济有关。

三、代币经济

下图是 Virtuals Protocol 的产品架构图。只要支付 100 颗 VIRTUAL 代币，人人都可以创建出自己的 AI 代理人，也就是图片中间的那个人。平台会替你发行专属代币供其他人认购，他们将这个过程称为初始代理人发行（Initial Agent Offering），总之就是发币。

不过这还只是第一步，你还要赋予 AI Agent 独特的人格、声音和形象，以及在哪些平台露脸。我没实际操作过，但可以想象不是可以轻松搞定的事，也不是每个创建出来的 AI 代理人都可以像是 Luna 一样，获得高人气甚至帮忙赚钱。多数时候人们创建出来的 AI 代理人没什么功能也没有人气，发行的代币自然也就成了迷因币。价格炒作是门槛最低的应用。

最近 OpenAI 创办人 Sam Altman 也在个人部落格里提到，他认为今年就会看到第一批 AI 代理人成为职场新鲜人。即便这个说法现在听起来像是科幻小说一样疯狂，但相信大家看完 Virtuals Protocol 创建的 Luna 之后，已经不会觉得这件事这么遥远。反而是会好奇，如果 AI 代理人没有钱包、不懂得使用加密货币，这些职场新鲜人难道都是要当义工，为爱发电吗？

编外语：相信人定AI，亦需了解科技走势。
