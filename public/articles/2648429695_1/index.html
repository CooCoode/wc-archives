SeeDAO｜朋友，NFT 和去中心化 IP

作者: Flow Xiong 发布时间: 2023-05-17 14:13:13
分类: NFT, 社区, 社区建设, IP
作者认为人们通过 NFT 来建立有意义的关系是更有效的， NFT 项目方应该致力于在社区内部建立深层次的关系。
摘要：随着新冠疫情的发展，人们越来越孤独，且人们渴望找到与自己志同道合的朋友，作者认为人们通过 NFT 来建立有意义的关系是更有效的。作者认为 NFT 项目方应该致力于在社区内部建立深层次的关系。好的 IP 是 NFT 的核心，而 Web3 等前沿技术只是用于锦上添花，作者列举了迪士尼和宝可梦的例子，因此创作一个好的 IP 和叙事是十分重要的。
概要
借助 Web3 和 NFT 的技术机制，NFT 项目可以让自己的 IP 趣味横生，让项目自身成为人们在线上线下收获友谊的土壤。
这是培育长期拥趸的不二法门。
要点

  • 互联网最伟大的馈赠，是让人们能够突破地域限制相互联结

  • 在互联网上不起眼的角落里，人们走到一起，谈论着他们喜爱的话题，在彼此之间找寻友谊和纽带

  • 由于新冠，生活再次陷入孤单──网络成了人们寻求联结的唯一渠道

  • 对彼此相连的渴望，将 NFT（一种多媒体形式）变成了社区联结的可视化机制

  • 大多数真正的、深刻的关系，都建立在大家一起为了共同的目标而努力的过程中；即使人们的相似之处并没有想象中的那么多。

  • 最初，人们加入各种 NFT 群组可能是为了赚钱，但「他们留下来，是因为人」

  • 项目应该着眼于在社区内部培育真正的关系，因为这比任何事情都更能带来长期的粉丝

  • 借助 Web3 和 NFT 的技术机制，NFT 项目可以让自己的 IP 趣味横生，让项目自身成为人们在线上线下收获友谊的土壤。这是培育长期拥趸的不二法门。

NFT 之前的生活

疫情之前，人们随着工作迁徙。
老朋友渐行渐远，新朋友慢慢出现——但人们仍然更为渴望同好。和工作伙伴一起玩耍固然不错，但人们希望能有更多喜欢同款游戏、一起做烘焙，或者在小爱好中共同探索真实自我的朋友。
许多人转向了互联网——连接全球人类的最强大工具。 围绕着共同的兴趣，无论是在小众论坛中组织运作，还是策划线下活动，又或者一起建设和玩耍，都让人们建立起了联结。
即使在情况「好」的时候，人们仍然积极地使用互联网提供的工具，渴望获得更多联接。但是，疫情来临之后呢？人们变得非常孤独……层层包围的那种孤独。
哈佛大学 2021 年的研究发现，由于疫情，「在 18 至 25 岁的人群中，有 61% 的人报告（其孤独程度）为高」。心理学家理查德·韦斯伯德（Richard Weissbourd）称：「如果你看看其他关于老年人的研究会发现，他们的孤独率很高，但似乎没有年轻人那么高」，「63% 的年轻人出现了大量的焦虑和抑郁症状」。
无法接触物理世界，迫使我们创造或重新发现有意义的线上连接方式，于是线上生活的时间翻倍了。观看派对、虚拟空间和视频群聊，这些例子都是那段时期为加强人们线上联结而建立的技术。
来源：哈佛大学关于疫情期间年轻人孤独情况和心理健康的研究
「我对年轻人的孤独程度感到惊讶，」研究带头人、哈佛大学教育研究生院（HGSE）的心理学家和高级讲师理查德·韦斯伯德（Richard Weissbourd）说。「如果你看一下其他关于老年人的研究会发现，他们的孤独率很高，但似乎没有年轻人那么高。」
如果结合美国疾病控制和预防中心 6 月份的数据，上述令人不安的统计结果甚至会更加令人担忧。该数据显示，63% 的年轻人称自己有大量的焦虑和抑郁症状。「这是一个我们真正担心的群体，」韦斯伯德说，他怀疑有几个因素在起作用。
小青年和年轻成年人可能特别容易受到影响，因为通常他们处于「从原生家庭到个人选择的家庭」过渡的阶段，韦斯伯德说──这意味着他们失去了与那些「孤独守护者」之间的关键联结。大学里的学生可能在努力适应环境，时常被思乡病烦扰；而那些离开学校的人也会感觉与重要的社会团体或社区失去了联系。他说，年轻人还要经常做出关于职业、个人生活以及人际关系的重要决定，又使得压力和孤独感倍增。
对于某些掌握小众技术的人来说，Web3 变得诱人起来。因为，建设一个摆脱大型科技企业控制的去中心化世界──这样的可能性让许多人转而加入开放社区，敲敲打打地搞起了区块链项目。
早期的区块链 ICO 时代（2014-2017）是坎坷的，但由于共同做过的工作，让许多在那时就形成的友谊和群体存续至今。然而，为了迎接下一波人潮的加入，技术的应用必须调整适应。
NFT 艺术是建立多媒体关联的方法
多媒体是一种极其强大的联结工具——与静态数字资产不同，它能激发多种感官，增强现实生活中的动态元素，让人们完全沉浸其中。
在过去，多媒体主要用于营销、广告等方面，但自从互联网兴起后，人们创造出了许许多多各种形式的多媒体，丰富了文化。Youtube、Instagram 和 Snapchat 都是用户生成的动态内容的例子，这些内容由强大的工具创造，并通过社交媒体广泛传播。
结合互联网的分享机制，用户生成的多媒体内容为社区文化制造了强大的副产品，也就是它们的纽带机制。
  • 多感官联想性的关系和联结
  • 内部迷因
  • 音乐推荐
  • 动态娱乐
  • 更强的品牌关联
  • 具有强大影响力的纽带
  • 以及更多
最近，一种组合（简短配乐 + 个人视频 + 视觉特效）极速流行起来——TikTok!
例如：如果你喜欢美式足球并且关注了 NFL Highlight 的 TikTok 账户，就很可能会收到对胃口的音乐推荐，因为这两者之间有着强大的文化关联：

  • 许多体育运动员是说唱乐迷

  • 许多千禧一代和 Z 世代也是说唱和体育爱好者

  • 说唱音乐欢快、「燃」，带劲儿

  • 体育比赛的精彩片段一般都很刺激

流行文化就是这样，人们在方方面面都会加入自己的理解，不断塑造和丰富流行趋势。例如，青年文化将 NFL 接球手 Justin Jefferson带热的 Griddy 舞，与 NBA YoungBoy 的 Right Foot Creep歌曲结合起来，成为了一种文化现象——甚至《堡垒之夜》也将这支歌舞制作成了动作特效。
来源：《堡垒之夜》中 Griddy 舞与 NBA YoungBoy 的 Right Foot Creep 歌曲结合的视频
好吧，那 NFT 呢？
NFT 许诺给我们一个社区。无论人们是否知道，它都是与各类群体建立视觉关联、围绕着小众爱好聚集到形形色色群组中的一个简单途径。
NFT 享有类似的副产品，因为它们也是一种多媒体形式（艺术+社区），但它不是音乐唱片，而是视觉化的社区关联。就像关注 NFL 的 Tiktok 会得到音乐推荐一样，NFT 作品成为了一种机制，使人们围绕共同的兴趣集结成社区，而作品则成为社区的外在表达。
例如，如果你买了像 Avime （受动漫启发的，第一个可以完全在链上定制的 NFT 项目，译者注）这样的动漫 NFT，不管是否知道，本质上都是在表达对自己深层文化兴趣──动漫的认同，而围绕它形成的社区也很可能拥有同样的兴趣。
深入 Avime 社区就会发现，他们的活动都是围绕着观看、重温动漫，聊新漫画作品，赞颂日本文化艺术等内容，因为人们总是围绕着共同的兴趣发展关系。
如果你有另一个NFT——Mannys Game 作为例子——3D 呆萌的艺术带来了另一群健康积极、才华横溢的信徒。这群人在 DC 服务器里的活动、对话和氛围（vibe）就完全不同，因为它吸引了不同类型的人，包括概念艺术家、艺术工程师，以及更多与该作品有视觉关联的人。
在一个没有这些视觉工具的世界里，我们不得不参与许多的活动，投入大量的精力来考查是否喜欢这个人或社区——很多时候，他们的兴趣实际上和我们并不一样。它并不意味着这些陌生人不能成为好朋友，只是意味着我们想要与那些能够分享我们最深层兴趣的人建立联结，全世界对线上联结的渴望正表明了这一点。
NFT 的世界里不乏大量艺作，与社区建立视觉关联轻而易举。
多媒体与互联网分享工具相结合，给我们提供了强大的关联机制，加速了建立最初关联的过程。我们可以利用这些机制来了解一个社区、品牌的氛围（vibe），并借助动态工具和内容来决定自己是否想成为它的一部分。
多媒体无处不在，而且已经被人们用了几个世纪之久了。但对这一小群技术宅来说，NFT 艺术成为了社区联结的机制。
NFT 艺术帮助人们在视觉上与社区产生联系。
深度关系是如何建立的？
大多数新的、深层次的关系是在人们一起做事时形成的——通常伴随着某种形式的竞争或共同的目标。
看一下电子游戏，来自全球各地的人们，不知道对方的姓名，却在彼此之间建立起了强大的、完全虚拟的关系——这往往是因为，在一个具有相同视觉关联（多媒体；）的游戏中，大家彼此合作去赢得胜利。
再看一下学校或工作场景，这些关系的形成大多数是因为，即使可能没有同样的深层兴趣，却共同面对着同样的压力：上同样的课、要参加同样的考试、归同一个老板管……
然而，如果你曾经安排过咖啡聊天或参加过网上活动，你就会知道，即使亲自到场，知道你们有类似的兴趣，并且有结交新朋友的心态，也很难与人建立真正的、深刻的关系。
物理世界对于建立关系是很好的，但对于发现和过滤关系却没那么有效。
在物理世界中，选择受限于物理连接能力。我们常常因为地理上接近而与一些人走得很近，即使与他们可能没有很多共同的兴趣。我们之前探讨过，即使在疫情之前，生活「很好」的时候，人们仍然渴望与同类建立联系，并转向互联网来实现这一愿望。
许多人知道网上有更多同类，并曾尝试在网上结交朋友，但都失败了。他们只是需要一个真实、有趣的机制来帮助构建深厚的友谊。
这时， NFT 就有了用武之地。
建立关系的完美配方
＂我是为了 X 而买入的，但我是为了人而留下的。＂
如果 NFT 只是艺术，这个故事可能会情节拖沓，或者可能讲不下去；但不管怎么说，都与钱有关，与人则没什么关系。
所有这一切结合起来，就形成了在网上建立有意义的关系的完美配方。
NFT 需要某种形式的经济贡献，许多人最初购买是为了获得经济回报。涉及到钱的时候，人们希望项目价格上升——没有人喜欢亏钱。
因此项目：

  • 组织社区活动来建立彼此间的关系

  • 建立 DAO 或＂工作群组＂，以进行任务委托和项目宣传，筹集资金，创造衍生品或作品，拓展业务等

  • 赞赏为营销付出的努力，和彼此之间在社交媒体上互关、回复

  • 创建拥有内部迷因和文化的社区。

  • 形成一种集体精神，即使一个项目财务表现不如意，也要和大家一起 WAGMI（We are all gonna make it，我们必将成功，译者注）
所有这些活动，无论是否有趣，加上金钱这层因素，都激励人们为同一个目标而一起努力——这正是建立深厚关系的方式。人们查看 NFT 的地板价、不同的稀有属性，在项目上合作，为项目传闻添油加醋等等，最终彼此间紧密相连。
老实说，人们所做的事情甚至可以说有点傻：一群成年人整天泡在 Discord 上，假装自己是对冲基金操盘手，倒腾些个猴子和海豹的小图片，还变成了朋友 <3
大多数人最初入局是为了发财，但一路走来所建立的友情，让他们留了下来。
小故事一则：

  • 一群孤独的小众技术宅，在找寻着人生的意义和人与人之间的联结

  • 利用多媒体副产品，NFT 让人们与某些项目和文化建立了视觉关联

  • 经济因素带来了激励，人们获得了合作的动力，觉得自己是「其中的一分子」

  • 人们因共同兴趣走到一起，为同一个目标共同努力，结成了紧密的纽带──关系就是这样产生的

人类希望从某些比自己伟大的事物中获得归属感。NFT 的经济属性 + 多媒体副产品引发的底层社区构建 + 现实中孤苦的人们——如此组合所形成的亚文化，就是我们结交朋友的完美配方 :)
令人失望的是，我们不得不花钱去弄清楚如何建立友谊。
＂我是为了 X 而买入的，但我是因为人而留下的＂。

将 NFT 的经验教训应用于知识产权的未来

我们所讲述的故事，可以在人与人之间建立起真实的联结。它也是构建一个魅力长存、能持续提供情感纽带的宇宙的途径。
最终，NFT 项目只是由去中心化的 IP 驱动的视觉化故事！
好的 IP 可以创造体验，但成功的 IP 可以捕捉想象力并与粉丝建立关系。顶级 IP，像漫威、乐高、宝可梦、马里奥、芭比等，即使你此刻没有正在接受它们的＂洗礼＂，也会窃取你的想象力、捕捉你的情感。它们创造的每一次互动，都会进一步加深你与它的传说或粉丝社区之间的某种关系。
在最成功的 IP 中，有些以孩子们为对象，但也有一些，例如《使命召唤》，则针对更成熟的受众。它们不断创造玩法，通过新颖、创新的方式与玩家建立纽带，让忠诚的、长期的粉丝能够不断享受自己喜爱的产品。例如，《使命召唤》推出了 战区，这是一款类似于《堡垒之夜》和 Apex 的多人大逃杀游戏，只是采用了《使命召唤》的元素，因此，其受众可以享受自己喜爱的 IP，同时，又通过一种年轻受众所青睐的新游戏风格建立深厚关系。
到目前为止，NFT 项目一直在围绕其宇宙努力开发相关传说、故事和背景。然而，要想成为一个伟大的 IP，我认为它们应该注重社区联结，通过创造性地使用 Web3/NFT 工具来创造有意义的体验，从而培养深厚的关系。
1. 建立一个好的 IP——如果没有好的内核，用 Web3 做任何扩展都没有意义。
2.使用 Web3、MR（混合现实）、IRL（现实体验）和其他实验性渠道，用你的 IP 创造有意义的、吸引人的情感体验。
3.用前人没有尝试过的方式来创造能让社区彼此联结的体验。
区块链能够实现稀缺性、所有权、可验证性等特性，为 IP 的体验和拓展提供了前所未有的新方式。我们开发出的游戏可以具备真正可交易的、新颖的数字资产，可以为形形色色的贡献构建出支付系统，等等。然而，这些只是创造更好的 IP 体验的工具；最终，还是需要有一个好内核。
真正的社区 = 长期的忠诚度
品牌应该关注在社区中发展真正的关系，因为这比任何事都更能创造长期粉丝！
看看一些最具标志性的 IP——许多人刚爱上它们的时候还是孩童，但即使受众长大成人，却仍然对扩展宇宙津津乐道，享受它们带来的怀旧情结，仍然是这些 IP 的拥趸。成年人参加游戏王比赛，玩口袋妖怪 GO，若无其事地去迪士尼乐园游玩，几乎和他们的孩子一样！
我们所讲述的故事，可以在人与人之间建立起真实的联结。它也是构建一个魅力长存、能持续提供情感纽带的宇宙的途径。
毕竟，有角色的故事才是好故事。
让我们来探讨一些例子吧！
迪士尼乐园：《哈啰阿古》
来源：迪士尼园区中海龟阿古（ Crush） 与孩子们互动对话
阿古（Crush）实际上不是人工智能，它是一种数字木偶！后台有一个真人，借助鬼斧神工的技术，在尼莫世界中为孩子和大人带来一场极富感染力的演出。
木偶师用机警的俏皮话带动全场观众，为许多家庭和个人建立了与尼莫世界之间的特别体验。它是如此令人入迷，以至于许多家庭千里迢迢从世界各地飞来，体验这一值得长久珍藏的时刻。
你可以在 NFT 纽约的 Doodles 活动中看到类似的、但不那么吸引人的体验。他们在屏幕上放了一个数字 DoodleBot，可能是由人工智能驱动的，你可以对它说一些短语，触发不同的反应。虽然这是一个很好的互动，但感觉它对体验 Doodles IP 没有什么作用（而且可能还花费了不少钱）。
与DoodleBot对话触发了不同的动画效果
两者的最终产品类似，但这种体验的魔力在于：人们对《海底总动员》的喜爱 + 迪士尼有意采用的巧妙技术 + 他们与社区和家庭培养的关系。阿古原本可以轻易采用反应没那么机警的人工智能技术，人们仍然会喜欢；但迪士尼之所以如此赞，是因为他们选择雇佣一个真实的人，有目的地使用工具来提升 IP 体验，为人们上演了一场精彩绝伦的、令人愉悦的演出。
让我们来探讨一下，如果还是使用同样的 AI，但只是换掉那个效果很差的 Doodle 角色，是否能够创造更有意义的体验：
为什么不使用呕吐 Doodle 的形象？这不是识别度最高的一个吗？
他们刚刚官宣 Pharrell Williams（美国著名说唱歌手以及时装设计师，译者注）为品牌创意总监，为什么不把他的 Doodle 放在里面？
为了庆祝 Doodles2（顺便说一下，它本该命名为 2Doodles，此处@大头俱乐部的 Jon ），如果他们搞个共创新 Doodle 的艺术体验活动，会怎么样？
NFT 项目：曼尼游戏（Manny’s Game）
曼尼游戏（Manny.Game) 是我很喜欢的 NFT 项目，它借助 Web3 技术成功地建立了纽带。它最初是积极信徒围绕 Manny 游戏化 3D NFT 形成的社区，在让人们享受 Mannyverse™️ 和叙事的同时，他不断提供建立更深关系的途径。曼尼将 3D 模型开源，让艺术家们可以制作 3D 配件，他组织现场活动，并使用 NFT 构建独特的体验（纹身店，成就系统等）──这些方式都让他出色地建立起社区联结，培养出了真正的友谊。
这些类型的活动许多项目都做过，但在 Manny’s Game 中，人们却真正参与其中。为什么？因为 1. 大家喜欢这个叙事；2. 他有意精心安排的每一个体验都使社区更紧密地联系在一起，将 Mannys.Game 的体验带到了一个新的高度。
曼尼纹身店（Manny’s Tattoo Shop）
从技术方面来看，纹身是 Manny NFT 上可更新的元数据标签，但他在纹身店内建立的体验是如此之丰富。每个 Manny 都有一个初始的纹理/稀有度，但纹身店扩展了每个 Manny 的个性化程度。
纹身店让社区创造力爆表，创造出了各种精彩或搞笑的设计。他还创建了一个 discord 机器人来显示服务器中所有的更新，它成了一个吸引人的循环体验，使社区真正更紧密地联结在了一起。 
同样，这不仅仅是个技术问题，而是社区体验──是以人们心中喜欢的叙事为根基，利用巧妙的技术构建出来的社区体验。
一个非 NFT 示例——《精灵宝可梦 Go》
口袋妖怪是最畅销的 IP，作为一个品牌，它在全球带来了近 1000 亿美元的收入。几乎每个人都喜欢宝可梦，它在许多人的童年生活中扮演着重要角色，甚至影响到几代人！
《精灵宝可梦Go》是一个很好的例子，因为他们以最有意义的方式扩展了 IP。每个人小时候都想和口袋妖怪交朋友，团队创造性地使用了有趣的混合现实技术（ MR），创造了一个免费的体验，任何人都可以在现实生活中成为口袋妖怪训练师。
他们并没有止步于此；游戏完全以社交方式架构。
《精灵宝可梦Go》＂没有大量的教程，它让玩家在相互之间寻找答案和建议；它鼓励玩家互相比较自己的收集，通过与他人的比较来了解自己的收集进度；它还有一些功能，让你与其他玩家合作，以便一起获得远比独自游戏更多的宝可梦＂——这些都为该 IP 带来了新鲜的、协作式的体验。
它的成功是毋庸置疑的，在第一周内就打破了 App Store 的记录。人们在当地的树林里聚会，并在生活中第一次徒步旅行，只是为了捕捉「快龙」或「梦幻」。
简直是天才。因为，它让人们能够相互联结，用前沿技术把人们带入他们热爱的世界。
这些只是一些想法和例子，本文并不是要阐述我们可以创造的所有不同体验的理论。然而，我认为那些成功者都会思索这样一个问题：怎样才能利用 NFT/Web3 技术，来对我的 IP 进行有意义的扩展，让我的社区能真正进行关系建设，创造出忠实的粉丝群体？
如果人们深爱某个叙事或某样东西，他们就会使用市面上的任何工具聚集起来──无论以物理或数字的方式。当他们聚在一处时，倘若我们能够提供建立新的、有意义的体验和关系的工具——我想我们就可以用这项技术创造出有意义的东西。
想一想 练一练
现在， Jacob at Zora 提出了一个有趣的问题：
《蒙娜丽莎》在被分享时是被稀释了还是被扩展了？
当然，感受都是主观的，但我认为它被扩展了，因为互联网让人们能够对这幅画作进行重新诠释。虽然它的衍生品没有经济激励（区块链可以实现这一点），但互联网使衍生品能够广泛传播，极大丰富了蒙娜丽莎在现代文化中带给人们的感受。
如果说媒体分享存在局限性，那么蒙娜丽莎本该只是又一幅伟大的画作。但在每个时代里，它都会被重新定义，也在今天的世界中被拓展和赋予新的意义。通过加入个人层面的解读：把悲伤蛙和小黄人裁切制作成迷因、在电影和演出中展示等等，我们让这幅古老的作品与时下密切相关。
你固然可以说这幅画作的＂原始＂价值已经被稀释，但我认为它并没有。正是当代的诠释让蒙娜丽莎为人所知，人们甚至可能因为它的新诠释才在自己的生活中努力学会欣赏。要我说，恰恰因为《蒙娜丽莎》拥有很好的 IP 内核 ，那些诠释才能成为对这一内核强有力的补充，最终拓展了「蒙娜宇宙」（哈哈）。
