DeSchool 创作激励计划启动，100w Token 等你拿！

作者: SeeDAO 发布时间: 2022-10-17 09:11:26
【首期主题】大DAO至简：DAO 内运营的那些事

DeSchool 是一款划时代的学习工具，其目标是探索Web3时代的教育机制，服务Web3 时代的学习者与建设者。

眼下，Web3经济体正在兴起，服务于工业时代的教育体系逐渐落伍。DeSchool团队相信，传统的高校行政系统将在去中心化运动的浪潮中被弱化，我们最终会形成一套与Web3经济体、与DAO这种组织形式相符合的能力认证体系与学习教育平台。而这正是 DeSchool 正在建设的方向。

DeSchool 目前已经上线的功能包括：

  • 一块专属的策展面板，实现 DAO 及各个公会的品牌露出、课程展示

  • 学员完成学习可获得 SBT

  • 白名单中的讲师可自主发布课程

  • 课程支持视频、测试题、notion 页面等组件的嵌入

同时，在 DeSchool 的经济模型设计中， 50% 的 Token 将会用来支持社区的发展、用户 learn to earn，以及最重要的是：激励生态种内容创作者。

因此，DeSchool 将在本月正式启动主题创作激励计划，开启每月一度的内容创作挑战赛，并设立奖金池激励！

【首期主题】

大DAO至简：DAO 内运营的那些事

【激励池】

100w Token

【周期】

10.11-11.11

在周期内根据主题上传课程视为参与成功

【玩法】

获得 Token 的方式有 3 种：

  • 上传视频。

    只要根据本月主题上传视频内容，就能拿到基础奖励。获得的带你赞、传播越多，分的的 Token 就越多。

  • 邀请讲师入驻。

    当你通过邀请码，邀请到讲师在 DeSchool 上传课程，即刻获得 Token 奖励。

  • 优质内容评选。

    主题策划结束时，会根据社区点赞传播和特邀 DAO，综合评价产生当期话题胜出者。

    本月有 5 个话题，因此会有 5 名胜出者，每人将分到 15w 枚 Token。

【话题】

在本月主题之下，还有5个细分话题，均可作为内容制作的方向。

  • 【on boarding心法】 结合你在各种DAO的实践，聊聊on boarding对不同主题的DAO所起到的实际作用，来分享你的经验和教训吧

  • 【运营工具十日谈】 文档、IM、论坛、投票、bounty、SBT，你还发现什么好用的生产力工具，快来奶一下

  • 【DAO里的会该怎么开】 在DAO环境中，沟通信息、达成共识的主要途径是开会。而如何高效的开会，是每个加入DAO化运营的新人，每个新DAO最头痛的。会究竟该怎么开，值得好好聊一聊

  • 【达成共识的若干途径】 不论是协作还是形成共同体，共识都是最最重要的根本。如何在免信任前提下与“陌生人”在DAO运营中有效达成共识，是个永恒的话题和挑战

  • 【协作的协，协作的作】 协作协作，究竟是“协”更重要，还是“作”更重要？
