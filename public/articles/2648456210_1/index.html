以太坊油价五年新低，把握时机做五件事

作者: 高重建 发布时间: 2024-09-05 01:00:00
人文为体，科技为用。 基建做出来就是给人用。
作者介绍：人文为体，科技为用。LikeCoin、DHK dao 发起人。
著有《区块链社会学》《财富自由主义》《Game 以载道》《好 game 有好报》等。

自从比特币第四次产出减半以来，币市上上落落；有人认为牛市已经悄悄溜走，也有人坚信比特币会重复过往几个周期的轨迹，明年才是真正的大牛市。

币价尚且好淡争持，应用却近乎全面淡静，作为区块链使用量最重要指标的以太坊油价（gas price），跌至五年以来的低位，ethgas.app 网站经常出现「1、1、1、1」的奇景，即使要求快速完成交易，也只需要1 Gwei的油价。不少投资者担心以太坊使用量低，产生通胀，拖低币价，前景堪忧。

对我而言，却是「最期待的画面出现了」。基建做出来就是给人用，手续费低是用户的福音，他日以区块链为底层的应用大规模采用，使用量上升，手续费自然重拾「合理」水平，用户与其担心币价，还不如好好把握这个五载难逢的机会，趁着应用闷局去做油价高昂时忌做的事。以下，是我的五项建议。

一、取消昔日通证授权

使用 Uniswap、Aave、OpenSea 以至其它服务，凡涉及 ERC-20 通证或 ERC-721 等 NFT，均需授权相关智能合约动用钱包内的资产（token approval）。一般服务为求方便，往往预设要求用户授权动用全额，而且权限永不过期; 久而久之，钱包总会授权了一大批智能合约，有权随时动用钱包里的资产和 NFT。

你或许会认为只要授权的 DApps 够可靠就没问题，不过，就算应用当时可靠，时间久了，难保会被黑客发现漏洞，利用通证授权，对你钱包中的资产予取予携。因此，最佳做法是定期清理过时的授权，就如你曾经授权银行大额转账至特定第三方，但可见将来已经不再需要转账予对方，当然是先取消授权为上，大不了有需要时再次授权。

要检查和取消钱包授权，Revoke.cash 是个好帮手。如果你跟我一样使用Rabby 钱包就更方便，只需使用内建的「Approvals」功能，就能列出帐号在各 EVM 链上的授权，并以信任度、时间等条件排序，只要多选各项目然后按「Revoke」，完成签署就能取消授权。授权动用资产需要手续费，取消授权也一样，我刚取消了一项授权，油价 1.6 Gwei，手续费 0.00004 ETH，即约 0.1 镁，不痛不痒; 换了以往油价高昂，取消每笔授权动辄两三镁，你很可能宁愿保留授权，反正多数不会出事。

二、把资产跨链至 L2

文首把以太坊油价处于历史低位归因于水净鹅飞，其实有欠公允; web3 关注度从过去两三年的高位回落是事实，但以太坊手续费低最主要是因为第二层网络 L2 搭建完成，很多过往在主网进行的交易已经转移至 L2，就我个人经验，近一年使用 L2 的次数远超于主链。宏观数据也说明这个趋势，查看Coingecko，当前以太坊主网总锁定价值（total value locked，TVL）约48 bil镁，前三大L2或侧链Arbitrum、Base及Polygon则分别锁定2.7 bil、1.5 bil及0.9 bil镁。主网一般是给贵族使用，交易畅旺时几镁甚至几百镁的手续费只是零头，平民如我日常只要使用 L2，就已经足够安全。

尽管如此，我经常接触 web3 用户，目测很多还没开始使用 L2，除了受些微学习门槛所挡，也因为得先把资产从 L1 跨链到 L2，而这些交易正正就是在主网上，手续费不低。我刚于Superbridge测试，一笔把ETH从以太坊主网跨链到Optimism的交易，以当时油价2.2 Gwei计算，手续费为0.0009 ETH即约2.3镁，还好，抠门的话还能捕捉油价跌至1 Gwei甚至更低的时机，再省1镁左右。

三、开立多签合约钱包

你可能以为多签钱包（multisig wallet）只适用于DAO等组织，事实不然。多签钱包有很多优点同样适用于个人，比如说，你可以跟伴侣开设多签钱包，就像银行联名户口一样，由其中一人签名即可动用（1/2），又或是两人签名方能动用（2/2）。

即使孤家寡人，多签钱包同样派得上用场，想要加强保安的，可以开设 2/2 多签钱包，由自己的两个钱包共同管理，比如一个储存在电脑版 Metamask，一个储存在手机版 Coinbase wallet; 反过来，想要避免丢失单一组助记词就失去资产，可采用类似方法，但设定为1/2签署即可动用。

不过，多签钱包的缺点也很明显，除了签名流程变得繁复，手续费也会大幅增加，单是在主链开设多签钱包，以往就可能需要数十镁，因此最适宜在当下处理。我刚测试了一下，使用Gnosis Safe在主链开设新的多签钱包，油价0.9 Gwei，费用0.000233 ETH，才0.6镁左右。

四、注册和设置 ENS

另一个单是注册所需手续费就让人心痛的是 ENS 网址，Ethereum Name Service。ENS 可说是以太坊版的 DNS Domain Name System，把难以记忆又容易输入错误的钱包地址，对应至简洁的名称，比如说，知道我的 ENS ckxpress.eth，要支持这个文字工作者（多谢 ），只要发 ETH、USDC 或其他密码货币到 ckxpress.eth 即可，多数钱包都能解读。

ENS 比 .com 域名便宜，注册费只需每年五镁或等值的 ETH，然而，过程中所花的手续费却往往比注册费高出几倍，妹仔大过主人婆。想建立 web3 身份，不妨趁机注册代表自己的 ENS，最近油价低水，手续费只需 1、2 镁; 即使已经持有 ENS 网址，亦可趁着低油价增加诸如头像、子域名、L2 网络帐号地址等各种设定，让你的 web3 身份更加完善。

五、整理资产、调配钱包、去芜存菁

假如你是 DeFi 或 NFT 活跃用户，或钱包已经沿用多年，账号里面多半有着一大堆通证，但很多要么是数量只有小数点后多个零的「尘埃」，要么价格已经归零，持有多少颗都没区别。别以为平日交易不多就没有这种情况，很多通证和 NFT 都会直接空投到钱包——尤其是用作钓鱼的。

除非你看到账号内一堆没用的零钱就受不了，否则清理这类通证属于非理性行为，因为无论是把尘埃兑换成 ETH，还是把诈骗通证转发到黑洞，手续费必然会高于通证本身的价格。强迫症患者不妨捕捉油费跌至 1 Gwei 甚至更低的机会来个大扫除，把碍眼的零钱和 NFT 通通清走。
