关于MMCM

About MMCM

- MMCM（Metaverse Modernized Classical Music Competition ）由SeeDAO&古典乐社团·登峰文社联合发起，旨在通过将web2.0创作者引入web3.0，共同探索古典乐形式、社会、理念和技术的更多可能性。SeeDAO聚集了社区中最多样和优秀的创作者，基于人人参与为未来人类社区建设更好的文化生态、更好地与时代结合这一理念，SeeDAO联合兼具知名度与专业性的古典乐社团登峰文社、Flow 联合主办了这场活动，也联合了Matrix World 、Mask Network 、myNFT作为友情合作方。

等众多实力强大的伙伴，与来自全球的演奏者、元宇宙的积极建设者共同举办了这场开拓性的全新活动。

-  作为一种以人文精神为价值取向的音乐形式，古典乐在其严肃形式之外更代表了一种文化精神。本次大赛名誉主席由英国皇家音乐学院（RCM）弦乐系主任Mark Messenger教授、“帕格尼尼比赛”首位意大利女性独奏音乐家Francesca Dego、立陶宛马格努斯音乐学院教授、钢琴家Julija Sadaunykaite等专业演奏家与来自11个国家29所音乐学院众多优秀青年演奏家担任。

愿景和目标

Vision and objectives

- 贴近大众、面向时代

我们希望做面向时代、面向大众的创作，当古典音乐变成纯粹师门传承和空洞无物的“先锋探索”，就迅速变成经典作品演奏技巧的比拼，而和它最广大的潜在受众拉开了距离。

- 依托web3.0，共同协作探索全新音乐生态

时至今日，古典音乐还保留着十九世纪音乐家——乐谱出版商——音乐厅——乐团这样传统的文化生态，这些传统渠道虽然成熟，但是其中的垄断和层级划分也阻碍了音乐进一步向大众的传播。我们认为，音乐的传播是可以在互联网上进行的；音乐本身也是可以共同协作产生的；甚至于，音乐的版权和给予音乐家的回报也是可以有更加现代方式的——这些都可以通过区块链和元宇宙技术来实现。

- 我们希望我们的声音是

大众的而非圈子的；

时代的而非历史的；

充实的而非造作的；

新潮的而非循旧的……

资深音乐家评委

Mark Messenger

MMCM大赛评委会名誉主席

Mark Messenger教授是一位全球公认的杰出小提琴家，指挥家，教授以及教育家。

现担任英国皇家音乐学院（RCM）的弦乐系主任。

作为独奏家，指挥以及室内乐演奏家，他与国际上的大音乐家经常合作，包括了梅纽因和马克西姆•文格洛夫。

Francesca Dego 

MMCM大赛评委会名誉主席

Francesca Dego是出生于意大利的小提琴家，她是众多国际比赛的获胜者。

尤其是在2008年，成为自1961年以来热那亚著名的“帕格尼尼比赛”首位意大利女性。

作为独奏音乐家，与德意志留声机公司签约发行了诸多唱片。

Julija Sadaunykaite 

新加盟评委

2015年马耳他瓦莱塔国际钢琴比赛大奖获得者。

目前任教于马格努斯音乐学院。

潘龑

大提琴演奏家，室内乐演奏家，上海音乐学院教师。

青年音乐家评委

乔丹

自幼学习小提琴，2013年以总成绩第一名考入中央音乐学院公费硕士，在校期间获研究生国家奖学金。后考入辛辛那提大学音乐学院硕士学习小提琴演奏，辅修乐队指挥。2017 开始在辛辛那提大学音乐学院攻读博士，从2018年起担任CCM 室内乐团首席。

曾在国家大剧院，中山音乐厅，纽约卡内基音乐厅演出，曾在法国波尔多与世界顶级小提琴大师 马克西姆 文格洛夫同台演出室内乐音乐会。

多年与当代作曲家合作并推广他们的新作品，参加美国中西部作曲论坛，与纽约作曲家圈合作出版专辑，并多次在纽约进行新作品世界首演。

陈沁妍

四岁开始学习钢琴，后考入西安音乐学院进行本科学习音乐教育钢琴方向，毕业后考入立陶宛维陶塔斯马格努斯大学音乐学院攻读第一硕士学位钢琴演奏专业，学习期间取得优异成绩，毕业后继续在立陶宛维陶塔斯马格努斯大学音乐学院攻读第二硕士学位钢琴伴奏专业至今。

本科期间曾获得国家励志奖学金。硕士期间，在立陶宛维陶塔斯马格努斯大学音乐学院就读，获得B类奖学金，同时担任学院助教。

在第一届瑞士国际音乐大赛成为唯一获得钢琴大师奖的中国学生。

陈露

自幼学习小提琴，本科毕业于中央音乐学院，获得小提琴演奏专业学士，并分别在美国辛辛那提音乐学院和英国皇家音乐学院（RCM）获得演奏硕士文凭，辛辛那提音乐学院小提琴DMA博士，获得辛辛那提音乐学院联邦奖学金。

2018年起入职登峰文社，现为登峰文社英国分社伦敦地区负责人。

2021年1 月进入英国BBC广播交响乐团与英国皇家音乐学院（RCM）人才培养计划（BBC Symphony Orchestra Pathways Scheme）。2021年8月在美国著名音乐出版社CD Baby发行个人专辑，2020年登上英国皇家音乐学院校刊人物志。

张驰

2004年出⽣于河南省孟州市，现就读中央⾳乐学院附中作曲系。曾师从于邹航副教授学习作曲，现师从与青年作曲家姚晨，同时师从⻘年钢琴家王天阳副教授学习钢琴演奏。

曾连续三年以“河南省⼗五⼤⻘少年钢琴家”身份在河南艺术中⼼⾳乐厅演奏。在台湾省台北、台中、⾼雄三地巡回演出。曾获得第22届纽约国际⾳乐⽐赛中荣获作曲A组第⼀名，第29届意⼤利“巴列塔之城”国际⻘少年⾳乐家⽐赛作曲⼆等奖，第9届阿⽶加拉国际⾳乐⽐赛作曲⼆等奖，第五届“骊鸣春晓”全国作曲⽐赛佳作奖。

MMCM大赛评委团秘书长

邱浚禾

自幼学习小提琴，2010年以第三名成绩考入广州星海音乐学院附中。附中期间获得过曾作为合唱团指挥，带领合唱团获得星海音乐学院附中2014年合唱比赛第一名。

2016年考入辛辛那提大学音乐学院主修小提琴演奏，同时学习乐队指挥，获得由美国学派小提琴教学泰斗迪蕾创办的斯塔林项目基金会授予的奖学金，以全额奖学金考取并参加美国Bowdoin国际音乐节。

曾与多位作曲家合作，作为辛辛那提音乐学院代表参加美国中西部作曲家论坛，并在Bowdoin 国际音乐节期间与哈佛大学作曲系合作首演音乐作品。曾与著名钢琴家Rohan De Sliva （著名小提琴家帕尔曼的长期合作伙伴）合作准备音乐会，也作为独奏演奏员获得与德国耶拿爱乐乐团的合作机会。2019年起转至于布拉格表演艺术学院（HAMU）学习。

2013年参与建立音乐家自治团体文兴社，2018年文兴社在巴黎改组为跨国艺术家团体登峰文社后，当选总社长，任职至今。自2019年起，以笔名“榕柳先生”发表艺术社评文章，致力于推广“古典音乐现代化”之理念。

团队一览

Teams

下面让我们一起来了解一下MMCM的主办与协作组织，看看都有哪些团体致力于元宇宙古典乐现代化的伟大事业！

- SeeDAO 

服务于创作者经济的DAO组织SeeDAO于2021年11月成立，致力于将创作者从Web2.0进入Web3.0，帮助内容创作者和内容消费者探索更好的生产关系和组织关系，目前针对创作者的运营服务中台已经搭建完毕，第一期面向创作者的“制作你的第一个NFT”集训营也已顺利开班。

- 登峰文社

登峰文社2018年创立于巴黎，整合了当时位于美国辛辛那提音乐学院、中央音乐学院、星海音乐学院、四川音乐学院中的具有进步性的音乐家或者音乐家团体。经过多年发展，目前成员已经分布在11个国家的29所音乐学院里面。

联合主办：

- Flow 

为下一代应用程序、游戏以及数字资产而构建的新区块链。Flow 是一个快速的、分散的、对开发者友好的blockchain、旨在为新一代游戏、应用和数字资产赋能。它基于独特的多角色架构，设计为无需分片即可扩展，从而在速度和吞吐量方面得到了巨大的改善，同时保留了一个对开发者友好的、符合ACID标准的环境。

赛事合作：

- Matrix World 

Matrix World 是一个去中心化的开放虚拟世界，它允许用户与同时运行在不同区块链上的沉浸式3D应用程序进行交互。这个世界由陆地组成，作为非同质化代币(NFTs)发行，这些token在区块链网络如 Ethereum 和 Flow 上永久存在。

- Mask Network 

旨在建立一个从 Web 2.0 到 Web 3.0 的桥梁。通过将优秀的去中心化应用生态系统带入传统社交网络，Mask 产品为用户所熟悉的 Web 2.0 功能提供了去中心化的选择。用户可以在不离开主流社交网络的情况下，享受安全、去中心化的社交、无国界支付网络、去中心化文件存储与分享等功能，并轻松领略 Web 3.0 世界的魅力。Mask Network 是所有权经济的坚定信仰者，我们倡导用户珍视自己的隐私数据、个人资产以及创建的虚拟空间并拥有主权。我们邀请所有看到这条信息的朋友加入我们，重新获得对个人数据的掌控。

- MyNFT 

Mynft是一个以用户为中心的NFT聚合平台。我们相信NFT是连接数字用户的核心媒介，当前将主要从三个方面探索NFT的聚合：聚合管理与展示用户的NFT；聚合用户基于NFT的社交关系；聚合资产的跨平台交易。
