NFT 评估和稀有度打分平台可以助你在 NFT 投资中先人一步。

这些工具是挖掘宝藏 NFT 项目的一把好手，具备评估 NFT 稀缺度、挖掘最新 NFT 项目、追踪巨鲸钱包、优化投资决策等功能。它们绝对是你在审慎抉择投资对象时必不可少的工具。

与其它加密领域一样，找到可靠的信息渠道是决定投资成败的关键。不论你是新手小白还是 NFT 收集爱好者，都要借助分析工具以评估资产的稀缺性和可及性。毕竟，谁不想淘到下一个像无聊猿一样的爆款项目呢！

小贴士：浏览OpenSea并不能帮你淘到优质 NFT。建议多使用文章中提及的这些工具。

幸运的是，已经有不少平台开发了这类工具。其中一些是免费使用的，另一些则需要付费解锁更多功能。下文中，我们总结了几种必备的 NFT 分析工具，希望能为你带来帮助。

* 注：以下列举的工具不分先后次序。

01.

Dappraddar

Dappradar 是一个功能丰富的分析工具，不论是新手还是资深 NFT 投资者都很适用。你可以在里面免费查看所有现有的去中心化应用（Dapp）的排名，平台还针对一些主流类型设立了单独的页面，如 NFT类、DeFi类、交易类、游戏类Dapp。

在NFT类Dapp的页面中，用户可以评估、追踪和挖掘新项目，实现一站式无缝衔接，用户还能通过投资组合追踪器（Portfolio Tracker）监控所购资产。该页面依据多项参数（如流动性、市值、价格）对项目进行综合排名。其中，NFT 售价排行榜、NFT 热度排行榜、交易所排行榜以及新项目筛选器可以向用户展示市场中的实时交易量和价格波动。

对于新人而言，Dappradar 也不失为一个不错的教育平台。在Blog页面中，可以找到不少加密领域重要话题相关的优质资源，包括专题文章和加密圈要闻速递。

主要优势

■  完全免费；

■  Dapp 和 NFT 排行榜；

■  投资组合追踪器（Portfolio Tracker）；

■  钱包追踪（可以追踪任何钱包，十分适合用于追踪NFT巨鲸的动向） 该关注哪些巨鲸？请看这篇文章: 

medium.datadriveninvestor.com/10-nft-celebrity-whales-that-you-need-to-follow-207ddceeaa00

■  NFT 价值评估；

■  深度行业报告；

■  支持代币转换。

02.

Rarity.Tools

Rarity.Tools 也是我最推荐的网站之一。它统筹整理了所有 NFT 艺术收藏品的实时数据，深受艺术收藏者和创作者的喜爱，特别是在衍生艺术领域。

Rarity 依据稀缺度对 NFT 艺术品进行整理排序，并会在网页推介最新项目。该平台的审美十分贴合这一艺术导向的市场，主界面设计很有创意，不像其他工具那样以图表和列表为主。

用户无需付费就能根据不同特征属性、成交量和交易价格对 NFT 项目中的艺术品进行筛选和排序。用户可以点击任何一个艺术品以查看更多信息，获取不同货币的价格曲线。Rarity.Tools 的一个特别功能在于，可以通过花费 2 ETH 的手续费，快速便捷地将新项目永久添至平台列表中。

Rarity.Tools 也很适用于探索即将发售的 NFT 项目，帮助用户在项目公开发售或铸造前获取第一手消息。

Rarity.Tools 可以帮助你依据稀缺度搜索项目，你还可以利用筛选器找到在售的稀缺NFT（下图为Humanoid 系列 NFT）。

主要优势

■  使用简单；

■  显示待发布 NFT mints；

■  以稀有度为参数的艺术品排行榜；

■  免费使用；

■  可快速添加自己的NFT项目。

03.

Etherscan 

Token Tracker

Etherscan 是一个区块链浏览器（即探索以太坊的窗口）。同时，它也可以用于追踪任何基于以太坊的代币，其中包括所有ERC-721代币。ERC-721是在以太坊上构建 NFT 的标准协议，目前，Etherscan 中囊括了21,669个ERC-721代币。最近，NFT工具套件增添了新功能，用户能够追溯NFT从铸造、分配到交易的完整历史以及价格信息。

本文中列举的许多工具都基于 Etherscan APL，并结合额外的数据分析功能，以提供多样化的深度见解。

主要优势

■  一款可以追踪所有钱包、交易、代币和智能合约的以太坊区块链浏览器；

■  支持所有 ERC-721 代币;

■  投资组合追踪器（Portfolio Tracker）;

■  免费访问，使用简单。

04.

Icy.Tools

Icy.Tools 是一项基础的 NFT 追踪工具。它直观清晰的界面对初学者十分友好，可用于快速市场调研。比如，页面中有一个基于交易量和售价综合排序的热门项目排行榜。

此外，Icy.Tools 还允许用户查询项目交易历史，以及查看任何钱包地址上的余额。然而，这些功能仅面向付费用户开放。如果用户需要获取详细数据信息、无上限追踪钱包地址、深度考察市场图表和设置定制化提醒的话，可以选择以每月 0.03 ETH 的价格订阅高级版本。

主要优势

■  使用简单；

■  NFT 项目排行榜；

■  钱包追踪；

■  新项目提醒。

05.

Moby

Moby 是一个小众但高效的追踪网站，可提供最新的实时 NFT 信息流 (feeds) 和统计数据。网站中的窗口期间隔从10分钟、60分钟到24小时不等，因此，投资者可以借此观测资产的实时动向，并根据最新趋势快速做出决策。

Moby 是最好的NFT观测工具之一，尽管数据图表仅对花费 0.1 ETH 订阅半年 Pro 版本的用户开放。该网站只显示 NFT 藏品列表，但相关排行榜和实时信息流是封锁的。订阅者可以解锁包括 NFT 信号、历史收藏数据、实时信息流等功能。

主要优势

■  统计数据和消息提醒（付费）；

■  使用简单；

■  提供 NFT 信号供投资参考；

■  显示最新 NFT mints；

■  钱包追踪（付费）。

06.

Traitsniper

Traitsniper 是一个用于挖掘最新 NFT 项目的小众网站。它专注于分析已发布和待发布的项目，考察 NFT 元数据，并根据它们的稀有度挖掘投资潜力股。

新项目一经发布，Traitsniper 就会在30秒到2分钟内分析其稀有度，准确率超过95%。

免费版和付费版用户分别享受不同的使用权限。比如，付费版用户可以获取最新藏品及其稀缺度数据的即时通知，而免费版用户要在 NFT 发布1小时后才能获取这些信息。如需订阅，平台会先将用户转至他们的Discord频道以获取更多信息，费用为 0.15 ETH/周或 0.5 ETH/月。

Traitsniper 的一个关键功能是用户可以查看待发布项目的相关内容，助你在 NFT 交易中占据优势。

主要优势

■  使用简单；

■  即时的稀缺度评级；

■  鼓励社区参与；

■  显示最新 NFT mints；

■  显示 NFT 发布日期。

07.

Raritysniffer

顾名思义，Raritysniffer 的功能与 Traitsniper 相似，即以稀缺度为参数对最新 NFT 项目进行实时排名。该平台仅提供最新 NFT 项目的相关信息，且功能对免费版用户开放。

如需获得潜在稀有 NFT 的深度分析，则需加入 Discord 频道进行订阅，费用为 1.5 ETH。订阅用户可在最新 NFT 潜力股公开发布前，抢先了解其稀缺度数据。

主要优势

■  使用简单；

■  专注于稀有度排名；

■  显示最新 NFT 项目；

■  鼓励社区参与。
