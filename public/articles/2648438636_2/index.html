Web2能干的事，还要啥Web3？

都说电音和说唱是Web3音乐的代表，但是知名电音厂牌“怪猫”（Monstercat）对Web3似乎并不买账。最近，他们推出了一些数字商品：数字会员卡、虚拟玩具等，都可以进行二级市场交易那种。在Web3世界的小伙伴听起来，似乎毫无疑问这就是NFT了，但实际上它跟Web3毫无关系，因为怪猫的商品完全是建立在Roblox上的。

怪猫其实不是第一次这么做，以小编查到的，至少2020年他们就跟Roblox合作推出了Roblox x Monstercat联名精选音乐专辑。这次的数字商品似乎是2021年怪猫在Roblox上开放的“Lost Civilization”任务的后续，只要完成任务就可以获得新的数字商品空投，而这些新的数字商品又将作为未来的会员卡。小编已经在油管上看到若干个up主在教人们“如何在Roblox里获得你的限定款怪猫玩具”。

这让musicben.eth等一众Web3音乐KOL陷入了沉思。。。Roblox不仅是Web2的中心化平台，还是要收税的那种，人们买的东西也完全不可能离开Roblox平台，但是为啥这么吃香呢？

可能就是因为……Web3没有人家好玩吧

我们上期报道的Coinbase为BASE搞的Onchain Summer，各大NFT平台和产品积极参与，但是从数字看最显眼的可能还是AnotherBlock，对，就我们提到了N次但总觉得哪里怪怪的那家流媒体版税共享平台。他们带来的3个open edition音乐NFT在短短几天里总计已经mint了超过5万个，官微也兴奋的宣布：大家等着分钱吧！

Black Dave的新social token

我们在第15期时写过美国的黑人Web3音乐人兼Web3布道者Black Dave。那时提到，Dave在2021年尝试发行过自己的ERC20 token，叫做Black Dave Verse Token，其用处是让Dave用你指定的内容rap。最近，他又发行了一个新的个人Token，依旧是ERC20的，叫做$KURO。

为啥这哥们儿这么执着于发行token呢？他是有挺长时间思考的，小编从他2022年陆续发的几个文章里摘录如下：

1. 社交代币和NFT各有各的用处，token的用途更广，比如可以拿来买NFT和服装等商品。

2. 社交代币根本不该有货币价值，也不需要有流动性。

3. Dave主张两者兼而用之，正好可以区分“喜欢我的人”和“喜欢我作品的人”。

4. 应该给两者以不同的体验

Muverse在今年年初宣布种子轮融资六百万美元，看来用的挺节俭。小编体验了下，抽奖的规则还是挺geek的，由一个简单公式确定，而公式里的变量来自于链上交易的哈希值，以及玩家用游戏代币MCT购买的一组号码。该代币可以直接从Uniswap购买，也可以在App中通过音乐游戏获得。
官宣及白皮书见：https://twitter.com/BlackDave/status/1691620464810922440

Sound.xyz乱花迷眼？检索新功能上线

自从Web3音乐平台扛把子sound.xyz前不久开放艺人入驻，很快就增加了上千位独立音乐人和近千首歌曲，这带来了一个不大不小的难题：作为听众，怎么才能从一大堆列表里找得到自己想要的音乐呢？  

最近，他们针对这个痛点推出了新功能：Explore页面。

其实说来并不复杂，在这个页面上，人们终于可以按多种条件来检索了，不光是流派、艺人，还能按收藏者，这似乎算是Web3音乐的一个新维度。

wavWRLD推出wavCurate

wavWRLD是在SeedClub的第五期孵化中出现过的一个项目，一开始旨在为NFT收藏者举办IRL演出。最近，他们推出了一个活动叫做wavCURATE，希望解决Web3音乐怎么被发现的问题。按照设计，wavCURATE的机制是这样的：

1. 每次为期一周，先设定一个主题，参加者每人提名一首歌曲

2. 根据这些歌曲生成一个播放列表（使用了spinamp和oohlala）

3. 人们收听这个播放列表，并匿名投票

4. 看投票结果是谁提名的，赢家获得一个NFT，并指定下个主题

是不是像某种接龙游戏的NFT版？目前这个活动已经举办了两期，其结果可以在sound.xyz看到。

官宣：https://twitter.com/wavWRLD_/status/1692237961411154225

Sound上的列表：https://www.sound.xyz/playlist/66103220-12f7-4233-9343-dfc17f66213f

FWB Fest 2023上崭露头角的Soulquest

本月早些时候著名文化类DAO FWB刚刚举办了一年一度的FWB Fest，我们略做过一点报道。这次想说说的是在这次FWB Fest 23上出现的一个音乐共创项目，叫做Soulquest。

这个名字是为了致敬90年代的著名音乐家小团体Soulquarians。这个以宝瓶座音乐人为主的团体搞了许多合作的音乐实验，创作出了一些影响深远的作品，但这个组织本身从始至终都是个非常松散的小团体，在2000年初就不复存在了。

如今，几个同样热爱音乐的年轻人希望复刻他们的音乐创作方式，用区块链的形式加以记录。在经过PartyDAO的众筹资助后，他们派了4个人来到FWB Fest 23。在此期间，除了完成了音乐创作，还记录下了完整的过程。这些音乐和纪录片如今都在链上拍卖。

祝他们能够持续下去。

Web3音乐人之星

Henry

Web3音乐人越来越多了，我们争取每期选取一个略加介绍。说不定是从sound.xyz上扔骰子选的，不构成任何投资建议啊~

本期Web3 音乐人之星

X Li

我们的报道中曾经出现过两位华裔音乐人，今天再介绍一位，在sound.xyz大放异彩的X Li。

X Li是现居LA的一位华裔音乐人。5月份时发布的首个NFT作品《Think I’m In Love With You》在sound.xyz上连续三周排名mint数榜首，一时间吸引了众多关注，被许多独立音乐人和社区当做“独立艺术家如何利用新兴数字平台获得成功”的典范。考虑到同期狗爷NFT的表现都不怎么样，这一数字更了不起。

X Li把这一成绩归功于过去他在社区中积攒的人气。从前在多伦多求学时，十几岁的X Li已经在亚洲社区小有名气。但是出身优渥的他也遭遇过重大的人生挫折，在那之后，他明白自己必须努力重新开始，而音乐就是他决定从事的事业。

他从2017年开始接触音乐制作，2019年开始真正开始制作。2020年时，X Li在北京待了一年。作为一个初出茅庐的新音乐人，在这段时间里X Li思考了很久——像自己这样的独立音乐人该怎样创造收入。当时其实他设想的是另做一个专供独立音乐人的中心化平台，自己吆喝自己卖。时间快进，疫情来袭，很多音乐人开始尝试NFT，X Li这才开始真正审视这种加密艺术形式，并激动地发现，这比他设想过的东西还要好。按他的感受，音乐人起码有99.99%都无法赚钱养活自己，在过去，只有几十几百个粉丝的音乐人从流媒体平台上几乎一分钱都赚不到，因为播放量过低。但是现在NFT这种发行形式使他们比较容易获得一些收入。

作为已经初尝胜果的音乐人，X Li对Web3音乐还保持着很理性的认识。在X Li看来，Web3音乐的问题除了Gas fee比较贵，剩下的就完全是音乐本身的问题了。Web3音乐人的作品质量着实还有很大空间，换句话说，真正有经验的优秀音乐人几乎还都没有进入这个领域。这些传统音乐人往往还挣扎在流媒体播放量的泥沼中，而一旦他们意识到Web3这条路，格局可能就不一样了。而对于已经在Web3试水的音乐人小伙伴，X Li最希望的是少一点炒作，多打磨作品质量。
