NFT美学

近年来，非同质化代币（Non-Fungible Tokens，简称NFT）的概念在数字艺术和区块链领域迅速兴起。NFT作为一种独特的数字资产，已经吸引了全球范围内的创作者和投资者的关注。在这个快速发展的领域中，华人创作者和企业家们也发挥着重要的作用。他们以创新的思维和独特的艺术视角，参与或创立了许多令人瞩目的NFT项目，为全球数字艺术行业带来了新的活力。在本文中，我们将介绍七个由华人参与或创立的NFT项目，探索他们的独特之处以及对数字艺术领域的贡献。他们分别是：Seed NFT、Weirdo Ghost Gang、CyberV、Sofamon、Nervape、INK、Nobody。

●

项目概况

简介：
Seed NFT, 是SeeDAO数字城邦的公民身份证明，也是获得城邦治理权益的认证。每个Seed NFT都是一颗独一无二的种子。
Seed NFT是参与节点共识大会与获得社区治理权的前提，与声望、积分和段位共同决定公民在SeeDAO的治理权限；Seed NFT持有者发起的创DAO提案，有机会得到SeeDAO社区基金支持；未来SeeDAO发行治理代币将以Seed NFT作为空投的依据。

动态NFT展示规则：

Seed NFT中人物眉间"太极星"的颜色会随着成员钱包中积分所处段位变化而改变，所有链上头像将动态显示。太极星颜色仅与当前持有人的段位相关，交易后将更新为新持有人的段位。

设计概念：

Seed的产生并不是一帆风顺，Seed起初的几次设计尝试里并没有明确的去向东方审美，虽然看起来很独特，但是这种独特无法支撑SeeDAO的精神内核。在吸收了一些社区的意见之后，设计师开始从原点思考SeeDAO的需求，结合之前积累的资料，尝试设计出了“太极星”这个意向符号。同时每个形象都有一个共同特征，闭上的眼精，代表修行禅定。
新版视觉重新阐释了“SeeDAO”的意义。“SeeDAO”的“See“不是像西方人那样用眼睛去看，更不是还有戴一个眼镜瞪大眼睛去看，我们是闭上眼睛用心去看的。”SeeDAO“也可以解释为每个人看见自己的”道“，而看见自己的道，正是领悟自身的过程。See是悟。

●

基本资料

官网: https://www.sofamon.xyz/
Twitter: https://twitter.com/sofamon_xyz
项目所在链: BASE

项目概况

简介：

Sofamon是一款基于Base链生态的Socialfi DAPP, 基于个人形象购买可穿戴物品后，可在Telegram生成专属贴纸与朋友分享和展示。它不仅仅只是链上可交易的贴纸应用，就如官方所描述那样，它也是一种社交层面新的表达方式。

项目介绍：

你可以理解它是Meme、NFT、Sticker、收藏品，亦可以理解为是一个加密艺术表达平台，它可以：
领取专属于你的可爱小兔子Sofamon，并在商店中为它精心挑选和购买可穿戴物品
通过收集到的物品装扮你的小兔子，创造出专属自己的贴纸，拥有独属于你自己的收藏品
展示自己的收藏品，并与朋友们在Telegram上分享你个人风格的贴纸
购买可穿戴物品及分享贴纸，产生胡萝卜（Carrot）积分，幸运的你可通过积分抽奖获得下一套可穿戴物优先购买资格
商店中的每个可穿戴物品都有一定的价值，随着贴纸的分享和传播，受欢迎的物品根据一定的算法将会升值。
目前商店中已出3套收藏品，2套官方团队创世系列，1套与 Eigen Layer 合作系列，每套收藏品中都有一款稀有可穿戴物。创世「NORA」系列的设计由泰国艺术家 @ChinpongR 与Sofamon团队合作创作完成。
根据团队顾问@0xCygaar在推特中的表达，Sofamon并不希望只做平台唯一的艺术创作者，未来将会有不同的艺术家和项目合作创作的系列推出。

Nervape

●

基本资料

官网: https://www.nervape.com/
Twitter: https://twitter.com/Nervapes
项目所在链: CKB

项目概况

简介：
Nervape 是一个基于 RGB++ 的神经猿 NFT 项目，其发行总量是 2777 枚。是第一个也是唯一一个以故事为驱动并具有可定制化内核的 3D NFT 的项目。
项目描述：
Nervape 讲述了第一批到达「第三大陆（The Third Continent）」的 Nervape，他们之间有关友谊、勇气和信任的故事。随着 Nervape 继续探索第三大陆，新的角色、独特的场景和新颖的道具，将作为 3D 版的 NFT 被添加到 Nervos 区块链上。
Nervape 的社区共创可以分两部分来讲。首先，有一个神秘角色将展开一个关于梦境的番外章节。在这个梦境篇章中，Nervape Character NFT 持有者将有机会通过持有、质押、销毁他们的角色 NFT 来对故事的走向进行投票。这个机制不仅会影响故事的方向，也会影响角色 NFT 的稀有度。
另一方面来说，Nervape 持有者在后期可以对Character NFT 拆分和重新组装， 因此，Nervape NFT 的总体外观其实将会是 Nervape 社区每个人共同创造的结果。并不是由 Nervape 团队独自完成的。除此之外，我们也在探索由社区或通过和艺术家合作设计来创建 Nervape NFT 的可能性。

团队介绍：

创始人：Echo

Twitter: https://twitter.com/Nervape77

基本资料

●

Twitter: https://twitter.com/inkonbtc
Discord: https://twitter.com/inkonbtc

项目概况

简介：
INK是于2023年底建立在比特币上的PFP类NFT项目，该项目以比特币标志性的橙色作为背景，并采用墨画风格绘制。
项目介绍：
INK分为两个系列，其中INK Custom是手绘定制，总量为150个，该系列采取了定向邀请的方式，比如dingaling和Thor等NFT巨鲸以及Memeland、Floor Protocol和Pixelmon等项目；INK Genesis是是总量为1万个的手绘生成艺术，在3月28日正式开启白名单发售之前，该NFT的价格被场外炒至4000到50000美元，远高于铸造费0.0058 BTC，足以可见市场的高预期。
与其他比特币NFT项目不同的是，INK是首批铭刻在Legacy Sats（遗产聪）上的10K系列。Legacy Sats的概念最早可追溯至Ordinals创始人Casey Rodarmor于2022年6月的比特币研讨会，即Ordinals面世半年前。彼时，Casey发放了41个纸钱包（Paper Wallet），每个钱包里含有10,000 Sat，他将公钥和私钥打印在纸上后分发给与会者。这部分就是Legacy Sats，总量为41万个，但其中近一半已被遗失，这使得Legacy Sats比罕见聪（Uncommon Sats）更加稀有。

团队介绍：

该项目是由4名在Discord相识的成员创立的，其中联合创始人zetman.btc是艺术收藏家，持有多个蓝筹比特币NFT；Joey Dodd是与DC漫画、梦工厂工作室、孩之宝公司、漫威娱乐和任天堂等知名公司合作的艺术家。
联合创始人：Zetman
Twitter: https://twitter.com/zetman_eth
联合创始人，在美华人，高级工程师，属龙，BTC极简主义者；发推频繁，希望更多华人参与ordinals生态，积极推广INK并与中文用户互动。艺术收藏家，持有多个蓝筹比特币NFT。
投资方：
BTC NFT项目INK近日在X平台宣布已获得孵化器CaesarEon的1000万美元投资，旨在推动发展比特币原生IP生态。本次投资中，200万美元将用于推动项目发展，800万美元将由CaesarEon用于在全球范围内发展和开发IP、建立去中心化品牌市场理念、招募人才并维持项目运转、扩大IP产业资源，为Web3行业提供去中心化IP构建工具等。
