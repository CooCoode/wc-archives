NFT 还合法吗？

上周的大事之一是美国 SEC 第二次对 NFT 开出了罚单（第一次也就在上个月），指 Stoner Cats 的项目方违规销售了未经注册的证券 —— 这里说的证券当然就是 NFT。

虽然双方很快和解，但这一个月来 SEC 连开两炮，使得不少 NFT 项目方心惊胆战。那些发行音乐 NFT 的 Web3 音乐人也充满了担忧。好不容易开发的谋生新途径，会不会被扣上非法的帽子？

理论上说，只要给人提供了 “买 NFT 会赚钱的，走过路过不要错过” 的预期，那还真很有可能被 SEC 的豪威测试给当做证券。但这恰恰就是迄今为止多数 KOL 所倡导的路线。所以有的音乐 NFT 项目比如 Anotherblock 上次发 Justin Bieber 歌曲 NFT 的时候干脆禁止美国用户参加，不过这也不是个长久之计。

长久之计是啥？没人知道。。。这河里它没石头可摸啊

续集：神人 Jonathan Mann 反击 SEC

我们之前报道过的神人 Jonathan Mann，就是连续 5000 多天坚持每天创作一首歌的那位 “连续写歌最多的人” 吉尼斯世界纪录保持者，听到 SEC 的行径大怒，连夜创作了歌曲 THIS SONG IS A SECURITY =_=

Jonathan 老哥自从踏入 Web3 以来，每天的歌都放在自己的网站上发行为 1:1 的 NFT 进行拍卖。说实话大部分都是很口水的歌，一般也就卖 0.0 几 eth，小编还买过。但是这次的上头行为引起了大家强烈支持，硬是拍到了 6 eth！而且 Twitter 上的评论区里热心群众开始踊跃点歌：下次写《This Song Is A Wells Notice》、《This Song Is A Judge Rejecting The SEC’s Theory》好不好，还可以凑个专辑咧！

Web3 音乐人之星

Henry

Web3 音乐人越来越多了，我们争取每期选取一个略加介绍。说不定是从 sound.xyz 上扔骰子选的，不构成任何投资建议啊～

本期 Web3 音乐人之星

对，就是上面说 Spotify 播放量达到百万的那位。

来自 LA 的 Sara Phillips 是一位在 NFT 界冉冉升起的歌手兼创作人。她是加拿大人，本来在蒙特利尔的乡下长大。在她的童年记忆里，父母和姐姐都爱听音乐。她经常偷偷拿姐姐的 CD 听，除了小甜甜布兰妮那些，姐姐还经常有些非主流风格的收藏品让年幼的 Sara 听不懂但大受震撼。这就是她童年的音乐启蒙。

高中时期的 Sara 开始学习乐器并参加学校的乐队。偶尔，她也抱着吉他瞎弹瞎编，但是因为害羞，从未公开发过。有好事的同学替她发到 Facebook 上，她还很生气。当她发现人们好像还挺喜欢看萌妹子弹琴唱歌以后，她开始有了自信，干脆经常自己发 Youtube 了。

大学毕业后 Sara 到纽约闯荡，喧嚣的曼哈顿让小镇青年大开眼界。从前她从未想过从事音乐这项职业，但是遍地音乐人的环境让她开始有了许多参与演出的机会，也磨炼了她演唱和创作的才能。朋友们告诉她，如果她想成为专业的音乐人，LA 才是更适合的地方，于是纽漂 Sara 变成了洛漂。

LA 的音乐界朋友很快向她 CX 了加密货币和 NFT，并把她拉进了一些社群。怀着试试看的心情，Sara 在 2021 年底发行了第一个音乐 NFT。一开始并没有人买，但是几个月后随着市场的火热，她惊异地发现几个 NFT 都被抢购一空，这让她在感慨 “太疯狂了” 的同时开始认真审视这种新形式。

在 Sara 看来，NFT 粉丝和 Spotify 上的听众是截然不同的 —— 她根本不知道 Spotify 上有谁在听她的歌，但是 NFT 的收藏者则是鲜活的，她也更愿意和这些人互动，用自己的才华回馈他们。

除了当做数字艺术品以外，Sara 坚信 NFT 还有更多用武之地尚待发掘。但是有一点 Sara 表示该改改 ——NFT 这个名字实在是太不受圈外人欢迎了。
