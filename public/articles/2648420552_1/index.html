W&M 报告 105-4：音乐行业的 Web3 情绪现状

作者: 翻译公会 Yuchen 发布时间: 2022-07-08 09:23:49
分类: 音乐行业, 音乐 NFT, W&M, 数字艺术, NFT
充满怀疑的时刻。

太长不看版：作为对第 1 季中围绕音乐 / Web3 项目的粉丝情绪的大量研究的后续，我们开始调查分析艺术家和音乐行业的专业人士对 Web3 的各种形式的感受，包括 NFT、DAO 和社交通证。我们的调查收集了来自多个行业角色的超过 250 位人员的答复，从艺术家、艺人经理和唱片公司高管到自由营销者和初创公司创始人。我们发现，无论参与 Web3 项目的程度如何，音乐行业的专业人士与粉丝一样，都对未来的发展前景有许多相同的担忧，特别是在涉及到教育和财务障碍以及法律和环境问题时。此外，虽然围绕音乐 NFT 的讨论持续两极分化，但对 Web3 的某些实现方式的熟悉程度并不会自动转化为对这些应用的积极情绪。

在上一个合作研究季，我们发现音乐 / Web3 的未来取决于营销、文化和公众对生态系统的认知，就像它取决于基础技术本身一样。

我们围绕这一论题建立了许多新兴的研究议程，强调了影响者的核心作用以及他们的举动对推动（或阻碍）音乐 / Web3 发展方面的影响。我们仔细研究了我们的艺术家 NFT 数据库中旗舰产品的新闻稿，并采访了知名音乐 NFT 项目背后的艺术家和初创公司们，从而强调他们的粉丝入驻策略。我们调查了 150 多位粉丝对音乐和 Web3 的看法，并建立了一个由四种粉丝角色组成的框架，从完全没有接触过 Web3 的人到以多种方式接触该技术的人。我们分析了粉丝对推特上数百个音乐 NFT 公告的反应，对粉丝的情绪（尤其是涉及到环境和财务便利性等关键主题的情绪）有了更客观全面的了解。

作为对 Web3 的粉丝情绪回顾的后续，我们的社区对另一方的情绪（也就是艺术家和音乐行业的专业人士）可能是什么样的产生了质疑。

我们的终极目标是：更好地了解——包括定量研究和定性研究——整个音乐行业对 Web3 的各种形式的感受。

最近音乐 / Web3 的头条新闻往往把音乐行业与 Web3 的关系描绘成一个情绪化的、不稳定的过山车，从 Hitpiece 等 NFT 平台的恐吓策略到 Snoop Dogg 等名人的 NFT 野心，二者的关系更多是由炒作而非实质驱动。如果你研究金钱的流向会发现，与此同时，主要的音乐公司一直在越来越认真地投资 Web3 初创公司，并雇佣更多 Web3 内部人才，这发出了一个信号，即他们把这项技术视为其长期商业战略的一个重要组成部分。

2022 年 2 月，我们进行了一项调查，测量了众多艺术家和音乐行业专业人士的 Web3 情绪。该调查询问了受访者对 Web3 和音乐（包括 NFT、DAO 和社交通证等应用）的熟悉程度、希望和担忧。希望我们的调查结果对广大受众（从希望了解音乐行业观点的软件开发者，到希望了解同行对 Web3 看法的艺术家及其团队）来说是有价值和可操作的。

研究方法

这个项目的研究方法基于一系列的研究类电话会议。Water & Music 的成员集思广益，为艺术家和音乐行业的专业人士制定了一份调查问卷。我们在 Tally 上建立了调查，最终收集到 252 人的回复。为了鼓励参与，我们为受访者制定了激励措施，如果他们提供自己的电子邮件地址，就可以获得原始和匿名的调查数据。将在季后被分发出去的大部分调查数据都在这篇文章及我们的数据看板中有所体现。

调查结束后，我们在基础的研究回答中寻找普遍的趋势，并通过对开放式回答进行数据分析和定性整理来检验我们基于先前的知识和在该领域的工作（例如我们以前的粉丝调查）所提出的假设，以得出关键主题。

我们选择通过 Tableau Public 以交互式数据看板的形式展示数据。读者可以根据自己的意愿通过下方提供的可视化工具探索数据。本报告将带你了解这些数据看板，突出关键趋势，同时还有对调查开放式回复的引用。

免责声明和限制条款

我们希望打消可能会影响到我们研究的各种偏见。首先，我们承认，由于我们组织的性质更倾向于DAO，所以许多 Water & Music 的成员在专业和个人方面都对 Web3 感兴趣。考虑到我们的许多成员都填写了调查表，因此获得的数据可能存在固有的偏见。

我们还要承认，我们故意不在调查中涵盖 Web3、NFT、DAO 或社交通证等术语的定义。这么做是为了让受访者根据他们自己对这些术语的定义进行操作，而不是用我们的定义自上而下地影响他们操作。我们在第 1.5 季报告里讨论了其中一些术语的模糊性。

受访者

角色和流派

我们要求受访者在音乐行业中选择一个最能准确描述他们的角色。最常见的选择是艺术家，占受访者的四分之一以上（ 26% ），其次是从事音乐技术工作的人（ 18% ），然后是唱片公司（ 14% ）。11% 的受访者选择了「其他」角色——包括但不限于音乐营销、音乐策划、音乐总监、广告主需求方（demand side platform）、分销商 / 标签服务、咨询和公关。

对于艺术家，我们询问他们是通过大厂牌或独立厂牌发行音乐还是自己发行。大多数参加调查的艺术家选择了自己发行（65%），其次是通过独立厂牌（32%）和大厂牌（3%）。虽然我们现在意识到许多艺术家选择通过不止一种方式发行，但我们可以自信地说，调查中艺术家的回答代表的是独立厂牌和个人的角度，而非大品牌的角度。值得注意的是，这与我们 过去的发现 是一致的，即在音乐 / Web3 生态系统中，绝大多数的销售和商业活动都来自独立和未签约的艺术家。

我们要求那些在角色上选择了 「其他」 而非艺术家和制片人的受访者自陈他们在当前职位上的资历水平。大多数在音乐行业工作的受访者将自己归类为高级管理层（主管 / 总监 / 副总裁 / 高级副总裁 / 执行副总裁 / 创始人）（55%），其次是高级职员（主管 / 总监 / 经理 / 制作人 / 买手）（18%）。自由职业者占 16%，其次是初级职员（7%）和「其他」（4%）。

我们还询问受访者哪一个广泛的流派最能描述他们通常专业操作的音乐，我们允许受访者多选，只要他们觉得合适。在所有受访者中，电子音乐（47%）和独立 / 另类音乐（43%）领先，其次是流行音乐（32%）和嘻哈音乐（31%）。我们还加入了一个开放式回答的选项。最常见的回答是实验音乐（3%）和新式非洲音乐（2%），其他各种回答包括基督教音乐、原声音乐和日本音乐。

Web3 画像

为了更加了解我们的调查对象对 Web3 的参与程度，我们问了两个问题。

第一个有区分度的问题与我们关于粉丝入驻策略的第一季报告直接并行，在那个报告中我们根据 Web3 的参与程度将粉丝分为四个不同的类别。

接着，这次我们以同样的方式对音乐行业的受访者进行分类，详见下方：

  1. 犹豫组：「我目前不拥有任何加密货币」（83 个答复，33%）

  2. 仅持有加密货币：「我有加密货币，但不拥有NFT」（71 个回答，28%）

  3. 非音乐NFT持有者：「我拥有加密货币和NFT，但没有音乐NFT」（49 个回答，19%）

  4. 音乐NFT持有者：「我拥有加密货币和音乐NFT」（49 位受访者，19%）

我们的第二个判断性问题是受访者是否在音乐行业内参加过 Web3 项目或与之有关联。样本中的大多数人（63%）在接受调查之前没有接触过 Web3 项目。此外，61%的受访者不持有 NFT。

对加密货币 / NFT 的熟悉程度和参与 Web3 项目可能性之间的关系也有少数例外。例如，一些「犹豫组」和「仅有加密货币」的受访者实际上已经参与过 Web3 项目的工作（表明这也许是一种健康的、参与性的怀疑态度），而许多 NFT 持有者还没有参与过一个项目。也就是说，大多数 「对加密货币持观望态度」的受访者（82%）还没有参与过 Web3 项目，而大多数音乐NFT持有者（69%）都参与过 Web3 项目。这与我们在第 1.5 季的研究中的发现相关联，即在 Web3 中，创造者和收藏家之间的界限是有意模糊的，许多音乐 NFT 持有者直接参与了塑造生态系统的成长和发展。

人口统计

最后，我们对收集受访者的人口统计资料很感兴趣。考虑到围绕加密货币持有者的刻板印象，也许是意料之内的，受访者中最常见的是居住在美国的千禧一代男性。

几乎 80% 的受访者集中在排名前四的国家：美国（50%）、英国（15%）、加拿大（7%）和澳大利亚（7%）。虽然样本量较小，但中国和荷兰的 NFT 持有者的集中度最高（每个国家有3名受访者，分别为 100% 和 67%），其次是美国，有 45% 的受访者持有 NFT。澳大利亚的 NFT 持有者的集中度最低，只有 11%的人持有 NFT，其次是哥伦比亚（25%）和英国（28%）。

正如我们预料，受访者大多数是千禧一代（57%），其次是X世代（22%）。但我们惊讶地发现，只有 11% 的受访者是 Z 世代——也许这表明年轻一代可能不那么看好音乐 / Web3 行业，或者表明 W&M 网络向其他世代倾斜。为了支持前一种解释，我们发现，在数据中探索代际和 Web3 画像之间的关系时，Z 世代的受访者是最不可能持有 NFT 的一代——只有 29% 的人持有 NFT，相比之下，千禧一代为 41%，X世代为 38%，婴儿潮一代为 33%。

在探索受访者世代和行业角色之间的关系时，令人惊讶的是，婴儿潮一代最集中的是音乐行业的演出现场工作人员（18%），而 X 世代最常见的行业角色是制作人 / 词曲作者（42%）。千禧一代在出版 / 音乐总监（83%）和唱片公司（71%）的角色中占了最大的密度，而 Z 世代最常见的角色是艺人经理、自由职业者和音乐技术（14%），三者并列。

虽然可能已经对 Web3 世界和音乐行业的性别差距习以为常，但当我们看到受访者中 71% 的受访者认为自己是男性，只有 19% 的受访者认为自己是女性，3% 的受访者认为自己是非二元性别时，还是会感到惊讶。男性受访者中的 42% 持有NFT，相比之下，女性受访者中的 28% 和非二元性别受访者的 43% 持有 NFT（尽管最后这一类的样本量也相对较小）。有趣的是，在探讨性别和角色之间的关系时，女性受访者的角色类别高度集中在「其他」，占 28%，其次是艺人经理（27%）、艺术家与出版 / 音乐总监，均占 17%。女性人数最少的角色是演出现场工作人员（0%）、唱片公司（11%）和制作人 / 词曲作者（13%）。

对 Web3 的熟悉程度和情绪

为了解我们的受访者对哪些 Web3 概念最熟悉，我们将重点放在音乐中最常见的三个 Web3 用例上：NFT、DAO和社交通证。

NFT 是人们最熟悉的 Web3 概念，94% 的受访者表示他们熟悉这个术语。第二个最熟悉的概念是DAO，70% 的受访者表示熟悉（24% 非常熟悉，46% 比较熟悉），30% 不熟悉。社交通证是最不熟悉的概念，有 33% 的人表示不熟悉。

当我们只关注「在 Web3 项目中工作过」的受访者以及他们参与的项目类型时，这种熟悉程度得到了进一步的细分。在这些受访者中，有很多人发起或参与过 NFT 项目（82%），参与 DAO 的比例略低（54%），参与社交通证的比例则低得多（只有 26%）。

这种回答的分布相对来说并不令人惊讶，它遵循基本遵循每个概念热潮涌现的时间，NFT 在 2020 年底和 2021 年初推动了 Web3 的讨论，而人们对 DAO 和社交通证的广泛兴趣在2021 年晚些时候才开始。Water & Music 之前的研究也支持这一点。根据 我们的音乐 / Web3 数据看板 记录，自从我们在 2020 年秋季首次开始跟踪生态系统以来，已经有超过 90 家音乐 NFT 初创公司成立。以音乐为中心的 DAO 的出现也同样稳定，但规模比 NFT 平台小，这在我们 关于音乐 DAO 现状的第 1.5 季报告 中有详细的说明。社交通证在参与度方面居后，是唯一一个许多参与过 Web3 项目的受访者没有参与过的类别。也就是说，考虑到最近的趋势是把社交通证作为艺术家围绕其作品 发展社区 的一种手段，我们预计在不久的将来会看到人们对社交通证的熟悉程度迅速提高。(值得注意的是，独立艺术家 RAC 的社区通证 $RAC 最近 通过 Coinbase 成为有史以来第一个在大型中心化加密货币交易所上市的艺术家通证）

虽然了解熟悉程度对于了解音乐行业如何参与 Web3 很重要，但对 Web3 的情感关联的深入研究能使我们更好地了解这些举措的势头和潜力。

在我们的调查中，我们让受访者说明他们对 NFT、DAO 和社交通证的态度是「不喜欢」、「喜欢」还是「中立」。结果表明，熟悉程度高并不一定意味着对 Web3 音乐应用有正面感受。

受访者最熟悉的用例——NFT，也是最两极化的：27% 的受访者说他们「不喜欢」NFT，其次是 32% 的人持「中立」态度，41% 的人「喜欢」NFT。最受欢迎的概念是DAO，有 44% 的人持积极态度，41% 的人持「中立」态度，15% 的人选择了「不喜欢」。受访者对社交通证（不熟悉程度最高的概念）的态度最中立，54% 的受访者选择「中立」，27% 选择「喜欢」，19% 选择「不喜欢」。考虑到围绕每个用例的流行话题，这些结果并不意外。例如，我们的第一季 关于粉丝情绪 和 粉丝入驻报告 显示了粉丝们的担忧以及针对 NFT 的发展的大量负面情绪，尤其是负面的环境影响、高成本和诈骗。

在调查中，NFT 和 NFT 项目在不喜欢哪些音乐 / Web3 项目的开放性回答中占主导地位。许多人反对高昂的价格，并指出对环境和道德问题的广泛担忧。另一方面，DAO受到了 更多正面报道（至少在文化领域而不是 技术或金融领域），这可能影响了我们看到的结果。正如我们在 关于音乐 DAO 的现状的第 1.5 季报告 中所探讨的那样，在 Web2 行业担忧的背景下，更扁平的组织结构的叙事和更透明的数据和资本流动是动员音乐社区的强大力量。当我们询问受访者是否看到或参与过任何他们喜欢或感到兴奋的音乐 / Web3 项目时，许多开放性回答谈到了对音乐 DAO 项目感到兴奋；许多项目被多次提及，如 Songcamp。

总而言之，对 NFT、DAO 和社交通证持否定态度的受访者与那些在任何时间段都「不打算在 Web3项目工作」的受访者之间有很大的重叠（占了每个类别中 10-13% 的「不喜欢」）。这些回答体现了受访者的防御，他们似乎坚决反对 Web3，而且无论技术使用情况如何，似乎都对探索它不感兴趣。

你参与过哪些Web3项目？

正如我们根据熟悉程度的结果所预期的那样，在从事过 Web3 项目的受访者中，NFT 是最常被提及的项目类型，37% 的受访者发行过至少一个项目，45% 的人目前参与了至少一个项目。在这些受访者中，只有相对较少的人（18%）没有参与过 NFT 项目。有趣的是，45% 的受访者表示他们目前正在参与的 DAO 项目，这表明人们对 Web3 技术的具体实施的兴趣增加，也许强调了对 DAO 的积极情绪的影响在扩张。正如我们在上文的熟悉程度部分所预期的那样，Web3 用例中参与人数最少的是社交通证，74% 的受访者没有从事过通证工作，只有 22% 的人目前正在从事通证工作，只有极少的 4% 的人已经发行过一个项目。

你打算参与Web3项目吗？

最后，许多没有参与过项目的受访者仍然对在 Web3 中工作的想法持开放态度，39% 的人表示他们有意愿在未来参与进来。有这个想法的最常见的角色类别是艺术家、艺人经理和在唱片公司工作的人。同样，38% 的受访者尚未决定是否会参与 Web3 项目，但对这种可能性持开放态度，特别是在艺术家、唱片公司和音乐技术工作者中。总的来说，大多数刚接触 Web3 的受访者（77%）至少对探索生态系统中的项目持开放态度，只有 23% 尚未参与项目的受访者表示他们打算保持原有的态度。换句话说，在所有调查对象中，只有 15% 的人表示他们「不打算在未来参与Web3项目」。这些受访者中的大多数要么是艺术家本人，要么是制作人 / 词曲作者。

Web3担忧

为了了解受访者中最普遍的与 Web3 有关的担忧，我们仔细地将参与过 Web3 项目的人和没有参与过的人进行了比较。虽然在我们的结果中出现的每个担忧都有足够的材料来支撑起一个专门的文章，但我们将只对前五个担忧作进一步的探讨。

综合样本中最常见的担忧是：

  1. 法律担忧（51%）

  2. 个人环境担忧（48%）

  3. 过程太复杂或看起来太复杂(46%)

  4. 对加密社区的看法（46％）

  5. 粉丝的成本(43%)

  6. 因环境问题而产生的粉丝抵制(42%)

  7. 艺术家的成本(37%)

  8. 困惑——太多选择 / 资源(36%)

  9. 担心项目的成功(36%)

  10. 其他（20%）

  11. 没有担忧（7%）

法律担忧

无论是否在 Web3 工作过，法律担忧都是最常被提到的问题，超过 50% 的受访者表达了这种担忧。鉴于在 Web3 的无数领域中存在的许多法律问题，包括一些 NFT 是否有资格成为证券的未决问题，这么多人有担忧是很合理的。法律相关的问题在音乐 / Web3 应用中尤为普遍，现有的版权法和在区块链上部署音乐 NFT 的许多方法之间存在着十分紧张的关系。我们在 关于定义音乐 NFT 所有权的第一季报告 以及最近发布的概述我们自己的 模块化音乐 NFT 合约框架的第 1.5 季文章 中展开讨论了这些紧张关系。

我们看到，有音乐技术背景的受访者将法律问题列为他们最关心的问题。这是合乎逻辑的，因为在一个涉及技术的音乐项目中，可能会出现许多法律责任和问题。如果你是一个会技术的建设者，对法律相关问题的处理将会决定你的产品是会成功还是不断收到诉讼，甚至可能是彻底失败。我们只需要看看最近的事，就可以知道音乐行业复杂的法律框架是如何给音乐初创公司带来生存问题的。有野心的技术项目，比如 ConsenSys 的 Ujo Music，甚至资金充足的行业联盟，如全球版权数据库项目，在遇到 全球音乐版权系统 这一复杂的法律环境时，也都只有死路一条。考虑到这一点，现在的音乐 / Web3 建设者的作品，尤其是提供新产品（比如收取版税的 NFT）的平台，本质上是在挑战法律和监管机构——等待正名，但在此期间继续盲目建设。

这一点在我们的评论中也有所体现，一位音乐技术领域的高级经理解释了其中的利害关系，他指出「法律是一条硬线，可以让你倒闭，尤其因为 Web3 是全球性的，但法律和法规不是。」这里的突出观点是，与各个地区中各异的「迟缓笨重」的合同不同，智能合约不受地域限制。各国如何执行各自的版权法这一错综复杂的问题，目前还没有映射到区块链的自动化和无地域限制的基础设施。

这一点在我们的评论中也有所体现，一位音乐技术领域的高级经理解释了其中的利害关系，他指出「法律是一条硬线，可以让你倒闭，尤其因为 Web3 是全球性的，但法律和法规不是。」这里的突出观点是，智能合约不受地域限制。各国如何执行各自的版权法这一错综复杂的问题，目前还没有映射到区块链的自动化和无地域限制的基础设施。

对于我们样本中的艺术家来说，法律问题在他们的担忧中排名较低（第五名），但还是出现在开放性回答中。当然，许多艺术家可能会有类似于前文所述的音乐技术领域的监管问题，特别是如果他们正在发行自己的 NFT 或通证项目。然而，艺术家们也表达了对 Web3 可能从潜在的侵犯版权的角度对他们产生负面影响的焦虑。一位摇滚乐艺术家告诉我们：「别人「铸造」他们不拥有的东西实在是太简单了，艺术家对此没有办法」。他还有些沮丧地指出，「到最后，创作作品的艺术家拥有这个作品」。这些不仅仅是艺术家们凭空的担忧。例如，OpenSea 已经公开证实，在用他们的免费铸造工具创作的 NFT 中，超过 80% 都是「剽窃的作品、虚假的收藏和垃圾邮件」。同样，作为现有主要流媒体平台的潜在 Web3 替代方案而受到关注的音乐流媒体初创公司 Audius，也同样面临着「被盗版侵袭」的情况。

鉴于 Web3 技术的快节奏的特点以及其发展推挤了现有法律和监管政策发展的事实，我们预计对法律问题的担忧将持续在对 Web3 感兴趣的音乐行业从业者中占领重要地位。

环境关切（包括个人关切和粉丝的抵制）

在过去的一年里，环境关切已经严重影响了音乐 / Web3 生态，包括随着 2021 年 NFT 的发展而产生的许多气候方面的担忧，也包括人们日益增长的对气候问题解决方案的兴趣。例如，许多关注气候问题的 Web3 工作者看到了为了加密安全而基于权益证明机制的区块链的潜力，这种区块链为常见的基于工作量证明的区块链协议（如比特币或以太坊）带来的高环境成本提供了解决方案。

在调查中，我们要求受访者将与个人道德有关的环境关切和与预期的粉丝抵制有关的环境关切区分开来。

个人对环境的担忧是受访者最常提到的第二大担忧，有48%的受访者选择了这个选项（更具体地说，是 48% 的曾参与过 Web3 项目的受访者和 47% 的未参与过 Web3 项目的受访者）。一些受访者对这些问题作了进一步解释：

「我觉得大多数人，像我一样，也许更担心既参与危害地球的事情却又同时又用演化和创新的内驱力来为其正名。当然，任何新技术都会随着时间的推移变得更高效，但在一个不断发展的空间里，没有时间去等待。不过这是内部斗争。」——曾在 Web3 工作的艺术家

「第二是环境关切，他们一直说这将在下一层得到解决，但如果它这么容易解决，那么为什么没有发生。」——未在 Web3 工作过的音乐技术领域的自由职业者

这些评论反映了一些人之间持续的紧张关系，有的人认为，为了换取 Web3 技术所承诺的未来的进步利益，一些短期的环境破坏是可以容忍的，而另一些人则不太愿意接受这些常常是未经证实的未来利益的说法带来的表面价值，因此更不愿意用当前的环境破坏换取他们认为未经证实的说法。

除了个人担忧之外，行业的另一个关键考虑因素是对因环境影响而导致的粉丝抵制的预期，受访者中有 42% 的人都有这个共同的担忧。在我们2021年的粉丝调查中，受访者的一个明显主题是强烈认为加密货币、Web3 和区块链的其他方面对环境不利。在「对加密货币持怀疑态度」的粉丝中，超过43% 的受访者声称，环境原因是他们尚未购买加密货币的首要原因。

这一次，我们的受访者意识到了许多粉丝对 Web3 项目的负面情绪。对于那些曾经参与过 Web3 项目的人来说，由于环境影响而引起的粉丝抵制是第三大担忧（53%）。这在唱片公司、艺术家、艺人经理、演出现场工作人员和制作人的角色中尤其频繁——基本上就是创作者本人或者是与他们密切合作的人。对于那些没有参与过 Web3 项目的人来说，粉丝抵制是第八大担忧，有 34% 的受访者选择了这一选项，这也许表明那些参与过 Web3 项目的人在自己试水 Web3 之后，对粉丝抵制有了更敏锐的认识。一位艺人经理说，他们因为粉丝的负面反应放弃了一个项目。「我们所做的艺术家NFT是个一次性的项目，而且在收到一些关于环境问题的抵制后，艺术家不支持它了。所有的推动力都没了。」

粉丝和艺术家的财务成本

43% 的受访者认为，粉丝的成本 是第五大担忧。对于那些曾经参与过 Web3 项目的人，特别是对艺人经理、音乐技术和制作人以及 NFT 持有者来说，这是他们最关心的问题。相比之下，对于那些没有参与过 Web3 项目的人来说，对粉丝的成本的担忧排名第五。

我们研究 Web3 粉丝情绪的第一季调查显示，即使是最以 Web3 为导向的回答（即「音乐 NFT 持有者」的回答）也将成本列为铸造音乐 NFT 之前的首要关注。这意味着成本是将粉丝转化为买家的影响最大的变量。有许多事件和因素是艺术家无法控制的，而艺术家最能控制的一个杠杆似乎对销售的转化最为重要。

为什么成本会有如此大的影响？它是对公允价值的感知吗？它是在一个饱受骗局现实和被夸大的骗局叙事纠缠的生态中的真诚之表现？是因为在市场上创造人为的稀缺性的欲望？还是因为卖光的压力？

一些在经济上最成功的音乐 / Web3 用例涉及为艺术家提供通过非传统方式筹集资金的工具。也就是说，由于围绕该技术的财务和教育摩擦，这些产品最终往往只能接触到艺术家受众中的一部分超级粉丝或富裕的收藏家。Latashá 通过高价的 1 / 1 音乐视频 NFT 获得了令人难以置信的成功，这些 NFT 最终全部售出。同样，Ibn Inglor 的 DANGER WORLD 众筹最终在 42 个支持者中筹集了 20 个 ETH，平均每个支持者 0.48 个 ETH，按今天的 ETH 价格计算，差不多有 1400 美元。很容易理解为什么像这样的销售可能会让许多艺术家的粉丝支付不起。

从面向行业的角度来看，这里的主要启示是成本会影响接受度。Henry Chatfield 是 Web3 生态系统中一位活跃的音乐经理，他对「bloghaus」时代和他最近与 Daniel Allan 一起推出免费 POAP 的经历有一个有深度的、富有启发性的音乐相关的见解。他提到了Lina Abascal的《不再孤独》（「Never Be Alone Again」）一书 —— 这本书对 2006 年至 2011 年之间的音乐进行描述，那是一个文化在数字技术的支持下得到传播的时期，当时音乐能通过博客和P2P网络被自由广泛地传播。「这本书强调了在这个时代，音乐是如何做到在很多方面 「免费」的」，以及像 A-Trak、Kid Cudi、Steve Aoki、Justice、Diplo 和 Chromeo 等艺术家，是如何在一个被亲切地称为 Bloghaus 的流派中获得了成功。正如 Henry 所指出的，我们今天在 Web3 音乐和 NFT 中看到的影响参与的经济障碍在当时并不存在。

当把 2000 年代中期这种更开放、更易达的音乐生态系统与围绕当前音乐 / Web3 生态系统的成本进行比较时，不难看出 Web3 可能会让一些人感到被冷落。例如，在 2021 年 10 月，Daniel Allan 完成了一个 Mirror 众筹，筹集了 20 个 ETH（当时约 16 万美元）。在总共 87 个支持者中，有 41 人捐赠了 0.1 个 ETH 或大约 400 美元。相比之下，2021 年 1 月，当Allan发布可自由认领的 POAP 时，超过 500 人给他的团队发了关于这个 POAP 的信息，300 多名粉丝完成了多重步骤认领了一个 POAP。对 POAP 发行的出人意料的反应表明，人们对 Allan 的艺术有广泛的需求和兴趣。这也暗示，对日常音乐爱好者来说价格更合理的音乐和 Web3 项目可能是艺术家成功的方法之一，同时避免了对排他性的抱怨。

W&M 在最近的两次调查中已经报道了粉丝进入音乐 / Web3 世界的高昂成本，但艺术家参与进来的成本也值得一提。对于个体艺术家来说，在以太坊主网上发行单一的定制智能合约可能需要 1500 到 2000 美元的矿工费——这还不包括与 Web3 开发者合作的费用，以及教育自己和粉丝了解 Web3 所需的资源，所有这些都是在创作艺术的时间和物质方面的基准成本之上。即使一个艺术家决定在第三方平台上铸造他们的作品，也至少有一个上市 NFT 的成本，以及出售时要支付的矿工费和转移以太坊时要支付的矿工费。在以太坊区块链的交易成本极高的情况下，其他成本较低的选择正在获得发展的势头，比如 alt-Layer 1 链（如 Solana、Flow）和 Layer 2 链（如Polygon、Optimism）。举个例子，冰岛组艺术家 Sigur Rós 最近在 Polygon 上推出了他们的 HEIMR 项目，这降低了交易成本。

对加密社区的看法

公众对加密社区的看法是一个值得关注的问题——是没有从事过 Web3 项目的人的第三大担忧，从事过的人的第五大担忧。

这个问题排名位于其他讨论问题之后。如果有人参与了一个声称会延续现有财富差距、扩散诈骗、伤害环境并播种「加密兄弟（crypto bro）」文化的生态系统，这说明了什么？这是艺术家、艺术家团队和唱片公司在加入 Web3 之前考虑的核心问题，并且在一个唱片公司的高级工作人员的书面回复中非常直接地表现出来，他告诉我们，「被看到或被认为与 「福音派（evangelicals）」和 技术兄弟（Tech Bro’s）接近，会对你的音乐事业产生不利影响。」

围绕加密货币普遍的负面联想是整个生态系统必须克服的真正障碍，以便它被需要的大规模接受，而这需要先实现目前涌入音乐 / Web3生态系统的所有资金的 所许下的承诺。对音乐 NFT 平台来说尤其如此，正如我们在最近的 第 1.5 季平台入驻研究 中所显示的那样，这些平台在很大程度上依靠艺术家作为入驻新用户的关键载体。

如果像一个唱片公司的高级职员告诉我们的那样，「很多艺术家担心那些没有参与 Web3 的人对他们的看法」，那么 Web3 的音乐平台清除障碍的第一步就是说服艺术家，他们声称提供给他们的完全公平和利益是真实的。随着加密货币的社会和金融波动，一家大型艺术管理公司的营销人员告诉我们，艺术家更愿意「让别人当小白鼠。我认为经理人和艺术家对 Web3 格外感兴趣，但他们希望正确地参与进来。」

过于复杂的流程

在 Web3 生态中做先行者的复杂和困惑，是所有受访者提到的第三大问题（46%）。我们在 先前的研究 中讨论过的一个观点是，如今很有争议的问题在于是否应该把 Web3 的复杂性以设计手段去除，以增加主流接受度。一位受访者说，他感到 「被信息淹没…不知道去哪里找资源 / 同行。」

我们并不清楚受访者是否认为这个过程对他们的团队或粉丝来说太复杂了。不过，书面回答提供了一些背景。一位艺术家分享到：「在学习 defi 的同时也要学习如何使用 MetaMask 的这种需求是真实的。」另一位则「不打算让粉丝们进入 MetaMask ——这对现在的大众来说过于复杂了」。一位受访者认为这些流程实际上与他们的目的背道而驰：「我的工作是让这些东西对艺术家更容易。但必须了解加密货币、钱包和通证才能参与一个空间并不会让事情变得更容易——它让艺术家的生活变得更难，还增加了额外的风险（波动性、前期铸造成本和交易费用），并且只对粉丝中非常有限的一部分人有价值，许多其他人则被拒之门外。」

Web3 的动机

对于那些在 Web3 项目中工作过的人来说，代表角色和 Web3 画像的 75% 的受访者 的主要动机是尝试新技术。68% 的受访者还表示，他们受到启发而从事一个项目，从而探索创造社区和连接艺术家与粉丝的新方法。有趣的是，「财务机会」和「同龄人在做这个 / 其他人在做这个」只是不到一半的参与过 Web3 项目的受访者的动机（分别为 32% 和 27%）。

你的项目是否成功？

对于那些已经启动过项目的人，我们问他们觉得自己的项目是否成功，以及为什么那样觉得。一些人回答说，他们的项目在各种财务和营销指标方面确实是成功的。一位电子艺术家指出，他们「从 60 秒的音乐中赚到的钱比我通过 Bandcamp 和流媒体发行的无数张 EP 和专辑赚到的钱还要多」；另一位爵士 / 布鲁斯艺术家指出，他们的项目是成功的，因为它登上了 OpenSea 的首页。

其他人发现他们的项目并不那么成功。一位音乐技术领域的高级经理指出，虽然他们的项目受到了明显的欢迎，但并没有转化为指标的结果。一位艺人经理说，在受到粉丝对环境问题的抵制后，他们的一次性 NFT 被艺术家完全放弃了。一位顾问澄清说，他们的项目「没有成功，因为概念非常典型，创作者没有完全理解 Web3 的意义、价值和机会。」

许多受访者表示，在 Web3 中发布项目的行为本身就是一种成功，因为涉及到教育和社区建设的后续效应。一位艺术家说，「无论是否售罄，发布项目的行为都是成功的。」另一位从事节日市场营销工作的受访者认为他们的发布是成功的，「因为它是一种学习的经验。」一位将自己归类为超级粉丝的受访者说：「我觉得我已经成功了，因为我现在认识了一群人，并感到自己被他们接受。」

其他回答强调了长期的规模化和持久性作为 Web3 成功的关键因素的重要性，而不仅仅是短期的销售。一位音乐技术领域的高级经理宣称，他们认为自己的项目是成功的，因为「在考虑规模化的未来工具时，实验性的视角是很重要的。」另一位艺术家宣称：

「我在判断 Web3 的成功时，首先要看是否成功推出一个项目。我相信音乐 NFT 最终会找到它们的收藏者，所以耐心是很重要的。对我来说，有些是超级快就成功了，有些则是在铸造 7 个月后才成功。我有几个未售出的 1 / 1 NFT，但它们在以太坊区块链上的这个事实？这是很重要的。我的整个尝试是植根于我们把文化放在链上的必要性。」

开放性回答的主题

在主要的调查回复提供了关于艺术家和行业对 Web3 的情绪趋势的数字化清晰度的同时，我们还收集了广泛的开放性回答。除其他事项外，这些问题要求受访者展开描述他们对 Web3 可能有的任何担忧，并描述他们喜欢或不喜欢的音乐 / Web3项目以及原因。

这些回答使我们能够进一步挖掘调查中出现的关键主题，并使我们的整体调查结果更丰富以及更有深度。在下方，我们将扩展一些在分析和编码书面答复的过程中出现的主要线索。其中一些广泛的主题，如对诈骗的关注以及对更多教育和更简单的用户体验的呼吁，对我们来说并不陌生。其他的回答围绕常见的 Web3 和音乐问题提供了考虑周到的细节和深度，包括 Web3 中缺乏音乐行业知识，以及通常被认为是不公平的组织结构的进步替代方案的 Web3 可能只是在 Web2 下重新创建平台资本主义结构的这一风险。

诈骗

粉丝们在我们关于粉丝情绪的第一季报告中表达的主要担心基本也是现在围绕 Web3 的主要焦虑。诈骗和需要更多的教育和更好的用户体验设计都排名靠前，粉丝们认为为了提高 Web3 的参与度，这些问题需要得到解决。

关于骗局，由于我们的调查是在 HitPiecel 的丑闻出现后发布的，所以失去了公平。HitPiecel 这个平台声称在其网站上销售音乐 NFT（或至少是为潜在可销售的 NFT 而展示音乐），但没有事先得到艺术家的许可。可以预见的是，这导致了巨大的抵制，包括许多 批评文章 和大量的负面社会媒体报道。这次惨败对我们调查的影响，可以从受访者对不喜欢的 Web3 项目的问题答复中看出；HitPiece 是受访者提到的最多的平台，仅次于「一切」。

虽然骗局显然仍是许多人的首要顾虑，但也有迹象表明，围绕加密货币的骗局说法正在出现变化，受访者的许多评论表明，尽管他们有担忧，但仍愿意接触这项技术。正如一位受访者所说：

「加密货币对我来说还是个相当新兴的东西，所以我会有许多标准的担忧（环境、诈骗、成本、公平等），这也是我正在进行的 Web3 研究的一部分。我仍然有许多这样的担忧，但没有让它们阻止我学习和参与 Web3 项目。」

其他评论认为骗局与目前出现的技术、工具和用户界面的状况有关。「我认为Web3仍然是一个由货币、钱包、混乱的『狂野西部』（在这里诈骗大量存在）组成的热乱局面。」这个框架是有先见之明的，这表明需要一个更好的用户界面，使坏人的一些最常见的骗局（如 网络钓鱼攻击 ）变得更加困难。

「对音乐行业一无所知」

HitPiece 争论的一个有趣的结果是，出现了对为什么这些项目被认为是有问题的更细微的看法。许多受访者只是简单地写了「骗局」，但也有许多更细微的观点，试图承认 Web3 与现有音乐行业的具体问题。

在那些更细微的观点中多次出现的一个值得注意的主题是，在音乐 / Web3生态系统中建设的人不知道他们在做什么，特别是在音乐行业方面。一位来自英国的独立电子艺术家受访者，直接提出了这一主张，他告诉我们，「有太多的人不知道音乐行业是如何运作的，他们提出了改变世界的主张，但只要稍加审查，这些主张就会分崩离析。」

我们听到的一个相关的主题，也是一个与HitPiece有关的主题，就是那些在音乐 / Web3生态系统中工作的人，要么不了解版权法的基本知识，要么就心甘情愿地藐视它，为艺术家创造了糟糕的结果。一位电子音乐和嘻哈界的制作人，在回答我们关于他们不喜欢的Web3项目的问题时，问道：「难道NFT世界中没有人尊重版权和知识产权所有权吗？」然后感叹道：「艺术家没有从这些企业中得到任何钱。」 通过 HitPiece，我们看到当艺术家的音乐未经许可被发上 HitPiece 网站时，他们的反应是多么消极。这种从创新的角度「快速行动和打破常规 」的压力与获得适当许可和尊重艺术家现有法律权利的需要之间的紧张关系反复出现，而且我们预计这种紧张关系将继续凸显。

一个唱片公司高级经理写的评论中提到了这些主题的变化，他告诉我们，在音乐 / Web3项目中，围绕 Web3 对音乐行业的实际利益的思考存在真空。他们指出，「大多数Web3项目都是用最抽象的术语来谈论的，而没有真正谈论他们正在做的实际工作是什么，以及它如何使行业受益，或者未来行业的支柱应该是什么。」 他还告诉我们，他们认为「在这个空间里，对于艺术家和厂牌来说，什么是好的作品？这存在着真正的知识和可及性差距。」部分原因是缺乏高质量的信息，以及对 Web3 与现有音乐行业结构之间关系的清晰思考。

另一些人对音乐 / Web3 初创公司向艺术家提出的一些关键前提表示不解。一家唱片公司的初级职员告诉我们对放弃创意所有权的具体担忧：

「艺术家终于在对他们的音乐权利拥有更大的所有权方面取得势头，我认为他们不应该把长期所有权交给粉丝——这是一条危险的道路。我们应该探索艺术家筹集资金的其他方式，我不认为它一定在加密世界里，如果这里的情绪必须集中在投资回报的话。」

上述相关主题都可以被解读为受访者对音乐 / Web3 生态系统与现有音乐行业和艺术家之间的关系普遍持负面看法，或者至少持怀疑态度。这对那些在音乐 / Web3 领域的建设者来说尤其成问题，因为他们的主要产品围绕着艺术家和他们创造的音乐，如果他们不能说服艺术家相信 Web3 的价值，那么他们就可能面临清算。

资本主义和金融化

在书面答复中，我们还看到了一个强烈的主题，告诉我们许多音乐行业的受访者最终并不相信 Web3 能把我们从资本主义的罪恶中拯救出来。

一位来自音乐行业从事高级管理的律师告诉我们，他们担心「『Web3』的新世界可能会重新生成旧世界的权力动态和资本主义最糟糕的部分，」然后他问道：「我们怎样才能在不重新实现这些东西的情况下参与进来？」他们答案的最后部分很有说服力，它表明音乐行业的一些人至少对 Web3 改善其结构的潜力持开放态度，但需要更多的时间、教育或生态系统中的建设者的更清晰的论据来说服他们。同样的情绪在一位从事电子和爵士音乐的艺术家的评论中出现，他告诉我们，他们「100%相信很多艺术家和粉丝的出发点是正确的，他们希望有一个更公平的音乐经济」，然后指出他们「只是不认为Web3 / 加密世界会做到这一点」，「即使能源问题得到解决，它仍然会导致根本上不公平和退步的安排。」

这个总体主题的根源在于对将艺术变成完全投机性商品的担忧。一位独立艺术家告诉我们，他们「不认为音乐被日益金融化并变成投机的或可交易的资产是互联网音乐的理想未来」，他还说，「互联网可以『首先』被用来建立音乐社区和粉丝，这在以前已经发生过很多次了，我们不需要用金融资产来引导」。另一位来自音乐行业发行方的受访者将对金融化的担忧与缺乏教育的担忧联系起来，认为「艺术家和粉丝总是会输，因为他们没有金融方面的专业知识，无法获胜。」

这些反应都表明，除了可能有或可能没有的结构性问题，Web3 还有一个持续的认知问题。许多音乐行业的人没有看到他们的长期优势，而许多潜在的不利因素却十分清晰，特别是在财务方面。事实上，我们的一些受访者谈到，他们不知道 Web3 如何或为什么需要 Web3 来改进 Web2 技术。一位唱片公司经理告诉我们，他们看不出 Web3 能解决任何「现有方法不能解决的问题，尽管它有一些新的想法」。另一位受访者告诉我们，「Web3 没有带来任何新工具」，他指出，他们已经可以让粉丝直接支持艺术家，「那些方法都不需要区块链。」

结语

有许多方法可以剖析蓬勃发展的音乐 / Web3 全局，以及其最活跃的面向行业的创造者和战略家的担忧。一些人正在完全跳出传统行业；另一些人正在努力建立 Web2 和 Web3 之间的桥梁，还有一些人正在等待时机。无论他们对 Web3 项目的参与程度如何，艺术家和音乐行业的专业人士对未来的发展前景有许多相同的担忧，特别是在涉及到教育和财务障碍以及法律和环境问题时。

重要的是，围绕音乐 NFT 的讨论持续两极分化，对 Web3 的熟悉并不能自动转化为对该技术的积极情绪。这表明，Web3 原生音乐公司投资的任何入驻策略都必须远远超出关于该技术如何运行的表面教育，而是要明确、严密地说明该技术如何真正为艺术家及其团队在 Web2 音乐行业中已有的选择增加价值。根据我们的调查，显然有许多音乐行业的人仍然认为来自 Web3 方面的信息不明确或不一致。

也就是说，音乐 / Web3 应用和社区正在迅速发展——在过去几个月里，出现了许多音乐 DAO 和艺术家社交通证——这很可能改变行业对该技术的心理模式。我们期待随着 Web3 的成熟和扩展对整个行业的情绪进行定期的温度检查。
