“地球上最强大的力量是什么？在 19 世纪是上帝；在 20 世纪是美国军队；而到了 21 世纪中期则是加密技术。正如阿桑奇所说，再多的暴力也无法解决特定的数学问题。因此你有多少核武器并不重要，如果财产或信息是由加密技术保证的，那么国家在没有解出那些方程之前就无法褫夺这些财产与信息。”
随着新的列维坦缓缓降临，我们是否能基于这种基于技术对抗强权，创造新的人生选择？
与此同时，在当今的世界，美国政府对于历史的篡改、对于意识形态的控制也是不争的事实：
“今天的美国机构大量报道俄乌战争，让和俄罗斯看起来很邪恶，而美国看起来很美好。它没有提到 1900 年的八国联军，或 1932 年的乌克兰大饥荒，因为这些都是负面的历史。
现代美国，让你所听到的历史，是当权者认为对其内部和外部对手有政治意义的历史。”
美国政府会通过在和平时期主动树敌，来加强内部凝聚、动员能力：
“暴行故事有时以恐怖分子为框架，有时以儿童为框架......但一般的概念是“发生了如此糟糕的事情，我们必须使用（国家）武力来防止它再次发生”。但人们通常完全忽略了这种力量本身所造成的死亡。
有时候暴行故事完全是假的：在伊拉克被诬陷持有大规模杀伤性武器之前，它还被诬陷把婴儿从保温箱中赶出去。”
我们是否有可能用区块链的技术，去沉淀真相，打破美国政府对于历史与意识形态的垄断？
事实上，技术已经在与权术展开了正面的竞争。本书提到了一些例子：
“在一起专利诉讼中，中国法院使用链上时间戳来确定优先权。一家公司证明它不可能侵犯另一家公司的专利，因为它在另一家公司申请之前就已经“上链”了。”
在这背后，是数学规律对于事实的捍卫：
“为什么比特币可以抵御篡改历史？为了伪造一个交易者，你需要伪造一个数字签名；为了伪造一个交易物，你需要破坏一个哈希函数；为了伪造一个交易时间，你需要篡改一个时间戳。并且你在伪造的同时，要保证在序列化区块头部格式的机制下，不能破坏与该交易有关的所有其他加密记录。”
而我们是否有机会基于这种底层技术，建立不被美国政府这样超大型主权国家控制的新型共同体？
在 《Network State》一书中，正在对于这种人类新型共同体进行定于与构想：
Network State 是一个高度一致的在线社区，具有集体行动的能力，在全世界范围内众筹土地，并最终获得此前已存在国家的外交 Recognition 。

A network state is a highly aligned online community with a capacity for collective action that crowdfunds territory around the world and eventually gains diplomatic recognition from pre-existing states.

这样的 Network State 将拥有自己的加密货币、由社会智能合约所约束的 Consensus Government、由众筹的实体领土组成的群岛、虚拟资本、以及一个链上的人口普查系统，以证明有足够多的人口、收入、不动产。
当然最宏观的，是一个由无数共识个体所组成的、超越民族国家和地理限制的共同体。

更多书中的精彩论点：
【三级】

今天的世界正在变成三极。

地缘政治博弈的二元竞争中，比特币正在成为新的力量。

这三极中的每一极都有一个真理的来源：媒体、党派，或协议（比特币）；每一极都有一个围绕该真理之源的数字经济：美元经济、数字***，或 web3 加密经济。

每一极点本身就是一个网络，它站在国家之外，纽约时报的网络给美国国家指明方向，党派的网络领导着国家机器，比特币的网络则站在所有国家之外，而每一极都有一个主导的意识形态。

【一诫】

一个正在诞生的 Network State 需要 “一诫”：

与科技相类似：它将一个初创社会集中在一个单一的道德创新上，就像科技公司把自己集中在一个单一的技术经济创新上。

不用很复杂，“140个字符”的主张听起来微不足道，“可重复使用的火箭”的主张似乎不切实际，但这些主张分别带来了 Twitter 和 SpaceX 的诞生。

每一个基于“一诫”的初创社会，都是以解构建制在某一特定领域的历史为前提，用新的“一诫”来取代它，然后证明“一诫”的社会经济价值，以吸引订阅者，也就是它的公民。

比方说，如果你能通过对肥胖症流行的深入研究，吸引 10 万名用户加入你的 Keto Kosher 无糖协会，并证明他们因此明显减轻了体重，你就证明了建制系统内在一个关键领域存在着严重错误，这将促使他们进行改革，或者不改革——在这种情况下，你将吸引更多的公民。

【道德创新】

当我们用 “一诫” 承载一种核心价值观，背后隐含着一种社会的道德创新。

我们想要一个怎样的、更理想的社会？每个人都可以提出自己的观点，呼唤与自己共识的伙伴，带着热情，将所有的注意力集中在一个痛点（一诫），并且基于互联网缔造 Network State。

和创业公司一样，任何走向新的社会政治秩序的个人实验都可能成功或失败，但是只要一些基于一诫的创业公司成功了，他们就可以互相复制经过验证的道德创新。
