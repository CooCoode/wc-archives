Unity 游戏共学招募捕捉下一个神作前的灵光乍现

2017 年最佳独立游戏《空洞骑士》描绘了一个恢弘的地下史诗（Underground），虫蚁世界里神权与王权交替，潮湿晦暗的甬道里，回荡着充满哲意的低语。

后来我知道，它是由三名澳大利亚主创在 Game Jam 的灵光乍现，随后在 Kickstarter 上超额众筹，开发近三年才成就的一部游戏史上的神作——这也成了这期「Unity 独游开发共学」的起源。

独立游戏，是由创造者们独立开发的游戏，没有资本、发行商等外部实体控制，是属于个体的叙事，用无限的内部空间，在物理世界扩展想象的边界。

如果说一个游戏就是一个小世界，游戏引擎就是一个「平行世界联邦」，作为目前最广泛使用的游戏引擎之一，Unity 服务了世界过半游戏的开发。

除了耳熟能详的《原神》《王者荣耀》等大型游戏，还有《纪念碑谷》《茶杯头》《捣蛋鹅》《Ori》两部曲等独立经典之作都是基于 Unity，它是迄今最好的游戏创作者解决方案之一。

而在 2D 游戏之外，3D 与 AR/VR 正在席卷下一代交互，Apple Vision Pro 发布会上的短片，让我们的得以一窥未来视界的奥妙，Unity 作为铺垫未来的魔法石，正在成为我们构筑新世界的基础工具。

cr: Cody Jenson

本期共学我们聚焦于 Unity 零基础入门，通过实际的游戏实战开发项目，学习和应用 Unity 引擎的各种功能和工具，让AI 工具成为你的助力，一起收获游戏开发的技能和乐趣。

在这里，成为专业的游戏开发者与否并不重要，我们希望你能遇到相似的灵魂，碰撞出灵感的火花，一起用工具实现天马行空的创意，捕捉下一个神作之前的灵光乍现。

共学只是通向这个世界的开始：我们将在技术的升力下、在 各种最新工具引擎的热气球中，向下看到生命里迥异的风景。
